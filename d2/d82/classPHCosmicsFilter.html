Detailed Description

PHCosmicsFilter is a SubsysReco module that filters cosmic ray tracks It provides methods to enable debug ntuple writing and to enable track creation Selection criteria can be configured with maximum distance to origin and minimum number of clusters The module accesses the node tree to obtain a TrkrClusterContainer and optional geometry information When track creation is enabled it creates a TrackSeedContainer with a configurable name Internal calculations include chi square evaluation phi addition phi difference point key conversion cost function and stub extraction using a Boost rtree Output ntuples are written to a ROOT file if the corresponding flag is set This comment was generated by openai/gpt-oss-120b at temperature 0.1.
