Detailed Description

Container for TPC space charge distortion correction histograms

Load histograms from an input file and save histograms to an output file Configuration flags include number of dimensions scale factor usage phi histogram unit and z interpolation handling Stores radial phi and z distortion histograms for two charge signs Stores temporary entry count histograms used during correction building

void TpcDistortionCorrectionContainer::save_histograms ( const std::string & destination) const

save histograms to out file

Save all distortion correction histograms to a ROOT file.

The method creates a ROOT file at the specified path, writes each non‑null histogram from the internal containers (m_hentries, m_hDRint, m_hDPint, m_hDZint) using its own name, and then closes the file.
