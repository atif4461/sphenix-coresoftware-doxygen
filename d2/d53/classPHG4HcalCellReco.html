Detailed Description

PHG4HcalCellReco reconstructs hcal cells from g4 hits within the fun4all framework It inherits from SubsysReco and PHParameterInterface to integrate with the node tree InitRun creates required node entries and loads parameters process_event converts g4 hit containers to cell containers applying timing window and optional fixed energy SetDefaultParameters defines default values for detector name hit node cell node timing window and energy checks Detector sets the target detector subsystem name checkenergy enables or disables energy conservation verification set_timing_window configures minimum and maximum hit times for cell building set_fixed_energy overrides cell energy with a constant value CheckEnergy validates total energy before and after reconstruction when enabled This comment was generated by openai/gpt-oss-120b at temperature 0.1.
