SvtxTrackState * SvtxTrack_v4::insert_state ( const SvtxTrackState * state)
overridevirtual

Insert a track state into the internal state map keyed by path length.

The function searches for the closest existing entry in the map using the path length of the supplied state. If no entry with the same path length exists, a deep copy of the supplied state is created and inserted. The iterator to the matching or newly inserted entry is then returned.

Parameters
statePointer to a const SvtxTrackState to be inserted.
Returns
Pointer to the SvtxTrackState stored in the map (either the existing matching state or the newly created copy). This comment was generated by openai/gpt-oss-120b at temperature 0.1.
