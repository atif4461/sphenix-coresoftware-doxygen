◆ best_track_from()

SvtxTrack * TrkrNtuplizer::best_track_from ( TrkrDefs::cluskey cluster_key)

Retrieve the SvtxTrack with the best quality for a given cluster key.

This method first checks an internal cache for a previously found best track. If not cached, it obtains all tracks associated with the cluster, selects the track with the lowest quality value, stores the result in the cache and returns it.

Parameters
cluster_keyIdentifier of the cluster (TrkrDefs::cluskey).
Returns
Pointer to the SvtxTrack with the lowest quality associated with the cluster, or nullptr if no tracks are found. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
