Detailed Description

brief Class responsible for writing GDML structure information extending parameter volume writer details Implements methods to write geometry hierarchy auxiliary data divisions replicas physical placements and optical surface caches Provides interface to enable optional export of energy cut values per logical volume Uses configuration object to control exclusion of volumes during export param config_input Pointer to GDML configuration used for export control member structureElement DOM element representing the top level structure node member borderElementVec Container of DOM elements for border surfaces member skinElementVec Container of DOM elements for skin surfaces member auxmap Mapping from logical volumes to auxiliary data lists member opt_vec Cache of optical surface pointers member reflFactory Factory for handling reflection surface creation member cexport Flag indicating whether energy cuts are exported member config Constant configuration object governing export exclusions
