Detailed Description

Subsystem reconstruction module that builds raw clusters from tower data.

This class inherits from SubsysReco and integrates with the Fun4All framework. It creates and manages a RawClusterContainer node, applies an energy threshold to towers, and optionally checks energy conservation for the specified detector.

Parameters
nameOptional module name, defaults to RawClusterBuilderGraph.

@method InitRun Called once per run to locate or create required nodes. @method process_event Called for each event to perform clustering. @method End Called at the end of processing to clean up resources.

@member _clusters Pointer to the output RawClusterContainer. @member _min_tower_e Minimum tower energy threshold for clustering. @member chkenergyconservation Flag to enable energy conservation check. @member detector Name of the detector subsystem (e.g. CEMC, HCALIN). @member ClusterNodeName Name of the cluster node in the node tree.

Note
The detector name must be set before processing events.
