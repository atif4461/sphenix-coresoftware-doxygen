Detailed Description

Intt pool class provides storage and decoding of intt packets It allocates a buffer of unsigned int values with configurable depth The pool tracks required depth low mark and current write position Methods allow adding packets retrieving raw and decoded values Decoding extracts hit information including bco fee channel id adc amplitude and counters The class maintains maps of last bco per fee and lists of fees per bco Verbosity and name can be set for diagnostic output Depth checks and next index handling are provided This comment was generated by openai/gpt-oss-120b at temperature 0.1.

virtual

Retrieve integer quantities from the INTt data pool based on a query string.

This method first ensures the INTt data are decoded, then interprets the what argument to return various integer values such as fee length, number of fees, unique fee identifiers, hit attributes, and other bookkeeping quantities. If the requested quantity is unavailable or the input indices are out of range, the function returns zero (or –1 for an invalid fee identifier request).
