Detailed Description

Container class for Tpc cylinder geometry objects indexed by layer identifier Provides methods to add retrieve and iterate over layer geometry entries Inherits from PHObject to integrate with Fun4All node tree infrastructure Identify method prints basic information to standard output AddLayerCellGeom overloads allow insertion by explicit key or by using internal key from geometry object GetLayerCellGeom returns pointer to geometry for specified layer GetFirstLayerCellGeom returns pointer to geometry of first stored layer getNLayers returns number of stored layers getBeginEnd returns constant iterator range over stored geometry map This comment was generated by openai/gpt-oss-120b at temperature 0.1.
