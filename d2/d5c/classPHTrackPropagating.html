Detailed Description

Base class for track seeding.

Base class for track propagation modules in the Fun4All framework Provides common functionality for initializing run processing events and finalization Manages access to cluster vertex and track maps from the node tree Allows configuration of the track map name and optional use of truth clusters Derived classes must implement Setup Process and End specific to the tracking algorithm This comment was generated by openai/gpt-oss-120b at temperature 0.1.
