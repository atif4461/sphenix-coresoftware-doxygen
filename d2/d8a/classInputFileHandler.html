Detailed Description

InputFileHandler manages a list of input files for sequential processing Allows adding files individually or from a list and opening them one by one Keeps track of opened files and provides query functions for open state repeat count and verbosity Supports execution of an external script before opening a file with optional arguments Provides virtual hooks for fileopen fileclose ResetFileList and IsOpen to be overridden by derived classes

◆ OpenNextFile()

int InputFileHandler::OpenNextFile ( )

Open the next file in the internal file list.

The method iterates over the queued file names, optionally prints diagnostic messages when verbosity is enabled, executes a user defined opening script if present, and attempts to open each file. On successful opening the function returns a success code; otherwise the file name is removed from the list and the next entry is tried.

Returns
InputFileHandlerReturnCodes::SUCCESS if a file was opened successfully, otherwise InputFileHandlerReturnCodes::FAILURE.
