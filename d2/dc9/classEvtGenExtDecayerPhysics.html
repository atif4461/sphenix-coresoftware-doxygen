void EvtGenExtDecayerPhysics::ConstructProcess ( )
overrideprotected

Construct the external EvtGen decayer and attach it to all decay processes of instantiated particles.

The method restores the default std::cout formatting, creates a G4EvtGenDecayer, loads the decay table from the configured file (searching locally and then in $CALIBRATIONROOT/EvtGen), and iterates over all particle definitions to set the external decayer for each G4Decay process. If no particle uses the decayer it is deleted. Verbose output is controlled by the member verboseLevel. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

Loop over all particles instantiated and add external decayer to all decay processes if External decayer is set
