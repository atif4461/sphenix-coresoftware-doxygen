Detailed Description

Implementation of SvtxTrackMap interface using an STL map Provides storage retrieval and iteration over SvtxTrack objects keyed by unique track identifier Supports deep copy via CloneMe reset semantics and standard container size queries All public methods forward to the internal TrackMap instance
