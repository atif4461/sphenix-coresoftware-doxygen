◆ process_event()

int TpcCentralMembraneMatching::process_event ( PHCompositeNode * topNode)
overridevirtual

event processing

Process a single event, perform CM cluster matching, fill output containers and distortion‑correction histograms.

This routine is invoked by Fun4All for each event. It retrieves the EventHeader and LaserEventInfo nodes, applies optional header‑based event indexing, selects laser events, reads the corrected CM cluster map, applies static, average and module‑edge distortion corrections, evaluates saturation, fills per‑side residual graphs, performs optional “fancy” matching of reconstructed clusters to truth positions, populates the CMFlashDifference container, updates per‑index residual maps, and normalises the per‑event distortion‑correction histograms. Histograms and trees are filled only when the corresponding flags are enabled.

Parameters
topNodePointer to the top PHCompositeNode that holds the full node tree for the current event.
Returns
Fun4AllReturnCodes status code – EVENT_OK on successful processing or ABORTRUN if required nodes are missing.
