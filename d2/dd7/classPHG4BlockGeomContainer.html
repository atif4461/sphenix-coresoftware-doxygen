Detailed Description

brief Container for PHG4BlockGeom objects indexed by layer identifier inherits from PHObject and provides map based storage of geometry objects supports adding geometry with explicit layer key or automatic key assignment retrieval by layer key returns pointer to PHG4BlockGeom utility functions report number of stored layers and iterator range for const access magnetic field value is stored as float initialized to NaN
