Detailed Description

InttXYVertexFinder is a SubsysReco module that reconstructs the XY vertex position using INTT detector data The module creates and manages INTTXYvtx and InttVertexMap nodes It provides configuration methods for period beam center histogram saving and QA drawing The processing sequence follows the standard Fun4All callbacks Init InitRun process_event End This comment was generated by openai/gpt-oss-120b at temperature 0.1.
