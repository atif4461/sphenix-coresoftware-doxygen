Detailed Description

Class provides analysis module that writes G4 hit information to a ROOT TTree It inherits from SubsysReco and implements required callbacks Init creates output node and histograms process_event extracts hit containers for configured detector and fills tree End writes tree and cleans up resources Detector sets name of detector to process BlackHoleName sets name of absorber node SaveHits enables or disables hit saving This comment was generated by openai/gpt-oss-120b at temperature 0.1.
