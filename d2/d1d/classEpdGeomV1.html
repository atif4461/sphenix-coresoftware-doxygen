Detailed Description

Implements the EpdGeom interface for version 1 geometry Provides getter functions for radius z and phi values of a tile identified by a key Provides setter functions for radius z phi and phi0 values of a tile identified by a key Internally stores tile radius values in an array of size 16 tile z values in an array of size 2 tile phi values in an array of size 24 and tile phi0 values in an array of size 12
