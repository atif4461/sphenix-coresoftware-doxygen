Detailed Description

Class provides utilities for counting and matching tracker clusters It stores keys for svtx and phg4 clusters and computes statistics Methods allow resetting adding cluster keys finding matches and retrieving counts Cluster location diagnostics are also available This comment was generated by openai/gpt-oss-120b at temperature 0.1.

int G4Eval::ClusCntr::addClusKeys ( SvtxTrack * track)

Populate svtx_keys with hit‑set and cluster key pairs from the supplied track.

The method clears the existing svtx_keys container, iterates over all cluster keys associated with the given SvtxTrack, converts each cluster key to its corresponding hit‑set key using TrkrDefs::getHitSetKeyFromClusKey, stores the (hitSetKey, clusterKey) pair, sorts the resulting vector, and returns the total number of entries added.

Parameters
trackpointer to the SvtxTrack whose cluster keys are to be processed.
Returns
size of svtx_keys after insertion (number of cluster‑key pairs added). This comment was generated by openai/gpt-oss-120b at temperature 0.1.
