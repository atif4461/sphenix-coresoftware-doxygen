int PHSiliconTpcTrackMatching::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run-specific resources and retrieve required nodes.

This method updates parameters from the macro, optionally creates an output ROOT file and TNtuple for track matching studies, prints the configured search windows, obtains necessary nodes from the top node, parses the magnetic field strength, and initializes the window matcher objects for spatial and angular residuals.

Parameters
topNodePointer to the top PHCompositeNode of the current Fun4All job.
Returns
Integer status code: Fun4AllReturnCodes::EVENT_OK on success, otherwise an error code indicating node retrieval failure.
