Detailed Description

MBDTriggerEmulator emulates the MBD trigger logic within the Fun4All framework It inherits from SubsysReco and implements the standard lifecycle methods The class acquires and creates required nodes for waveforms primitives and trigger output Configuration methods allow setting of charge time and slewing lookup table files Processing methods convert raw waveforms to primitives and evaluate trigger decisions Internal arrays store charge time hit counts and output words for each trigger region Counters track number of processed events passed triggers and other diagnostics
