Apply a trivial shower depth correction.

This virtual method copies the input position (x, y, z) to the corrected output coordinates (xc, yc, zc). It serves as a default implementation that can be overridden by derived classes to incorporate detector‑specific depth‑dependent corrections based on cell indices and deposited energy.

◆ Momenta()

void BEmcRec::Momenta ( std::vector< EmcModule > * phit,
float & pe,
float & px,
float & py,
float & pxx,
float & pyy,
float & pyx,
float thresh = 0 ) const

Compute first and second order energy moments for a collection of EMCal modules.

This routine scans the supplied vector of EmcModule objects, identifies the tower with the maximum energy, and then calculates the energy weighted centroid and covariance relative to that tower. The results are returned through reference arguments: total energy, centroid coordinates (x,y) and the variance‑covariance matrix elements (xx,yy,yx). Only towers with amplitude greater than thresh contribute to the moments.

Parameters
phitPointer to a vector of EmcModule objects representing the hit towers.
peReference to a float that will receive the total summed energy.
pxReference to a float that will receive the x coordinate of the centroid.
pyReference to a float that will receive the y coordinate of the centroid.
pxxReference to a float that will receive the variance in x (second moment).
pyyReference to a float that will receive the variance in y (second moment).
pyxReference to a float that will receive the covariance between x and y.
threshEnergy threshold; towers with amplitude below this value are ignored.
Returns
None. Results are provided via the reference parameters.
