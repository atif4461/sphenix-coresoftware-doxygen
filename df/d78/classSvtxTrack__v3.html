overridevirtual

Erase a track state at the specified path length.

Searches the internal state map for the entry keyed by pathlength. If found, the associated state object is deleted and the entry removed. The function returns the number of remaining states after the operation. If no entry is found, the map is left unchanged and its current size is returned.

Parameters
pathlengthThe path length key identifying the state to be erased.
Returns
The number of states remaining in the track after erasure.
