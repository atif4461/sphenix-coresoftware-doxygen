◆ Add_32()

void CaloCalibEmc_Pi0::Add_32 ( )

Merge CEMC eta‑phi histograms in a 32‑bin pattern.

This routine iterates over three eta thirds and, for each 16‑by‑16 block of eta‑phi bins, adds the contents of the corresponding shifted histograms (offset by 0 or 16 bins in eta and phi) to the base histogram. The operation is performed only when either the eta pattern offset or the phi pattern offset is non‑zero, effectively aggregating neighboring histogram sectors into a coarser 32‑by‑16 representation.

int CaloCalibEmc_Pi0::InitRun ( PHCompositeNode * topNode)

Initialize run‑specific objects, histograms and the event TTree.

Called for first event when run number is known. Typically this is where you may want to fetch data from database, because you know the run number. A place to book histograms which have to know the run number.

This method is invoked once per run when a new run is encountered. It creates the output ROOT file, allocates all analysis histograms, and sets up the per‑event TTree with branches for event number, cluster counts and cluster properties. The event counter is reset to zero.
