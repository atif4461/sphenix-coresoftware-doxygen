Detailed Description

Container for MbdPmtHit objects associated with each PMT in the MBD detector.

Inherits from PHObject and implements the standard PHObject interface for identification resetting validity checking and printing

Provides methods to set and get the number of PMTs and to retrieve individual MbdPmtHit objects by index
