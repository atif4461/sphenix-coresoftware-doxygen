void PHG4GDMLDetector::ConstructMe ( G4LogicalVolume * logicWorld)
overridevirtual

construct

Construct the GDML defined detector geometry and place it inside the provided world volume.

The method reads the GDML file specified by m_GDMPath, retrieves the top volume named m_TopVolName, applies the configured rotations and translation, creates a physical placement and registers the logical volume with the owning subsystem. If m_skipDSTGeometryExport is true the placed volume is excluded from DST geometry export.
