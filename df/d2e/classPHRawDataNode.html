Detailed Description

PHRawDataNode is a concrete PHDataNode that stores raw detector data as an array of PHDWORD The node holds metadata including length ID wordLength and hitFormat Provides accessor and mutator methods for the metadata fields Implements write method to allow serialization via PHIOManager This comment was generated by openai/gpt-oss-120b at temperature 0.1.
