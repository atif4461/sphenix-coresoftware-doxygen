Detailed Description

Utility class for geometric and visualization calculations in the sPHENIX Geant4 framework Provides static methods for rapidity coverage length conversion and eta theta conversions GetLengthForRapidityCoverage computes detector length for a given radius and pseudorapidity Overloaded GetLengthForRapidityCoverage uses the globally set pseudorapidity coverage SetPseudoRapidityCoverage stores a global pseudorapidity value for subsequent calculations SetColour assigns visualization attributes based on material name get_theta converts pseudorapidity to polar angle theta get_eta converts polar angle theta to pseudorapidity get_etaphi returns pseudorapidity and azimuth from Cartesian coordinates Overloaded get_eta computes pseudorapidity from radius and longitudinal position line_and_rectangle_intersect tests intersection of a line segment with an axis aligned rectangle lines_intersect computes the intersection point of two line segments circle_rectangle_intersection evaluates overlap between a circle and a rectangle sA computes signed area contribution used in polygon winding calculations md5sum returns the MD5 checksum string of a file Private static member _eta_coverage holds the configured pseudorapidity coverage value
