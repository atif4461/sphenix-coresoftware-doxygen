Detailed Description

simple truth vertex smearing algorithm

Implements fast simulation reconstruction of MBD vertex Provides initialization and event processing hooks for Fun4All Allows configuration of time and longitudinal smearing parameters Internal method creates required nodes in the PHCompositeNode tree Uses GSL random number generator for smearing operations This comment was generated by openai/gpt-oss-120b at temperature 0.1.
