int QAG4SimulationTpc::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run specific resources and histograms for the TPC QA module.

This method is called once for each new run. It creates the evaluation stack, retrieves the truth information container, obtains the TPC geometry, builds internal layer‑to‑region mappings, and registers all required histograms with the global histogram manager. Re‑initialisation is prevented on subsequent calls.

Parameters
topNodePointer to the top level PHCompositeNode containing the full node tree for the current job.
Returns
Fun4AllReturnCodes::EVENT_OK on successful initialisation, Fun4AllReturnCodes::ABORTRUN if required geometry nodes are missing.
