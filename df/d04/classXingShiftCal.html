Detailed Description

Class responsible for calculating and applying crossing shift corrections using spin and polarization data Provides initialization per event processing calibration and database commit interfaces Manages packets for blue and yellow spin polarization asymmetry and fill patterns Stores event counters thresholds and pattern arrays for up to 120 bunches and 6 triggers Implements methods to compute crossing shift write results to CDB and perform QA on SpinDB
