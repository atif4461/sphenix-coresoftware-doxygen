std::set< SvtxTrack * > SvtxTrackEval::all_tracks_from ( PHG4Hit * truthhit)

Retrieve all SvtxTrack objects that share the same truth track identifier as the given PHG4Hit.

This method scans the SvtxTrackMap, extracts cluster keys for each track, obtains the corresponding truth hits via the ClusterEval component, and selects tracks whose associated truth hits have a matching track ID to the input PHG4Hit. Optional caching of results is performed when enabled.

Parameters
truthhitPointer to the PHG4Hit whose truth track ID is used for matching.
Returns
Set of pointers to SvtxTrack objects that are linked to the specified truth hit.

◆ create_cache_track_from_cluster()

void SvtxTrackEval::create_cache_track_from_cluster ( )

Populate a cache that maps each cluster key to the set of tracks containing that cluster.

The function first verifies that required node pointers are available; if not it increments an internal error counter and exits. It then iterates over all tracks in the track map, obtains the list of cluster keys associated with each track, and for each cluster key updates the cache. If a cache entry for a given cluster key already exists, the current track is added to the existing set; otherwise a new entry is created containing the current track. After processing all tracks the function sets a flag indicating that the cache is valid.

Returns
None
