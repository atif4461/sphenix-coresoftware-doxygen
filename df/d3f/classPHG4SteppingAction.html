Detailed Description

Base class for detector stepping actions in the Fun4All framework.

Provides the interface for processing a single Geant4 step within a given volume. The pure virtual method UserSteppingAction must be implemented by derived detectors and returns true when hits associated with the step are processed.

Verbosity control

The Verbosity methods allow setting and querying the output level for debugging.

Scintillation handling

GetScintLightYield returns the number of scintillation photons for a step, requiring a custom SCINTILLATIONYIELD property. GetVisibleEnergyDeposition returns the energy available for scintillation in GeV.

Extraction of local hit coordinates

StoreLocalCoordinate extracts pre‑ and post‑step positions and stores them in a PHG4Hit.

Light correction utilities

SetLightCorrection defines inner and outer radial correction parameters. GetLightCorrection provides the correction factor for a given radius or (x,y) position. ValidCorrection indicates whether correction parameters have been set.

Hit node naming

SetHitNodeName allows the stepping action to obtain G4HIT node names from the subsystem instead of constructing them manually.

Init method performs full initialization of histogram manager and output file

Init and InitWithNode can be overridden for detector‑specific setup.

Interface pointer handling

SetInterfacePointers can be used to obtain pointers to other nodes in the PHCompositeNode tree.

Printing support

Print provides a generic interface for outputting detector‑specific information. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
