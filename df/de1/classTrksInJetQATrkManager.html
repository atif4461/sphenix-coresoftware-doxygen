Detailed Description

===========================================================================

Track histogram manager for TrksInJetQA module

This histogram manager defines what to histogram from tracks. Provides track quality assurance management for jet analysis Enumerates track types and histogram identifiers Stores per track QA variables and fills histograms Inherits base manager functionality for histogram definition
