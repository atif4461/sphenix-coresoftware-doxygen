Detailed Description

Helper utilities for INTt cylinder geometry transformations Provides static methods to locate segment and strip centers and to convert between local and world coordinate systems using Acts geometry Methods operate on generic Surface objects and require a pointer to the ActsGeometry instance The find_segment_center method fills a location array with the global coordinates of the segment centre The find_strip_center method computes the global position of a specific strip given its segment and column indices and returns the result via a location pointer The get_world_from_local_coords overloads convert a 2D or 3D local coordinate to a TVector3 world position The get_local_from_world_coords performs the inverse transformation This comment was generated by openai/gpt-oss-120b at temperature 0.1.
