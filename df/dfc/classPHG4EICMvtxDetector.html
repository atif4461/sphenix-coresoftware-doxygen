Detailed Description

Class representing the MVTX detector for the EIC simulation within the Fun4All framework Provides construction of geometry volume identification utilities and configuration handling Manages active layer flags absorber status black hole settings and stave layout parameters Interfaces with display actions and parameter containers to support visualization and runtime configuration

Determine whether a given physical volume corresponds to a sensor.

This method checks if the supplied G4VPhysicalVolume pointer matches any of the stored sensor volumes for the current detector layer. If verbosity is enabled, diagnostic information about the volume name is printed to standard output.
