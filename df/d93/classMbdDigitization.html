Initialization, called for at overall initialization.

Initialize the MbdDigitization module.

This function is called once when the module is registered with the Fun4AllServer. It creates the required nodes in the provided top node, instantiates a PDG database for particle information, and configures a Gaussian timing resolution function with sigma fixed to the module timing resolution.
