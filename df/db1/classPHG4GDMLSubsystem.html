Detailed Description

PHG4GDMLSubsystem is a generic detector built from a GDML import.

PHG4GDMLSubsystem integrates GDML geometry into the Fun4All framework It creates detector and stepping action nodes registers hit containers and provides accessors for the underlying PHG4Detector InitRunSubsystem builds geometry and hit nodes process_event forwards hit lists to the stepping action Print outputs subsystem information GetDetector returns the associated detector instance SetDefaultParameters initializes configurable parameters
