Detailed Description

Class representing level 1 trigger output version 1 Provides accessors and mutators for trigger type keys bits words and event identifiers Implements interface defined in LL1Out Stores trigger words in a map from word index to vector of unsigned ints Supports query of validity reset identification and trigger decision logic

◆ LL1Outv1() [2/2]

LL1Outv1::LL1Outv1 ( const std::string & triggertype,
const std::string & ll1type )

Construct LL1Outv1 object for a specified trigger and LL1 type.

Parameters
triggertypeName of the trigger type used to obtain the trigger key and id.
ll1typeIdentifier for the level 1 subsystem.

The constructor resolves the trigger key and numeric id via TriggerDefs, stores the provided type strings, allocates a vector for trigger bits and determines the number of trigger words based on the trigger id (jet, photon, mbd, or pair). It then populates the trigger bits vector with appropriate words for each channel.
