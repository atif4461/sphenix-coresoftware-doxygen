Detailed Description

TPCMap provides mapping from FEE identifiers to geometric properties of TPC pads It supports retrieval of layer index pad number radial coordinate and azimuthal angle Map names can be configured via setMapNames Internal data is stored in a map keyed by a composite identifier This comment was generated by openai/gpt-oss-120b at temperature 0.1.
