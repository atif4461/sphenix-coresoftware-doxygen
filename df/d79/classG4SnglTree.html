Detailed Description

G4SnglTree class provides functionality to create a ROOT tree of Geant4 single hits Init method performs full initialization of node access and output file process_event method loops over detector hit containers and fills tree process_hit method extracts hit information for a given detector name and id End method writes and closes output file AddNode registers a detector node name with optional detector id Private members store output filename node postfix set detector id map tree pointer event tree structure and output file pointer This comment was generated by openai/gpt-oss-120b at temperature 0.1.
