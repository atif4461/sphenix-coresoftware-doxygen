Detailed Description

PHSimpleVertexFinder implements a simple vertex finding algorithm for sPHENIX reconstruction It operates on SvtxTrackMap and produces SvtxVertexMap using configurable cuts Configurable parameters include beam line cuts track dca cuts track quality cuts outlier pair cut track pt cut and zero field flag The module integrates with the Fun4All SubsysReco framework and accesses nodes via GetNodes and CreateNodes It provides setter methods to adjust analysis cuts and to require MVTX hits The algorithm computes pairwise dca between tracks removes outlier track pairs and builds vertex candidates Results are stored in multimap structures linking tracks to vertices and track pair PCA information
