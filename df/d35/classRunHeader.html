Detailed Description

RunHeader class stores run level metadata for Fun4All processing Provides interface to set and retrieve run number and timestamps Implements PHObject virtual methods Reset identify isValid Supports generic float and int attribute storage via name value pairs Timestamp helpers map BORTIME and EORTIME keys to integer values NoWarning static method suppresses virtual function warnings
