◆ ConsistencyCheck()

int PHG4OuterHcalDetector::ConsistencyCheck ( ) const

Verify geometric and configuration consistency of the outer HCal detector.

Checks that the inner radius is smaller than the outer radius, that the scintillator tile thickness does not exceed the scintillator gap, that scintillator inner and outer radii are ordered correctly with respect to each other and to the detector radii, and that the magnet cutout radii are not smaller than the corresponding scintillator or steel radii. The function aborts execution if any check fails.

Returns
Zero on successful validation. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

◆ ConstructHcalSingleScintillators()

void PHG4OuterHcalDetector::ConstructHcalSingleScintillators ( G4LogicalVolume * hcalenvelope)

Construct the outer HCal scintillator geometry and associated cutouts.

This method builds the extruded solid representation of each scintillator tile for the outer HCal, applying eta coverage, tilt angle, magnet cutout, and inter‑tile gaps. The tiles are intersected with a large envelope solid and stored in the member vector m_ScintiTilesVec. In addition a steel cutout solid for the magnet region is created and assigned to m_SteelCutoutForMagnetG4Solid. Optional display of the volumes is enabled when the SCINTITEST macro is defined.

Parameters
hcalenvelopePointer to the logical volume that encloses the HCal scintillator system.
Returns
None (void). The constructed solids are cached in class members. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

◆ SetTiltViaNcross()

void PHG4OuterHcalDetector::SetTiltViaNcross ( )

Compute and set the scintillator tilt angle from the number of crossings.

The method reads the integer parameter ncross from the detector configuration. If ncross is zero or a tilt angle is already defined, the parameter is marked as unused and the function returns. Otherwise the tilt angle is calculated geometrically using the inner, middle and outer radii of the outer HCal, the number of scintillator plates, and CGAL intersection operations on the corresponding circles and lines. The computed angle (in radians) is stored in the member variable m_TiltAngle and the configuration parameter tilt_angle is updated (in degrees).
