◆ fillEvent()

int fillEvent ( HepMC::GenEvent * evt)

Fill a HepMC::GenEvent with particles and heavy‑ion information extracted from a Hijing event.

This routine reads the Hijing event record (np, nt, ncoll, etc.), computes the participant eccentricity, constructs the HepMC::HeavyIon object, and attaches it to the supplied HepMC::GenEvent. It then creates the primary interaction vertex, adds the incoming projectile and target particles, and iterates over all Hijing final‑state particles to generate appropriate HepMC::GenParticle objects. Decay history handling, vertex creation based on spatial offsets, and spectator filtering are performed according to the configuration flags (e.g. keepSpectators, m_vertexOffsetCut). The function returns 0 on successful event construction.

Parameters
evtPointer to the HepMC::GenEvent that will be populated.
