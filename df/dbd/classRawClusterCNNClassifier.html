Detailed Description

RawClusterCNNClassifier is a SubsysReco module that loads an ONNX model and applies a convolutional neural network to raw calorimeter clusters The module reads cluster information from a configurable input node optionally filters clusters by minimum energy and writes the CNN output to a configurable output node Model path node names and minimum cluster energy can be set via public setter methods before initialization During Init the required nodes are created and the ONNX session is instantiated The process_event method retrieves the RawClusterContainer formats the input tensor runs inference and stores the result in the output node
