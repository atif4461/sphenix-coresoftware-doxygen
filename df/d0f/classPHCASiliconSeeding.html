◆ Setup()

int PHCASiliconSeeding::Setup ( PHCompositeNode * topNode)
overrideprotectedvirtual

Initialize module for a new run and configure geometry and seeding parameters.

This method is invoked by InitRun. It sets the track map name, retrieves the run number, determines the appropriate strobe width for the MVTX raw data, initializes detector geometry, and configures layer‑dependent search windows for silicon seeding. The INTT layers (3–6) receive a minimum z‑window of 1 cm due to their larger z resolution.

Parameters
topNodePointer to the top PHCompositeNode of the current run.
Returns
Fun4All return code indicating success (EVENT_OK) or an error code from InitializeGeometry.
