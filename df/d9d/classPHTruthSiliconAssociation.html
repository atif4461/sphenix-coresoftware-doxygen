Detailed Description

Module associates simulated silicon clusters with truth particles within Fun4All framework Implements required SubsysReco interface methods for initialization run processing and cleanup Retrieves G4 truth hit and cluster containers from node tree and builds track seeds from silicon clusters Manages random number generation using gsl rng wrapped in unique_ptr with custom deleter

Custom deleter for gsl_rng used by the random number generator in the module This comment was generated by openai/gpt-oss-120b at temperature 0.1.
