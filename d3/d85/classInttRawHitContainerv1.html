Detailed Description

Container for raw INTt detector hits for a single event.

Manages a collection of InttRawHit objects using a TClonesArray. Provides methods to add hits, retrieve hit count and individual hits, and to reset the container between events. Implements PHObject interface functions for identification and validity checking.

Note
The underlying TClonesArray is allocated on first use and owned by this container. It is cleared in Reset().
