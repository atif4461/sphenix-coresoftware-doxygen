Detailed Description

Base class for track seeding.

Provides track fitting functionality within the Fun4All framework Inherits from SubsysReco and implements the standard lifecycle methods Init sets up node pointers and resources InitRun prepares per run configuration and calls Setup process event invokes the derived Process implementation for each event Setup is a virtual hook for derived classes to acquire node references Process is a pure virtual method that must be implemented by concrete trackers Member variables hold pointers to cluster hitset vertex and track maps Track map name can be configured via set track map name method This comment was generated by openai/gpt-oss-120b at temperature 0.1.
