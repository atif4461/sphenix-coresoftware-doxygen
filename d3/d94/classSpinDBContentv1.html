Detailed Description

brief Container for spin database version one data

Provides getters and setters for run number qa level fill number bad run flag crossing shift Stores polarization values and errors for blue and yellow beams Stores spin pattern codes for both beams Stores scaler counts for mbd vertex cut mbd no cut zdc no cut Stores asymmetry and phase measurements with uncertainties for forward and backward directions of both beams Implements the SpinDBContent abstract interface This comment was generated by openai/gpt-oss-120b at temperature 0.1.

◆ InitializeV1()

void SpinDBContentv1::InitializeV1 ( )

Initialize all SpinDBContentv1 data members to sentinel error values.

This method resets run number, QA level, fill number, bad run flag, and crossing shift to the value returned by GetErrorValue(). It then loops over all beam crossings (0 … GetNCrossing()-1) and initializes the polarization, polarization error, systematic polarization, yield polarization, yield error, systematic yield, bunch pattern, yield pattern, scaler counts, and bad bunch flag for each crossing using GetErrorValue(). Finally, it sets the crossing angle parameters, asymmetry values, asymmetry errors, phase values, and phase errors to the error sentinel.
