Detailed Description

Handles conversion of Geant4 material definitions to GDML.

Collects isotopes elements and materials and writes them into a GDML materials element using Xerces DOM API Provides helper functions to write atomic mass density pressure temperature mean excitation energy and material properties Supports both modern and legacy Geant4 property vector types based on version macro This comment was generated by openai/gpt-oss-120b at temperature 0.1.
