Detailed Description

Module for estimating tower based background density (rho) in jet analyses.

DetermineTowerRho inherits from SubsysReco and implements the standard Fun4All callbacks InitRun and process_event. The module is configurable through a set of public methods that control the rho calculation method, input sources, jet clustering algorithm, and selection parameters.

Configuration interface

  • add_method(TowerRho::Method method, std::string node) : register a rho calculation method and optional output node name.
  • add_input(JetInput *input) : add a generic JetInput source for background estimation (add_tower_input provided for backward compatibility).
  • set_algo(Jet::ALGO algo) / get_algo() : select and query the jet algorithm used to cluster background jets (default KT).
  • set_par(float value) / get_par() : set and query the jet algorithm radius parameter (default 0.4).
  • set_abs_eta(float eta) / get_abs_eta() : define the absolute eta acceptance for tower inputs (default 1.1, with tower specific aliases for compatibility).
  • set_jet_abs_eta(float eta) / get_jet_abs_eta() : define the absolute eta acceptance for jet inputs.
  • set_omit_nhardest(unsigned int n) / get_omit_nhardest() : number of hardest towers to exclude from the rho calculation (default 2).
  • set_ghost_area(float area) / get_ghost_area() : ghost area used in the FastJet background estimator (default 0.01).
  • set_jet_min_pT(float pt) / get_jet_min_pT() : minimum jet transverse momentum for jets entering the background estimate (default off).
  • print_settings(std::ostream &os) : dump the current configuration to the provided output stream.

Internal functionality

  • CreateNodes(PHCompositeNode *topNode) : creates required output nodes in the Fun4All node tree.
  • CalcPercentile(const std::vector<float> &sorted, float percentile, float nempty) : static helper to compute a percentile value from a sorted vector.
  • CalcMedianStd(const std::vector<float> &vec, float n_empty_jets, float &median, float &std_dev) : static helper to compute median and standard deviation of a distribution.
  • get_jet_selector() const : builds a FastJet selector based on the configured eta ranges and jet pT cut.

The class stores configuration parameters as private members and maintains vectors of input JetInput pointers, output node names, and selected rho methods. It uses a sentinel value VOID_CUT to indicate disabled cuts.
