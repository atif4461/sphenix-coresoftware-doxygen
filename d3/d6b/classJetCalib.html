Detailed Description

JetCalib class provides calibration of reconstructed jets within the Fun4All framework It inherits from SubsysReco and operates on input and output node collections The module reads calibration functions from the conditions database and applies optional vertex and eta dependent corrections to jet transverse momentum Configuration methods allow setting input node output node vertex node jet radius and toggling the application of vertex and eta dependent calibrations This comment was generated by openai/gpt-oss-120b at temperature 0.1.
