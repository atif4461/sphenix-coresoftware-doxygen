◆ isEmpty()

bool Fun4AllHistoManager::isEmpty ( ) const

Determine whether all histograms managed by this instance are empty.

The method iterates over the internal histogram map and checks each entry. For objects derived from TH1 it evaluates both the number of entries and the integral; for THnBase objects it checks the number of entries. If any histogram contains entries or a non‑zero integral the function reports the histogram name and returns false. Histograms that are not TH1 or THnBase are reported as such but do not affect the emptiness test.

Returns
true if no histogram contains entries (or a non‑zero integral); false otherwise.
