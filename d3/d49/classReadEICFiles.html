Detailed Description

ReadEICFiles module integrates EIC input files into the Fun4All node tree.

This SubsysReco derived class opens a ROOT file containing an eic‑smear TChain, retrieves the event tree, creates a node on the PHCompositeNode hierarchy and provides the generated event record to downstream modules. It also supports selection of the first entry and custom node naming.
