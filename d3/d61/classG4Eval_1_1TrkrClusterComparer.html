Detailed Description

Utility for comparing truth and reconstructed tracker clusters.

Provides methods to initialize access to cluster containers in the node tree Retrieves cluster locations for PHG4 and SVTX clusters and evaluates matching criteria based on phi and z extents The functor operator returns a pair indicating whether two clusters match and the associated distance metric Configurable width parameters control the matching tolerances

Public members store the most recent comparison state including positions sizes deltas and detector region flags Geometry information such as phi step sizes and z step sizes is obtained from the ActsGeometry object during initialization

Typical usage construct with optional width parameters call init with the top node and container names then invoke the functor with cluster keys to obtain a match result This comment was generated by openai/gpt-oss-120b at temperature 0.1.
