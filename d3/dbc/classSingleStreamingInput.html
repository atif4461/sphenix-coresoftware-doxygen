void SingleStreamingInput::ClearCurrentEvent ( )
virtual

Clear the current event data from the streaming input.

This function is called interactively to discard the event associated with the most recent beam clock. It logs the beam clock identifier, releases any packet resources tied to that beam clock, and removes the beam clock entry from the internal bookkeeping structures.

Returns

inline

Clear entries from the FeeGTML1BCOMap that are older than or equal to the specified bunch clock.

Parameters
bclkReference bunch clock value used as the cutoff threshold.

The function iterates over each key‑set pair in the internal m_FeeGTML1BCOMap. For each set it removes all elements whose value is less than or equal to bclk, ensuring that only more recent entries remain in the map. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
