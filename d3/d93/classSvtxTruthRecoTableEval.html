Detailed Description

Class SvtxTruthRecoTableEval implements an analysis module for evaluating truth and reconstructed particle maps Inherits from SubsysReco and integrates with the Fun4All node tree Provides framework callbacks Init InitRun process_event ResetEvent and End Allows configuration of a minimum momentum threshold for entries in the truth map via setMinMomentumTruthMap Manages creation of required nodes and filling of truth and reco maps Member variables include a flag for scanning primaries a momentum threshold a pointer to the truth map a pointer to the reco map and a unique pointer to an evaluation stack This comment was generated by openai/gpt-oss-120b at temperature 0.1.
