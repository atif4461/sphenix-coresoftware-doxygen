Detailed Description

===========================================================================

Jet histogram manager for TrksInJetQA module

This histogram manager defines what to histogram from jets. Manager for jet level track QA histograms Provides enumeration of jet types and histogram identifiers and fills jet histograms based on provided jet and optional track collection
