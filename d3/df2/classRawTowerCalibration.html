Detailed Description

calibrate ADC value to measured energy deposition in calorimeter towers default input DST node is TOWER_RAW_DETECTOR default output DST node is TOWER_CALIB_DETECTOR

int RawTowerCalibration::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize the module for a new run.

This method is invoked by Fun4All at the beginning of each run. It searches the node tree for the DST composite node, creates the necessary analysis nodes, and validates the selected calibration algorithm. If the DST node cannot be found or an unsupported algorithm is chosen, the run is aborted.

Parameters
topNodePointer to the top-level PHCompositeNode of the current Fun4All job.
Returns
Fun4AllReturnCodes::EVENT_OK on successful initialization, Fun4AllReturnCodes::ABORTRUN if the DST node is missing, an unsupported calibration algorithm is selected, or an exception occurs while creating nodes. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
