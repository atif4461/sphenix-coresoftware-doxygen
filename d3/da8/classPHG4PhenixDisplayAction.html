Detailed Description

PHG4PhenixDisplayAction provides a display action for Phenix geometry in the Fun4All framework It maintains a map from logical volumes to material names and a collection of visualization attributes The constructor takes a name identifier for the action The destructor releases allocated visualization attributes ApplyDisplayAction is called for each physical volume during the Geant4 visualization pass AddVolume records the material name associated with a logical volume This comment was generated by openai/gpt-oss-120b at temperature 0.1.
