◆ SetInterfacePointers()

void PHG4OuterHcalSteppingAction::SetInterfacePointers ( PHCompositeNode * topNode)
overridevirtual

reimplemented from base class

Locate and assign hit containers for the outer HCal detector.

This method constructs the node names for the hit and absorber hit containers based on the detector's super detector name (if defined) or its own name. It then searches the supplied top node for these containers using the Fun4All findNode utility and stores the resulting pointers in the member variables m_Hits and m_AbsorberHits. If a container cannot be found, an error message is printed; a verbose message is printed for a missing absorber hit container when the verbosity level exceeds one.

Parameters
topNodePointer to the top PHCompositeNode of the current Fun4All node tree.
