Detailed Description

Container for trigger primitives used in sPHENIX analysis framework.

Provides map based storage keyed by TriggerDefs TriggerPrimKey and methods for insertion retrieval and iteration over stored primitives Implements PHObject interface with Reset identify and validity checks Supports retrieval of full primitive range for const and non const contexts This comment was generated by openai/gpt-oss-120b at temperature 0.1.
