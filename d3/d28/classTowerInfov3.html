Detailed Description

TowerInfov3 extends TowerInfov2 to include waveform data.

Provides methods to reset and clear the object preserving base class behavior Implements accessors for number of samples waveform values and mutators Supports copying of tower information from generic TowerInfo instances Stores a fixed size waveform array of 31 samples initialized to zero
