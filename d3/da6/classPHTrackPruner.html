Detailed Description

PHTrackPruner is a SubsysReco module that prunes svtx tracks based on configurable quality cuts The module retrieves seed containers and track maps from the node tree and applies pt and cluster count thresholds for mvtx intt tpc and tpot detectors User can set names of input and output containers and adjust cut values through setter methods The module implements InitRun process_event and End callbacks required by Fun4All This comment was generated by openai/gpt-oss-120b at temperature 0.1.
