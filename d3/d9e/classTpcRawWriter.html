◆ process_event()

int TpcRawWriter::process_event ( PHCompositeNode * topNode)
overridevirtual

Process a single event for the TpcRawWriter module.

This method is invoked by the Fun4All framework for each event. It retrieves digitized hit containers for the MVTX, INTT, Micromegas and TPC subsystems from the supplied top node, converts the hits into raw hit representations and stores them in the corresponding raw hit containers under the DST node. Geometry information and tracking geometry are also accessed to correctly map TPC pads and time bins. The method performs consistency checks on the packed TPC data and reports any mismatches. If any required node is missing the method aborts the run; otherwise it returns a success code.

Parameters
topNodePointer to the top PHCompositeNode of the current processing chain. The node tree contains the input digitized hit containers, geometry containers and the destination raw hit containers.
Returns
Fun4AllReturnCodes::EVENT_OK on successful processing of the event, or Fun4AllReturnCodes::ABORTRUN if a required node cannot be found. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
