Detailed Description

CosmicSpray class generates cosmic ray particles for simulation InDetector method returns true if a point is inside the detector volume defined by radius R Constructor allows optional module name and detector radius with default values set_gen_min_momentum configures the generator minimum momentum set_gen_max_momentum configures the generator maximum momentum InitRun and process_event are framework callbacks invoked during run initialization and event processing This comment was generated by openai/gpt-oss-120b at temperature 0.1.
