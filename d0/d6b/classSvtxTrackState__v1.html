Detailed Description

SvtxTrackState_v1 implements the SvtxTrackState interface for a single track state It stores path length position momentum and covariance matrix in a compact triangular format Provides getters and setters for Cartesian coordinates momentum and derived kinematic quantities Supports standard PHObject methods identify Reset CloneMe and validity flag Name attribute can be used to label the state

SvtxTrackState_v1::SvtxTrackState_v1 ( float pathlength = 0.0)

Construct a SvtxTrackState_v1 with a given path length.

Parameters
pathlengthPath length along the track at which the state is defined.

Initializes all position components to zero, all momentum components to NaN, and sets every element of the symmetric error matrix to zero. The state name is initialized to UNKNOWN to indicate an undefined track state.
