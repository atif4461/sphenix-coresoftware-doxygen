◆ free_all()

void nanoflann::PooledAllocator::free_all ( )
inline

Release all memory blocks managed by the allocator and reset its state.

Frees all allocated memory chunks

The routine walks the linked list of allocated blocks starting from the global base pointer. For each block it retrieves the pointer to the previous block, frees the current block and moves to the previous one. After the list is empty the internal initialization function is called to reestablish a clean allocator state for subsequent allocations.
