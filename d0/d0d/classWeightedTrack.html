Eigen::Affine3d const & plane ) const

Returns the derivatives of the residuals w.r.t. the parameter given by index for the track intersection with the given plane.

Compute the derivative of the measurement residual with respect to a track parameter.

Parameters
indexInteger index of the track parameter for which the derivative is evaluated.
planeAffine transformation defining the measurement plane.
Returns
2‑dimensional vector containing the residual derivative projected onto the measurement plane.

The function determines the path length at which the track intersects the given plane, obtains the partial derivative of the track state with respect to the specified parameter at that path length, and projects this derivative onto the measurement plane.
