Add the contents of another space charge matrix container to this container.

The method first verifies that the grid dimensions phi bins r bins and z bins of the supplied container match those of the current object If the dimensions are inconsistent the operation aborts and returns false When the dimensions agree the cell entry counts left hand side matrices and right hand side vectors are incremented elementwise with the values from the argument
