Detailed Description

Fun4AllRunNodeInputManager handles input of run level node data from DST files It derives from Fun4AllDstInputManager and overrides file opening and event loop methods The manager disables synchronization checks and provides stub implementations for sync related interfaces It operates on a specified node name within a top node hierarchy
