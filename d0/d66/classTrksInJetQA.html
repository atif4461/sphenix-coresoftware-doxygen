Detailed Description

============================================================================

Tracks-in-jet QA

This Fun4All module generates histograms to QA low-level info of tracks found inside a jet, as well as some kinematic information.

void TrksInJetQA::SetTrgToSelect ( const uint32_t trig = JetQADefs::GL1::MBDNSJet1)
inline

Enable trigger based event selection and set the trigger identifier.

This method activates the internal trigger selection flag and stores the specified trigger code. The default argument selects the MBDNSJet1 trigger defined in the JetQADefs GL1 namespace.

Parameters
trigTrigger identifier to be selected (default is JetQADefs::GL1::MBDNSJet1).
