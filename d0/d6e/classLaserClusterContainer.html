Detailed Description

Cluster container object.

LaserClusterContainer provides an abstract interface for managing LaserCluster objects within the sPHENIX reconstruction framework It inherits from PHObject and defines type aliases for convenient map based storage The class declares virtual methods for adding removing and querying clusters as well as size information Concrete implementations must provide storage and retrieval semantics
