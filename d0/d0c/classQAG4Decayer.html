◆ process_event()

int QAG4Decayer::process_event ( PHCompositeNode * topNode)
overridevirtual

Process a single event and fill heavy‑flavor QA histograms.

This method is invoked by the Fun4All framework for each event. It retrieves the histogram manager, accesses the G4 truth information node, loops over all generated particles, identifies heavy‑flavor vertices, computes kinematic deviations, proper decay times, and invariant masses, and fills the corresponding QA histograms. Branching fractions are also filled based on the decay channel mapping. Optionally, an ntuple is filled if m_write_nTuple is enabled. The event counter is incremented before returning.

Parameters
topNodePointer to the top PHCompositeNode of the current Fun4All job.
Returns
Integer status code; returns Fun4AllReturnCodes::EVENT_OK on successful processing of the event.
