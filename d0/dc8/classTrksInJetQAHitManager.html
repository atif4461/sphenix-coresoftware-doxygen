Detailed Description

===========================================================================

Tracker hit histogram manager for TrksInJetQA module

This histogram manager defines what to histogram from tracker hits. TrksInJetQAHitManager manages hit level quality assurance histograms for tracks in jets Inherits from TrksInJetQABaseManager Provides enums for subsystem type and histogram identifiers Defines struct HitQAContent to store energy adc layer phiBin zBin Public method GetInfo extracts hit information into HitQAContent Private method FillHistograms fills histograms based on subsystem type Overrides DefineHistograms to create histogram definitions
