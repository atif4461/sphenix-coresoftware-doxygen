Detailed Description

Cluster CDF Calculator provides methods to load probability profiles from ROOT files and evaluate cumulative distribution functions for electromagnetic clusters The class manages histograms for photon and pi0 templates for different matrix sizes and stores precomputed inverse covariance matrices and mean deviations Public interface includes loading a profile file setting verbosity and computing the CDF for a given set of cell energies and cluster energy Internal helpers load histograms matrices compute joint deviation and retrieve CDF values from template histograms Constants define matrix dimensions and grid sizes used in the analysis This comment was generated by openai/gpt-oss-120b at temperature 0.1.
