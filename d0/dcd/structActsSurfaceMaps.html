Detailed Description

ActsSurfaceMaps provides mapping between detector hitsets and Acts surfaces Constructor default isTpcSurface returns true if surface belongs to TPC isMicromegasSurface returns true if surface belongs to Micromegas getSurface returns surface for given cluster key and pointer getSiliconSurface returns surface for silicon detector hitset getTpcSurface returns surface for TPC hitset and subsurface key getMMSurface returns surface for Micromegas hitset m_siliconSurfaceMap maps hitset to surface for MVTX and INTT m_tpcSurfaceMap maps layer to vector of surfaces for TPC m_mmSurfaceMap maps hitset to surface for Micromegas m_tGeoNodeMap maps hitset to TGeoNode pointer m_tpcVolumeIds stores Acts volume identifiers for TPC m_micromegasVolumeIds stores Acts volume identifiers for Micromegas This comment was generated by openai/gpt-oss-120b at temperature 0.1.
