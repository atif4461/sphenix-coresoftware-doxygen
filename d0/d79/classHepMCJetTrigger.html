Detailed Description

HepMCJetTrigger class inherits from SubsysReco and implements a jet based trigger using HepMC truth information The module selects events containing at least a configurable number of jets above a transverse momentum threshold Configuration parameters include trigger threshold number of required jets and optional hard event limit Internal counters track total processed events and number of events passing the trigger The class provides methods to evaluate event quality reconstruct jets with fastjet and count jets above threshold
