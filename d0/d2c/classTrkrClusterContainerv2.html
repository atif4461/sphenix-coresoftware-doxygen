Detailed Description

Cluster container object.

TrkrClusterContainerv2 implements version 2 of TrkrClusterContainer with fixed size three dimensional map Provides reset identify addClusterSpecifyKey removeCluster getClusters findCluster and size methods Uses private constants max_layer max_phisegment max_zsegment and array of Map m_clusmap
