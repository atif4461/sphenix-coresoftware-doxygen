Detailed Description

Fun4All output manager that writes HepMC generated events to an ASCII file.

This class derives from Fun4AllOutputManager and provides functionality to open an output stream, write HepMC::GenEvent objects, embed optional comments, and tag events with an embedding identifier. The embedding identifier distinguishes primary collisions (ID = 0), signal sub‑events (positive ID) and background or out‑of‑time pile‑up (negative ID). The manager also supports optional compression of the output file via a filtered stream.

Parameters
mynameName of the output manager instance; defaults to "HEPMCOUT".
filenameName of the output file; defaults to "hepmcout.txt".
Returns
None.
Note
The outfileopen method is overridden to perform no action because the file is opened lazily when Write is called.
See also
Fun4AllOutputManager for the base class interface.
