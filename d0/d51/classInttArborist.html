Detailed Description

InttArborist module extracts raw INTTRawHit and GL1RawHit data from the node tree and stores them in a ROOT TTree The module creates an output ROOT file defines branches for small and large data types and fills the tree for each event InitRun locates the input nodes creates the output file and tree and registers branch pointers process_event reads the raw hit containers clones the data into branch vectors and fills the tree EndRun writes and closes the ROOT file Small branches store per hit values using a short integer type large branches store per event scalars using a 64 bit integer Internal maps associate branch names with branch storage vectors and scalar values This comment was generated by openai/gpt-oss-120b at temperature 0.1.
