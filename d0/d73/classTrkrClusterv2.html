float TrkrClusterv2::getPhiSize ( ) const
overridevirtual

Compute the azimuthal (phi) size of the cluster.

The phi size is derived from the underlying size attribute using the rotate helper which projects the size onto the azimuthal direction. The result is scaled by a factor of two and the square root is taken to obtain the full phi extent.

Returns
Phi size of the cluster. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
