Detailed Description

Implementation of TriggerRunInfo version one.

Stores up to sixty four trigger entries each with a name a bit an initial prescale a current prescale and three scaler values Provides setters for trigger definition prescale and scaler values Provides getters for prescale scaler and trigger bit lookup by name or by bit value Implements identify method for stream output
