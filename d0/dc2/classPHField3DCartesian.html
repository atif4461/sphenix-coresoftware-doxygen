Detailed Description

Class implements a three dimensional Cartesian magnetic field for Geant4 Constructor loads field map from file and applies optional scaling and geometry limits Destructor releases resources GetFieldValue returns magnetic field vector at given spacetime point following G4ElectroMagneticField convention GetFieldValue_nocache provides same functionality without internal caching Private data includes filename domain bounds step sizes cached field values hit statistics and containers for map lookup

Retrieve magnetic field vector at a given point using trilinear interpolation of a Cartesian field map.

access field value Follow the convention of G4ElectroMagneticField

Parameters
[in]Pointspace time coordinate. x, y, z, t in Geant4/CLHEP units
[out]Bfieldfield value. In the case of magnetic field, the order is Bx, By, Bz in in Geant4/CLHEP units
pointArray of four doubles containing the spatial coordinates x y z (fourth element unused).
BfieldOutput array of three doubles that will be filled with the magnetic field components Bx By Bz in Tesla.
Returns
Void. The Bfield array is modified in place. Zero field is returned for points outside the map or for invalid coordinates.
