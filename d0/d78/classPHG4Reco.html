◆ InitRun()

int PHG4Reco::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize the PHG4Reco module for a new run.

This method is invoked once per run to configure the Geant4 environment, including world material, global magnetic field, detector geometry, user actions, and optical processes. It also initializes all registered subsystems, sets up the input event node reader, optionally exports geometry to DST or a file, and prepares the random number store based on verbosity settings.

Parameters
topNodePointer to the top PHCompositeNode of the current Fun4All node tree.
Returns
