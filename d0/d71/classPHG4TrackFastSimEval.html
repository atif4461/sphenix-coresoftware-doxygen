overridevirtual

Write evaluation trees and histograms to the output ROOT file and return status.

This method changes to the output file directory, writes the track and vertex evaluation TTrees, and writes the two 2‑D histograms that compare reconstructed momentum differences versus truth η and momentum. The ROOT file is left open (closing is handled elsewhere). The input topNode pointer is unused.
