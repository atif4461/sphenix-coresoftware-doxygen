Detailed Description

Digitizer module for Micromegas detector in sPHENIX simulation framework Implements SubsysReco interface and PHParameterInterface for run initialization event processing and parameter handling Provides methods to add noise threshold pedestal conversion factors and random number generation

Functor used to free gsl random number generator resources Compatible with unique_ptr custom deleter semantics This comment was generated by openai/gpt-oss-120b at temperature 0.1.
