Detailed Description

PHG4GenHit module generates synthetic Geant4 hits for a specified detector element It stores kinematic parameters phi theta eloss and layer index The module is invoked for each event via process_event Configuration setters allow external control of phi theta eloss layer and detector name This comment was generated by openai/gpt-oss-120b at temperature 0.1.
