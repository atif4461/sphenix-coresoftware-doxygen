◆ InitRun()

int PHG4TpcDirectLaser::InitRun ( PHCompositeNode * topNode)
overridevirtual

run initialization

Initialize run‑level resources for the direct laser simulation.

This method is invoked at the beginning of each new run. It ensures that the G4TruthInfo container exists in the DST node, locates the appropriate G4Hit container for the configured detector, and creates or retrieves the SvtxTrackMap used to store per‑event laser parameters. Geometry information is obtained from the TPC geometry container to compute drift length and half width. Laser parameters are loaded from a ROOT file and the internal laser setup is performed. Configuration values such as stepping pattern, number of steps, and electron production rates are printed for verification.

Parameters
topNodePointer to the top PHCompositeNode of the current node tree.
Returns

void PHG4TpcDirectLaser::SetDefaultParameters ( )
overridevirtual

default parameters

Initialize default simulation parameters for the TPC direct laser detector.

Sets gas mixture properties based on a 50/50 Ne/CF4 mixture at 20 C and 760 Torr, including energy loss per unit length, total number density, and derived electrons per keV. The function configures the default number of electrons per GeV of deposited energy and the default number of electrons generated per centimeter by the laser.
