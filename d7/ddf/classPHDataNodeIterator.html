Detailed Description

A special PHOOL node iterator that simplifies finding and adding data nodes.

The methods are templated, rather than the class, to allow the same iterator object to be used with different data node types.

Author
Kyle Pope

Adds a data node called "name" to the tree, and inserts "data". The data node is added at the current "directory" of this iterator object, so remember to "cd" to the desired location in the tree!

PHDataNodeIterator provides functionality to manage a collection of data nodes within the Fun4All node tree It supports adding input/output data nodes and maintains internal bookkeeping for node ownership

PHIODataNode is a templated node type that stores a pointer to user data together with an associated name It is used for objects that can be written to a DST and participates in the Fun4All node hierarchy
