Detailed Description

Class PHG4ParticleGeneratorVectorMeson Generates vector meson particles with configurable decay channels and vertex distributions Supports uniform and gaussian sampling for vertex and size Provides interfaces to set kinematic ranges eta rapidity momentum pt and mass width Allows adding decay particles by name and specifying decay vertex offsets Integrates with PHG4InEvent to produce G4 hits in Fun4All framework
