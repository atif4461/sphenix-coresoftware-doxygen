Detailed Description

Fun4AllDstInputManager handles DST input files within the Fun4All framework It derives from Fun4AllInputManager and provides methods for opening closing files event looping synchronization branch selection and caching The class manages node trees for run information and synchronization objects and interfaces with PHNodeIOManager for IO
