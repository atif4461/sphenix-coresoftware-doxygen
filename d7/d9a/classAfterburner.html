Detailed Description

Afterburner provides an interface to apply flow afterburner algorithms to HepMC events It manages algorithm selection random engine ownership and kinematic ranges The class supports legacy setters and a generic flowAfterburner entry point Reaction plane angles psi n can be set and retrieved for harmonics up to six Internal state includes algorithm pointer random engine pointer eta pt limits and plane shift

◆ vn_func()

double Afterburner::vn_func ( double x,
void * params )
static

Evaluate the flow-modified azimuthal function for a given angle.

This routine computes the residual between the input angle x (plus the contribution from harmonic flow terms) and the reference angle phi0. The harmonic coefficients vn and phase angles psi are taken from the user‑supplied parameter block. The function is typically used as a callback for root‑finding or minimisation algorithms.

Parameters
xAzimuthal angle at which the function is evaluated (radians).
paramsPointer to a contiguous block of floats containing:
  • element 0 : reference angle phi0,
  • elements 1‑6 : harmonic coefficients vn[0] … vn[5],
  • elements 7‑12 : phase angles psi[0] … psi[5]. The array must contain at least 13 elements.
Returns
The value (x + 2 * sum_{n=1}^{6} vn[n-1] * sin(n*(x-psi[n-1]))/n) - phi0.
