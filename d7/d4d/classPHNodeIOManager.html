Detailed Description

brief Manages I/O of PHCompositeNode trees using ROOT files

Provides methods to open read write and close ROOT files containing PHCompositeNode hierarchies Supports selective object reading branch mapping compression settings cache control and buffer split level configuration Integrates with Fun4All node tree framework and inherits from PHIOManager

note Ownership of ROOT objects is transferred to Fun4All node tree the manager does not delete nodes after insertion

see PHIOManager for base I/O interface This comment was generated by openai/gpt-oss-120b at temperature 0.1.
