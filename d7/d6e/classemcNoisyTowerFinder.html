Detailed Description

Brief description of a SubsysReco module that identifies noisy towers in calorimeter data The module can be configured with energy thresholds in ADC counts and in GeV Geometry can be set for inner hadronic calorimeter or outer hadronic calorimeter Provides methods to initialize run process event end and to fill histograms calculate cutoffs write CDB tree and find hot towers Includes static utility to compute median of a vector of floats This comment was generated by openai/gpt-oss-120b at temperature 0.1.
