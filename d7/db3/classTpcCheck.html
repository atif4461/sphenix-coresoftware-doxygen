Detailed Description

TpcCheck analysis module for TPC raw hit validation This module inherits from SubsysReco and implements Init and process_event to access TPC raw hit node and perform checks The module allows configuration of event node name and BCO range It tracks seen bunch crossing clocks to avoid duplicate processing name optional module name default TpcCheck topNode pointer to top node of the Fun4All node tree integer status code returned This comment was generated by openai/gpt-oss-120b at temperature 0.1.
