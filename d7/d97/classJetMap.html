Detailed Description

JetMap class provides a container for Jet objects and associated source identifiers It defines iterator types for map traversal and source set traversal The interface includes methods to query and modify algorithm type and parameter to manage source identifier collection and to access insert erase and iterate over stored Jet instances Pure virtual method vec returns a vector of Jet pointers for sorted access
