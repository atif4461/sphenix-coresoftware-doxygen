double PHGenFit::Track::extrapolateToLine ( genfit::MeasuredStateOnPlane & state,
const TVector3 & line_point,
const TVector3 & line_direction,
const int tr_point_id = 0 ) const

Extrapolate the track to a specified line and obtain the measured state on that plane.

This method retrieves the track point with measurement and fitter information for the given point identifier, constructs a Kalman‑fitted state, and extrapolates it to the line defined by a point and a direction vector. The resulting state is returned via the reference argument and the path length along the track to the line is returned. If the required track point is missing or an exception occurs during extrapolation, a sentinel value (WILD_DOUBLE) is returned.

Parameters
stateReference to a MeasuredStateOnPlane object that will be filled with the extrapolated state.
line_point3‑vector specifying a point on the target line.
line_direction3‑vector specifying the direction of the target line (must be normalized).
tr_point_idIdentifier of the track point to be used for the extrapolation.
Returns
Path length from the original track point to the line, or WILD_DOUBLE on failure. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

double PHGenFit::Track::extrapolateToPlane ( genfit::MeasuredStateOnPlane & state,
const TVector3 & O,
const TVector3 & n,
const int tr_point_id = 0 ) const

Extrapolate a track to a specified detector plane using the Kalman filter.

track_point 0 is the first one, and -1 is the last one

This routine retrieves the track representation and the track point identified by tr_point_id, constructs a Kalman fitted state, and propagates it to a destination plane defined by the point O and normal vector n. The extrapolated measured state is returned via state and the path length travelled during the extrapolation is returned as the function value. If the required track point or fitter information is missing, or if the extrapolation throws an exception, the function returns the sentinel value WILD_DOUBLE.

Parameters
stateOutput measured state on the destination plane after extrapolation.
OOrigin point of the destination plane.
nNormal vector of the destination plane.
tr_point_idIdentifier of the track point containing measurement and fitter info.
Returns
Path length of the extrapolation or WILD_DOUBLE on failure. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
