Detailed Description

PHG4TpcGeomContainer is a container for PHG4TpcGeom objects indexed by integer layer identifiers It inherits from PHObject to allow integration with the Fun4All node tree The class provides methods to add a geometry object for a specific layer or automatically assign the next available layer Retrieval methods return a pointer to the geometry for a given layer or the first geometry in the container Iteration support is provided through ConstRange returning begin and end iterators over the internal map The number of stored layers can be obtained with get_NLayers This comment was generated by openai/gpt-oss-120b at temperature 0.1.
