int EventPlaneReco::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run‑time resources and build the trigonometric cache for sEPD channels.

Called during initialization. geometry is available

This method retrieves the EpdGeom geometry node, allocates a cache of cosine and sine values for each harmonic and each sEPD channel, and stores the pre‑computed trigonometric factors used later in event‑plane reconstruction.
