Maps status codes onto labels.

Returns a constant reference to a map linking Stat enumeration values to their string labels.

This inline function creates a static std::map that associates each Stat enum entry (Good, Hot, BadChi, NotInstr, NoCalib, Unknown) with a human‑readable label. The map is initialized on first invocation and then reused, providing efficient access to descriptive names for status codes throughout the framework.
