Detailed Description

Display action for the INTT detector geometry.

Implements the PHG4DisplayAction interface to assign visual attributes to logical volumes belonging to the INTT subsystem. The constructor registers the action with a user supplied name. The ApplyDisplayAction method is invoked by the Geant4 visualization manager for each physical volume; it looks up the material name associated with the logical volume and applies a corresponding visual attribute. Volumes can be added to the internal map via AddVolume. The class maintains a map from logical volume pointers to material names and a vector of allocated G4VisAttributes objects for lifetime management.
