Detailed Description

template<int H, int F, int S, int MaxE, int MinE, int dE>
struct CaloStatusMapperDefs::HistDef< H, F, S, MaxE, MinE, dE >

Helper struct to define histograms.

This is a lightweight struct to organize the axis definitions for status, eta, phi axes and make histograms using them.
