Detailed Description

Jet trigger based on Pythia8 jet properties.

Provides configuration of eta acceptance minimum jet transverse momentum jet radius parameter minimum leading particle momentum fraction and minimum number of jet constituents Implements Apply to evaluate the trigger condition on a Pythia8 event Includes utility to print the current configuration This comment was generated by openai/gpt-oss-120b at temperature 0.1.
