Detailed Description

Jet class representing a reconstructed jet in the sPHENIX framework Provides accessors for kinematic variables and jet properties Supports multiple source types and clustering algorithms via ALGO and SRC enums Implements PHObject methods for identification and cloning

IterJetTCA struct provides iterator functionality over jets stored in a TClonesArray Allows increment and dereference to obtain Jet pointers This comment was generated by openai/gpt-oss-120b at temperature 0.1.
