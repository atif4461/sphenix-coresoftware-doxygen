Detailed Description

Provides functionality to compare truth and reconstructed clusters based on configurable tolerances in phi z and time The init method retrieves cluster containers from the node tree Tolerance setters allow per detector configuration Operator returns true when clusters match according to selected criteria Internal state stores last comparison details This comment was generated by openai/gpt-oss-120b at temperature 0.1.
