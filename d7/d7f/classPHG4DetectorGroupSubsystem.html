void PHG4DetectorGroupSubsystem::BlackHole ( const int detid,
const int i )

Enable or disable the blackhole mode for a specific detector.

This method sets the integer parameter named "blackhole" for the detector identified by detid to the value i. When enabled, the detector absorbs all particles without generating hits, effectively acting as a blackhole in the simulation.

Parameters
detidIdentifier of the detector for which the blackhole mode is set.
iInteger flag (typically 0 to disable, 1 to enable) for blackhole mode.
