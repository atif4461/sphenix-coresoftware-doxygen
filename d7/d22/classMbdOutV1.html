Detailed Description

Implementation of the MbdOut interface version 1.

Provides storage for reconstructed vertex position and its error event time zero and its error and charge and timing information for the north and south arms of the MBD detector

Supports resetting the object identification output validity check cloning and copying and accessor mutator methods for all stored fields
