◆ get_ring_qvector()

std::pair< double, double > Eventplaneinfov2::get_ring_qvector ( int ring_index,
unsigned int order ) const
inlineoverridevirtual

Retrieve the Q‑vector for a given detector ring and harmonic order.

This function checks that the requested ring_index is within the valid range of the internal ring_Qvec container. If the index is out of range a pair of NaN values is returned. Otherwise the stored Q‑vector for the specified ring is passed to safe_qvec which applies the requested harmonic order and returns the calibrated components.

Parameters
ring_indexZero‑based index identifying the detector ring.
orderHarmonic order of the Q‑vector to be retrieved.
Returns
A std::pair where the first element is the Qx component and the second element is the Qy component; both are NaN if the ring index is invalid.
