Detailed Description

Reconstruction module for the MBD detector integrated with the Fun4All framework Provides initialization run processing and finalization hooks Configurable options include fit only mode calibration pass charge processing trigger selection Manages creation and retrieval of node tree objects such as event packets geometry and vertex map
