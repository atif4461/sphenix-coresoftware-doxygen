Detailed Description

TH2INTT provides a TH2Poly based histogram for INTT geometry It stores content indexed by server and ladder identifiers and draws geometry lines for ladders Public methods allow setting and retrieving scalar content using string or integer based keys The Draw method overrides TH2Poly drawing to include ladder lines Private helpers construct ladder line objects and fill internal position maps based on detector geometry parameters
