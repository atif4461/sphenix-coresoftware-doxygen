Detailed Description

Base class for detector cell objects Provides storage for energy deposition maps from Geant4 hits Supports addition of ionization energy and scintillation light yield Enumerates property identifiers for geometric indices and energy quantities Supplies virtual getters and setters for generic property access Implements interface required by PHObject base class
