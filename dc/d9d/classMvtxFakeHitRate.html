Detailed Description

MvtxFakeHitRate module calculates fake hit rate for MVTX detector It inherits from SubsysReco and implements standard Fun4All callbacks InitRun prepares geometry and hot pixel mask process_event extracts raw hit information and fills analysis trees End writes output ROOT file and releases resources SetOutputfile configures name of output file SetMaxMaskedPixels configures maximum number of masked pixels SelectLayer configures target layer for analysis StartFromCDB configures loading of hot pixel mask from CDB Private members store configuration state hit map containers raw event header raw hit container ROOT trees and histograms Helper functions retrieve node pointers FillCurrentMaskTree calculate fake hit rate and compute noise threshold This comment was generated by openai/gpt-oss-120b at temperature 0.1.
