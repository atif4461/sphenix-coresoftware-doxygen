Detailed Description

Fun4AllEventOutStream provides an interface for writing event data to an output stream It derives from Fun4AllBase and collaborates with Fun4AllEventOutputManager The class maintains an event buffer and a list of packet pointers It supports adding and dropping individual packets as well as packet ranges Buffer resizing and manager access are handled through protected methods Configuration flags control packet filtering behavior This comment was generated by openai/gpt-oss-120b at temperature 0.1.
