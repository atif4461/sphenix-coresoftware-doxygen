Detailed Description

TpcSimpleClusterizer clusters TPC hits into simple clusters within the Fun4All framework It inherits from SubsysReco and implements InitRun process event and End callbacks The module accesses hit cluster and association containers from the DST node and uses ActsGeometry for detector geometry Configuration options include a sector fiducial cut and optional hit to cluster association Internal utility tests sector boundaries using PHG4TpcGeom Pedestal subtraction and shaping offset corrections are applied using predefined parameters This comment was generated by openai/gpt-oss-120b at temperature 0.1.
