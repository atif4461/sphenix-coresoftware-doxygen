Detailed Description

PHCosmicSeedCombiner combines cosmic track seeds from silicon and TPC detectors It retrieves geometry cluster and seed containers from the node tree Seeds are merged when their phi and eta differences are below configurable cuts The module follows the SubsysReco interface used by Fun4All This comment was generated by openai/gpt-oss-120b at temperature 0.1.
