Detailed Description

TpcLaserQA analysis module for TPC laser data quality monitoring.

This module derives from SubsysReco and provides InitRun and process_event implementations to create and fill ROOT histograms that characterize laser event counts cluster multiplicities and saturation patterns in the two TPC wheels

Note
Histogram names are prefixed with a module specific string obtained from getHistoPrefix
Warning
The module expects the presence of TPC wheel nodes in the DST subtree and will operate on TH1 and TH2 objects managed by the ROOT framework This comment was generated by openai/gpt-oss-120b at temperature 0.1.
