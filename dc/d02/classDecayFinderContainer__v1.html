Detailed Description

DecayFinderContainer v1 stores decay objects indexed by unique identifier and provides map like interface for analysis modules It inherits from DecayFinderContainerBase and implements required virtual methods for cloning resetting and iteration The internal container is a standard map from unsigned int to Decay
