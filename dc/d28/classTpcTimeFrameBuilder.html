◆ fillBadFeeMap()

void TpcTimeFrameBuilder::fillBadFeeMap ( )

Fill the internal map of masked front‑end electronics (FEE) using the TPC decoder bad FEE calibration stored in the conditions database.

This routine queries the CDB for the entry named TPC_DECODER_BAD_FEE, loads the associated calibration tree, reads the number of masked FEE entries (N_MASKED_FEES) and populates the member container m_maskedFEEs with the pairs (EBDC, FEEID) for each masked channel.
