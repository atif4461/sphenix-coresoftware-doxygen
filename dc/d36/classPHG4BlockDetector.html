Detailed Description

PHG4BlockDetector implements a block detector within the Fun4All framework It derives from PHG4Detector and constructs the block geometry in the world volume The class provides methods to query whether a physical volume belongs to the block It stores a pointer to the parameter container a pointer to the display action The layer index and super detector name can be set and retrieved Constructor arguments are a pointer to the subsystem a pointer to the node tree a pointer to the parameters a detector name string and an optional layer number
