CLHEP::HepLorentzRotation PHHepMCGenEventv1::get_LorentzRotation_EvtGen2Lab ( ) const
finalvirtual

LorentzRotation to translate from hepmc event frame to lab frame.

Construct the Lorentz transformation from the EvtGen frame to the laboratory frame.

This method builds a CLHEP::HepLorentzRotation using the internally stored boost beta vector and the rotation axis and angle. The boost is applied first, followed by the spatial rotation.

Returns
A CLHEP::HepLorentzRotation representing the combined boost and rotation from the EvtGen coordinate system to the lab coordinate system. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
