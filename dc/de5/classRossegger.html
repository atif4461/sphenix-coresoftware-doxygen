This implementation follows the Rossegger formulation (equation 5.65) and evaluates the series expansion over the specified number of orders. Compute the Rossegger Rnk quantity for zero‑based indices.

This routine evaluates the Rossegger Equation 5.46 for a given integer index n (counted from zero) and a parameter mu. The calculation uses the class members pi, L, a, b, and the helper functions limu and kimu.

Parameters
nZero‑based integer index used to compute the mode number.
muContinuous parameter entering the Bessel‑type functions.
Returns
The evaluated value of Rnk according to limu(mu, betana) * kimu(mu, betanb) - kimu(mu, betana) * limu(mu, betanb).

◆ RPrime()

double Rossegger::RPrime ( int m,
int n,
double ref,
double r )

Compute the Rossegger R′ coefficient for given orders and radii.

This routine validates the input indices against the configured order limits and checks that the radial arguments lie within the interval [a,b]. If any check fails an error message is printed and a zero value is returned.

The calculation follows Rossegger Equation 5.65: Rmn2(ref,r) = (BetaN/2) * [ Km(BetaN·ref)·(Im‑1(BetaN·r)+Im+1(BetaN·r)) – Im(BetaN·ref)·(Km‑1(BetaN·r)+Km+1(BetaN·r)) ].

The implementation uses the ROOT TMath wrappers kn and in for the modified Bessel functions Kₘ and Iₘ, respectively.

Parameters
mOrder index for the Bessel functions (must satisfy 0 ≤ m < NumberOfOrders).
nIndex selecting the BetaN coefficient (must satisfy 0 ≤ n < NumberOfOrders).
refReference radius (must satisfy a ≤ ref ≤ b).
rTarget radius (must satisfy a ≤ r ≤ b).
Returns
The computed R′ value; returns 0 if any input argument is invalid.
