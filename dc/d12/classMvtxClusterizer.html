overridevirtual

event processing

Process a single event for MVTX clustering.

Retrieves hit, raw hit, cluster, and cluster‑hit association nodes from the supplied top node, resets existing MVTX clusters and associations, executes the appropriate clustering algorithm (raw or digitized), prints the resulting clusters, and returns a status code.
