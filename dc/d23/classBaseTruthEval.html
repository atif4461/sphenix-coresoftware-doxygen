Detailed Description

BaseTruthEval provides utilities to query Geant4 truth information stored in the Fun4All node tree It extracts pointers to truth containers supports strict and verbose modes and offers methods to retrieve embed flags vertices primary showers and particles parent relationships and to compare objects for identity The class also supplies hit based queries when full simulation nodes are available This comment was generated by openai/gpt-oss-120b at temperature 0.1.

PHG4Particle * BaseTruthEval::get_primary_particle ( PHG4Hit * g4hit)

which primary particle resulted in this truth hit?

Retrieve the primary PHG4Particle associated with a given PHG4Hit.

Parameters
g4hitPointer to the PHG4Hit for which the primary particle is requested.
Returns
Pointer to the primary PHG4Particle or nullptr if not found or on error.

The method first verifies that reduced node pointers are available; if not it increments an error counter and returns nullptr. In strict mode an assertion checks that the input hit pointer is valid. When not in strict mode a null hit also increments the error counter and returns nullptr. The associated particle is obtained via get_particle and its primary ancestor is resolved with get_primary_particle(particle). In strict mode the existence of the primary particle is asserted; otherwise a missing primary increments the error counter. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
