◆ InitRun()

int RetowerCEMC::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run specific configuration and node tree entries.

This method is invoked at the start of each new run. It creates the required analysis node under the provided top node, determines the first phi index, and loads the appropriate energy fraction tables. When the weighted energy distribution flag is set to 1 the weighted fractions are retrieved, otherwise the standard fractions are used.

Parameters
topNodePointer to the top PHCompositeNode of the current job.
Returns
Fun4AllReturnCodes::EVENT_OK indicating successful initialization.
