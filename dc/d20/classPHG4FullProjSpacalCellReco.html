Detailed Description

Class implements reconstruction of SPACAL cells from Geant4 hits Provides initialization per run event processing and event reset Allows configuration of detector name energy conservation check and timing window Uses LightCollectionModel to apply optical photon collection effects Stores mapping from cell identifier to PHG4Cell objects Checks total energy of hits against cell energy when enabled This comment was generated by openai/gpt-oss-120b at temperature 0.1.
