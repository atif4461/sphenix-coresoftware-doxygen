Detailed Description

DumpPHGenIntegral provides functionality to dump the contents of a PHGenIntegral node from the Fun4All node tree. The class inherits from DumpObject and overrides the process_Node method to perform the actual dumping operation. The constructor requires the name of the node to be processed and the destructor follows the base class virtual destructor semantics.
