Detailed Description

Analysis module that creates a ROOT ntuple with tower hit information.

Retrieves tower hit containers from the Fun4All node tree, fills histograms of hit multiplicity and energy loss for each tower, and writes the results to a ROOT file specified by the module constructor

Init method performs full initialization of histogram manager and output file

process_event method loops over configured tower nodes, extracts hit counts and energy deposits, fills corresponding TH1 histograms and the TNtuple

End method writes the TNtuple and histograms to the output file and releases allocated resources

AddNode method registers a tower node name together with its tower type string and detector identifier used during event processing

full initialization

Initialize the G4TowerNtuple analysis module.

This method creates a new histogram manager, opens the output ROOT file, defines an ntuple for tower information, and books two energy histograms covering the ranges 0‑1 GeV and 0‑100 GeV. The histograms are stored in the internal eloss vector for later filling during event processing.
