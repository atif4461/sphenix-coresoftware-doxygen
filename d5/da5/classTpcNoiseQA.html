Detailed Description

TpcNoiseQA analysis module for TPC noise quality assurance Inherits from SubsysReco and overrides InitRun process event and End Creates histograms of pedestal mean and RMS for north and south sides Stores per FEE channel average ADC value RMS and counts Provides methods to create histograms and generate histogram name prefix Uses file name supplied at construction and stores ADC samples vector
