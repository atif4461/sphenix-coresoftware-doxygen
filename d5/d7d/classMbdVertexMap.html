Detailed Description

Container class for MbdVertex objects indexed by unique identifier Provides interface for query insertion removal and iteration over stored vertices Implements PHObject interface required by Fun4All framework Default implementation returns placeholder values for empty map

MbdVertexMap::ConstIter MbdVertexMap::find ( unsigned int idkey) const
virtual

Retrieve a constant iterator to the vertex with the given identifier.

This method searches the MbdVertexMap for a vertex whose key matches the supplied unsigned integer identifier. If the vertex is found, a const iterator pointing to the corresponding entry is returned; otherwise the iterator returned is equal to the map’s end iterator.

Parameters
idkeyUnsigned integer key identifying the vertex to locate.
Returns
Const iterator to the found vertex or the map end iterator if the vertex does not exist in the map.
