TriggerPrimitiveContainerv1::TriggerPrimitiveContainerv1 ( TriggerDefs::TriggerId tid,
TriggerDefs::DetectorId did )

Construct a TriggerPrimitiveContainerv1 for a specific trigger and detector.

Parameters
tidTrigger identifier specifying the trigger type.
didDetector identifier specifying the detector subsystem.

The constructor selects the number of primitives and the number of sums per primitive based on the provided trigger and detector identifiers. It assigns the appropriate primitive identifier (calPId, jetPId, pairPId, mbdPId) and creates a TriggerPrimitivev1 for each primitive index. For each primitive a vector of unsigned int sums is created for each sum index and associated with a unique TriggerSumKey. All primitives are then added to the container using add_primitive.
