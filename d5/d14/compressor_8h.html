Allocate or reuse a location identifier.

The routine first attempts to reuse a previously freed location stored in loc_vec. If loc_vec is non empty the last element is removed and returned. When loc_vec is empty a new identifier is created as the current size of loc_set_vec, a new empty set is appended to loc_set_vec and the identifier is returned.

Parameters
loc_vecPointer to a vector of integer locations used as a free list
loc_set_vecPointer to a vector of integer sets representing allocated location containers
