Detailed Description

ParticleFlowElement base class for particle flow elements in sPHENIX reconstruction Provides abstract getters and setters for kinematic quantities track and calorimeter clusters Default implementations return NaN sentinel values PFLOWTYPE enumeration defines element categories UNASSIGNED MATCHED CHARGED HADRON UNMATCHED CHARGED HADRON UNMATCHED EM PARTICLE UNMATCHED NEUTRAL HADRON LEFTOVER EM PARTICLE Inherits from PHObject to integrate with Fun4All node architecture Supports ROOT IO via ClassDefOverride
