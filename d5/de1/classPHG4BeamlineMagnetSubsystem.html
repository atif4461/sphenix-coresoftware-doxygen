Detailed Description

PHG4BeamlineMagnetSubsystem implements the beamline magnet detector subsystem within the Fun4All framework It derives from PHG4DetectorSubsystem and provides run initialization creation of detector geometry stepping action and hit nodes InitRunSubsystem creates the detector object places it on the DETECTORS node creates the stepping action on the ACTIONS node and registers hit containers for output DST process_event retrieves hit containers from the top node and forwards them to the stepping action for per event processing Print outputs subsystem information according to the SubsysReco interface GetDetector returns the underlying PHG4Detector instance SetDefaultParameters configures default geometry and magnetic field parameters for the subsystem The private member m_Detector holds the concrete PHG4BeamlineMagnetDetector instance
