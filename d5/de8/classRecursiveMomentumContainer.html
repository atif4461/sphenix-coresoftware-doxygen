Initialize iteration over a 3‑dimensional container of track collections.

The method resets the internal indices (x_pos, y_pos, z_pos) to the origin of the container and searches for the first non‑null element. If a null entry is encountered the indices are advanced in a deterministic order: first the z index, then y, then x. When a non‑null container is found, the method returns the result of its begin() call, which yields a pointer to the first TrivialTrack in that sub‑container. If no non‑null element exists in the entire 3‑D array, the method returns nullptr.

bool RecursiveMomentumContainer::insert ( TrivialTrack & track)
virtual

Insert a TrivialTrack into the hierarchical momentum container.

The function first checks whether the track momentum components lie within the container bounds. If the track is out of range the insertion fails and false is returned. Otherwise the appropriate sub‑container index in the three‑dimensional momentum space is determined. If the sub‑container does not yet exist it is created – either as a regular RecursiveMomentumContainer or as a leaf RecursiveMomentumContainerEnd depending on the current recursion level. The insertion request is then forwarded recursively to the selected sub‑container.

virtual TrivialTrack * RecursiveMomentumContainer::next ( )
inlinevirtual

Retrieve the next TrivialTrack from a three dimensional container structure.

The iterator walks through the containers indexed by x_pos, y_pos and z_pos. If the current block is null the iterator advances to the next block according to a predefined order (z then y then x). When a block is entered for the first time the iterator is reset to the beginning of that block; otherwise it requests the next element from the current block. If the end of a block is reached the iterator moves to the subsequent block. The function returns a pointer to the next TrivialTrack or nullptr when no further tracks are available.
