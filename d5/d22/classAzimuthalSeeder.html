Detailed Description

AzimuthalSeeder is a SubsysReco module that creates azimuthal track seeds from silicon clusters It retrieves geometry cluster and seed containers from the node tree builds seed objects containing cluster keys and global positions and optionally fills diagnostic histograms Configuration methods allow enabling histogram output and setting an outlier residual limit The module implements the standard Fun4All lifecycle methods Init InitRun process_event and End
