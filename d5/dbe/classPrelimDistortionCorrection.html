Detailed Description

Implements a preliminary distortion correction module for the Fun4All framework Inherits from SubsysReco and provides hooks for initialization run processing and end of job The module can be configured to use truth clusters fixed cluster errors and specific gas mixture fractions Provides methods to publish refitted track seeds and to retrieve required nodes from the node tree Internally holds a TpcDistortionCorrection utility and containers for distortion correction data Uses an ALICEKF fitter and optional Acts geometry This comment was generated by openai/gpt-oss-120b at temperature 0.1.

◆ process_event()

int PrelimDistortionCorrection::process_event ( PHCompositeNode * topNode)
overridevirtual

Process a single event applying preliminary distortion corrections to TPC track seeds.

The method iterates over all track seeds in the internal track map, selects those belonging to the TPC, offsets the cluster Z coordinate to zero, applies configured distortion correction modules, restores the original Z offset, stores the corrected positions, refits the seeds with the ALICE Kalman filter and publishes the updated seeds back to the node tree. The function performs no action when not running in pp mode.

Parameters
topNodePointer to the top PHCompositeNode (unused in this implementation).
