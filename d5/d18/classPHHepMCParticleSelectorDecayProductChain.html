Detailed Description

Particle selector for HepMC based events Will write out only _theParticle and _theDaughters (if specified) Special case: when _theParticle=0, all particles in _theDaughers list will be written out no matter where they come from / Class PHHepMCParticleSelectorDecayProductChain selects HepMC particles based on a specified decay product chain
/ SetParticle defines the PDG id of the particle to be kept in the output
/ AddAncestor registers PDG ids of allowed ancestor particles in the decay chain
/ AddDaughter registers PDG ids of allowed daughter particles of the target particle
/ get_embedding_id returns the embedding id used to select the event type
/ set_embedding_id sets the embedding id for the event to be processed
/ process_event processes the top node of the Fun4All event and applies the selection criteria
/ GetParent searches the HepMC event for a parent particle that matches one of the registered ancestors This comment was generated by openai/gpt-oss-120b at temperature 0.1.
