Detailed Description

CentralityInfo stores centrality related quantities for an event It defines an enumeration PROP with identifiers for MBD charge sums sEPD energy sums and impact parameter The class provides virtual methods to query existence set and retrieve raw quantity centile and centrality bin values Derived classes implement actual storage
