◆ CheckTiltAngle()

int PHG4InnerHcalDetector::CheckTiltAngle ( ) const

Verify that the detector tilt angle is within physical limits and intersects the inner radius.

This routine checks that the absolute value of the tilt angle does not exceed 90 degrees. It then constructs a line representing the tilted scintillator and tests for an intersection with the inner cylindrical surface. If the angle is invalid or the line does not intersect the inner radius the program prints an error message and terminates. On successful validation the function returns zero.

Returns
Zero if the tilt angle is valid and an intersection is found.

◆ ConstructSteelPlate()

G4VSolid * PHG4InnerHcalDetector::ConstructSteelPlate ( G4LogicalVolume * hcalenvelope)

Construct the extruded steel plate solid for the inner HCal.

Parameters
hcalenvelopePointer to the HCal envelope logical volume (currently unused).
Returns
Pointer to a G4VSolid representing the steel plate geometry.

The method calculates the lower and upper edges of the steel plate by intersecting perpendicular lines to the scintillator gaps with the inner and outer cylindrical boundaries using CGAL geometry primitives. The four corner points are adjusted from a secant to a tangent at the cylinder centre and used to create a G4ExtrudedSolid. The resulting solid volume is stored in m_VolumeSteel and scaled by the number of scintillator plates.
