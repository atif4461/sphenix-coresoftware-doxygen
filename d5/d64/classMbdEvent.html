Detailed Description

MbdEvent class provides reconstruction of MBD detector data It handles initialization of run termination and clearing of event data Raw data can be set from Event objects or from packet arrays when not in online mode Processing includes packet handling raw container channel post processing and calculation of output objects Configuration setters control simulation flag raw dst flag fits only debug verbosity and evaluation mode Accessors return reconstructed vertex position time and charge as well as per arm hit counts charge and timing information Calibration utilities manage gain corrections timing offsets sample maximum calibration and pedestal determination Internal data structures store geometry calibration signals raw waveforms and derived quantities

◆ FillSampMaxCalib()

int MbdEvent::FillSampMaxCalib ( )

Fill histograms with ADC samples and perform on‑the‑fly sample‑maximum calibration.

This method loops over all MBD front‑end channels, extracts the sample index and ADC value for each time sample, fills the corresponding 2‑D waveform histograms, and determines the local maximum ADC value for the channel. If the maximum ADC exceeds a threshold (20) the sample‑maximum histograms are filled. After a predefined number of events (_no_sampmax) the method triggers the calculation of the sample‑maximum calibration (CalcSampMaxCalib), marks the calibration as completed, and configures each channel’s pedestal range and pre‑sample settings based on the newly obtained calibration constants.

Returns
Integer status flag (always 1 on successful execution).

◆ PostProcessChannels()

void MbdEvent::PostProcessChannels ( MbdPmtContainer * bbcpmts)

Processing after all channels have been calibrated, to remove outliers, etc.

Post‑processes BBC PMT channels to examine timing and charge information.

Iterates over all BBC PMTs, retrieves hit time from the time channel (tt), hit time from the charge channel (tq), and the charge (q). When verbose mode is enabled and tt is NaN while q is positive, diagnostic waveforms for the corresponding front‑end electronics channels are drawn.

Parameters
bbcpmtsPointer to the MbdPmtContainer containing BBC PMT hits for the current event.
