Retrieve the chi‑square of the SVTX vertex containing the largest number of tracks.

The method searches the internal vertex map for SVTX type vertices. If none are found, NaN is returned. Among the SVTX vertices, the vertex with the highest track multiplicity is selected and its chi‑square value is returned.

Returns
Chi‑square of the selected vertex, or NaN if no SVTX vertices are present. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

◆ get_error()

float GlobalVertexv3::get_error ( unsigned int i,
unsigned int j ) const
overridevirtual

Retrieve the covariance error element (i,j) for the global vertex.

The function first attempts to locate a silicon vertex (SVTX) in the internal vertex map. If found, it selects the SVTX vertex that contains the largest number of associated tracks and returns its error matrix element specified by the indices i and j. If no SVTX vertex is present, the function falls back to a muon detector (MBD) vertex; only the z‑coordinate error (i==2 && j==2) is defined for MBD vertices. When neither vertex type is available or the requested element is undefined, NaN is returned.

Parameters
iRow index of the error matrix element (0‑based).
jColumn index of the error matrix element (0‑based).
Returns
The requested error value as a float, or NaN if unavailable. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
