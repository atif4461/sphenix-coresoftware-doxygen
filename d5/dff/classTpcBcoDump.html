Detailed Description

Class TpcBcoDump provides functionality to dump TPC BCO information into a ROOT file It inherits from SubsysReco and implements InitRun process_event and End callbacks The module creates an output TFile and TTree and stores event identifiers and BCO values User can set output file name via OutFileName method
