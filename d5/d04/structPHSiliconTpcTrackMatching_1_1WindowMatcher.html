WindowMatcher stores selection windows for positive and negative charge tracks.

The class holds three dimensional lower and upper bounds for positive and negative windows and minimum transverse momentum thresholds for each charge sign

Parameters

Determine whether the silicon‑TPC X coordinate difference falls within the matching window.

Parameters

Convert a three‑element array into a human‑readable string representation.

Parameters

Returns
A string containing the formatted expression. If the second element is zero, only the first element is returned.
