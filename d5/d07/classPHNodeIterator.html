Detailed Description

Iterator utility for traversing PHCompositeNode hierarchies within the Fun4All node tree Provides methods to list child nodes locate nodes by name or path change the current traversal context and apply operations to each node Maintains an internal pointer to the current PHCompositeNode and a list of its direct subnodes for efficient navigation
