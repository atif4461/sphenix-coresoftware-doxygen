Detailed Description

BcoLumiReco module integrates beam crossing and luminosity information into the Fun4All processing chain It inherits from SubsysReco and implements Init and process_event callbacks The module maintains a sliding window of three beam crossing identifiers previous current and future and corresponding event numbers SyncObject pointers provide thread safe access to shared data structures CreateNodeTree constructs the required nodes in the PHCompositeNode hierarchy This comment was generated by openai/gpt-oss-120b at temperature 0.1.
