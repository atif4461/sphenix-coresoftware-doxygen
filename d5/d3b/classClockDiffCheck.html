void ClockDiffCheck::FillCaloClockDiffSngl ( CaloPacket * calopkt)

Fill per packet clock difference histogram and update internal packet state.

Parameters
calopktpointer to CaloPacket containing identifier and BCO timestamp
Returns
void

The function checks whether the packet identifier is present in the internal map If the identifier is not present a new histogram named clkdiff<id> with 100 bins from 0 to 99 is created A tuple containing three maximum uint64 values a histogram pointer and a validity flag is stored in the map If the identifier already exists the stored clock value is updated with the current BCO The clock difference is calculated with respect to the previous clock value handling 32 bit rollover The calculated difference and a true validity flag are written back into the tuple Verbose diagnostic messages are printed when the module verbosity level exceeds the configured thresholds This comment was generated by openai/gpt-oss-120b at temperature 0.1.
