ClusKeyIter ClusKeyIter::begin ( ) const

Obtain an iterator positioned at the first cluster key of the associated track.

The iterator is initialized according to the seed type attached to the track. If the track contains no data the returned iterator is marked as empty. For silicon seeds the iterator is set to the beginning of the silicon cluster key range. For TPC seeds the iterator is set to the beginning of the TPC cluster key range.

Returns
A ClusKeyIter object representing the start of the cluster key collection for the track. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
