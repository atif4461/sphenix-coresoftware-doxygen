uint64_t SingleGl1TriggeredInput::GetClock ( Event * evt,
int pid )
overridevirtual

Retrieve the BCO clock value from the specified packet of an event.

This function obtains the packet with the given pid from the event, extracts the 64‑bit BCO (beam crossing) clock value, and returns it. If the packet cannot be found, an error message is printed and the maximum uint64_t value is returned.

Parameters
evtPointer to the Event object containing the packet data.
pidIdentifier of the packet to be accessed.
Returns
The extracted clock value, or std::numeric_limits<uint64_t>::max() on failure.
