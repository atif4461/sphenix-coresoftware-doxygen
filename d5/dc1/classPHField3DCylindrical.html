void PHField3DCylindrical::GetFieldCyl ( const double CylPoint[4],
double * BfieldCyl ) const

Interpolates the magnetic field at a given cylindrical coordinate.

This method performs a trilinear interpolation of the stored 3‑D field maps (Bz, Br, Bphi) using the supplied point (z, r, phi). The input array CylPoint contains the longitudinal position z, the radial coordinate r and the azimuthal angle phi (the fourth element is ignored). The interpolated field components are returned in the three‑element array BfieldCyl as (Bz, Br, Bphi) in the same units as the internal field tables. Points outside the defined map region are ignored and the output is left at zero. Verbosity level greater than two triggers diagnostic prints of the input point, boundary checks, interpolation weights and the final field values (converted to gauss).

Parameters
CylPointInput array of size 4 containing (z, r, phi, unused).
BfieldCylOutput array of size 3 that will hold the interpolated magnetic field components (Bz, Br, Bphi).
Returns
None (void). The result is stored in BfieldCyl. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

void PHField3DCylindrical::GetFieldValue ( const double point[4],
double * Bfield ) const
overridevirtual

Compute the magnetic field vector at a given Cartesian point using a cylindrical field map.

This method converts the input Cartesian coordinates (x, y, z) to cylindrical coordinates (z, r, φ), queries the underlying cylindrical field representation via GetFieldCyl, and then transforms the resulting cylindrical field components (Bz, Br, Bφ) back to Cartesian components (Bx, By, Bz).

The function checks that the z‑coordinate lies within the configured field region ([minz_, maxz_]). If the point is outside this region the field is set to zero.

Parameters
pointArray of four doubles containing the Cartesian coordinates of the query point. Elements 0, 1, and 2 correspond to x, y, and z, respectively; the fourth element is unused.
BfieldOutput array of three doubles that will be filled with the Cartesian magnetic field components Bx, By, and Bz.
Returns
