Detailed Description

Quality assurance module for G4 simulation using KFParticle Inherits from SubsysReco and implements Fun4All callbacks Loads required nodes evaluates clusters builds track objects and interfaces to KFParticle container for particle reconstruction Configuration includes mother particle name and mass window Provides methods to set track map name and retrieve histogram prefix This comment was generated by openai/gpt-oss-120b at temperature 0.1.
