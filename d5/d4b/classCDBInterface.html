Detailed Description

Interface to the conditions database for sPHENIX analysis modules.

Provides singleton access to a SphenixClient instance and methods to enable or disable interaction with the CDB during a Fun4All job The class inherits from SubsysReco so that it can be inserted into the processing chain and receive callbacks such as End and UpdateRunNode URL construction follows the pattern domain filename and is stored in an internal set to avoid duplicate requests This comment was generated by openai/gpt-oss-120b at temperature 0.1.
