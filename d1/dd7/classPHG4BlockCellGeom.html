Detailed Description

PHG4BlockCellGeom stores geometry parameters for a block cell in the G4 simulation It provides accessors for layer binning scheme and bin boundaries in z x and eta The class inherits from PHObject and implements the identify method Bin parameters include number of bins minimum value and step size for each dimension Helper methods validate the binning method and compute bin indices and centers
