Detailed Description

BEmcProfile provides methods to load a probability profile from a file and to evaluate and predict electromagnetic calorimeter tower energies based on module lists and kinematic variables The constructor loads the profile data from the specified file The destructor releases allocated resources GetProb computes the probability for a given list of modules tower count energy polar and azimuthal angles GetTowerEnergy returns the energy of a specific tower identified by its indices PredictEnergy estimates the expected energy and its uncertainty for a given particle trajectory PredictEnergyR returns a corrected energy based on radial distance IsLoaded reports whether the profile data has been successfully loaded Verbosity getters and setters control the diagnostic output level Internal data members store loading status thresholds array dimensions histograms and verbosity

BEmcProfile::BEmcProfile ( const std::string & fname)
explicit

Load BEmc profile data from a ROOT file.

Parameters
fnamePath to the ROOT file containing the profile histograms

The constructor opens the file, reads the energy and theta binning histograms, stores the bin contents in internal arrays, and loads the mean, sigma and r4 histograms for each combination of energy, theta and particle type. Errors during file access or histogram retrieval are reported to standard output and the object remains unloaded. On successful completion the internal flag bloaded is set

float BEmcProfile::PredictEnergyR ( float energy,
float theta,
float phi,
float rr )

Predict the energy response at a given radial distance.

This method interpolates the stored profile histograms (hr4) in both incident energy and polar angle (theta) to estimate the expected energy deposition at radius rr. The interpolation is performed logarithmically in energy and quadratically in theta. The function validates the input parameters and returns zero if the profiles have not been loaded or if the inputs are out of the allowed range.

Parameters
energyIncident particle energy (must be greater than zero).
thetaPolar angle in radians (must be non‑negative).
rrRadial distance from the detector centre (must be greater than one).
Returns
Predicted energy response; returns 0 on validation failure.
