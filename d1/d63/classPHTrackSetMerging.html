Detailed Description

Base class for track seeding.

Module merges two input track maps into a single output track map within the Fun4All framework Provides hooks for initialization run setup event processing and finalization Configurable input and output map names are set via public setters Handles retrieval and creation of required node objects such as cluster vertex and track maps Derived classes must implement the Process method to define the actual merging algorithm This comment was generated by openai/gpt-oss-120b at temperature 0.1.
