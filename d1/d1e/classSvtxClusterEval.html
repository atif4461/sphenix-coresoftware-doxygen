◆ max_truth_cluster_by_energy()

std::pair< TrkrDefs::cluskey, std::shared_ptr< TrkrCluster > > SvtxClusterEval::max_truth_cluster_by_energy ( TrkrDefs::cluskey cluster_key)

Retrieve the truth cluster that best matches a reconstructed cluster based on the highest‑energy contributing truth particle.

This method first determines the truth particle that contributes the most energy to the specified reconstructed cluster. It then searches among all truth clusters associated with that particle, selecting the one that lies within layer‑dependent position tolerances (r‑phi and z) relative to the reconstructed cluster. The search respects different resolution parameters for MVTX, TPC inner/mid/outer, INTT, and MMS layers. If a matching truth cluster is found, its key and a shared pointer to the cluster are returned; otherwise a pair of zero key and nullptr is returned.

Parameters
cluster_keyIdentifier of the reconstructed cluster to be evaluated.
Returns
Pair consisting of the truth cluster key and a shared pointer to the matching TrkrCluster; returns (0, nullptr) if no match is found. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
