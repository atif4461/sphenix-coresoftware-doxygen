Detailed Description

SyncReco module integrates with Fun4All framework to provide synchronization of reconstruction steps It inherits from SubsysReco and implements required lifecycle callbacks Init creates or validates node tree and prepares resources InitRun handles run specific initialization process_event processes each event and may enforce a specific segment number SegmentNumber allows external override of segment identifier useful for embedding scenarios
