Detailed Description

Compares reconstructed jets to truth jets.

Class to evaluate jet reconstruction performance within the Fun4All framework It reads reconstructed and truth jet collections identified by configurable names It optionally fills ntuples with jet kinematic variables for external analysis It provides diagnostic output of input and output object ancestry Configuration includes selection of strict mode and toggles for reco and truth evaluation Results are written to a ROOT file with configurable name This comment was generated by openai/gpt-oss-120b at temperature 0.1.

Plan: This module will trace the jet constituents to the greatest contributor Monte Carlo jet and then test one against the other.
