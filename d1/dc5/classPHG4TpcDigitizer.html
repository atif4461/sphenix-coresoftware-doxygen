int PHG4TpcDigitizer::InitRun ( PHCompositeNode * topNode)
overridevirtual

run initialization

Initialize run dependent configuration for the TPC digitizer.

Parameters
topNodePointer to the top level PHCompositeNode of the current job.
Returns
Fun4All return code EVENT_OK on success or ABORTRUN if the DST node is missing.

The method adds the pedestal to the ADC threshold, computes the charge to voltage conversion gain and the noise conversion gain, and derives the threshold in millivolts. It locates the DST composite node in the node tree, aborts if the node is not present, and calls the cylinder cell ADC scaling routine. When verbosity is greater than zero it prints the maximum ADC per layer and the energy per ADC scaling for each layer. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
