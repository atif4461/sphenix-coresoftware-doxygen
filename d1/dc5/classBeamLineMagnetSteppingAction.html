Detailed Description

Class implements stepping action for beam line magnet detector Constructs with pointer to detector and parameter set Destroys allocated resources Called for each Geant4 step to accumulate energy deposition and create hits Sets interface pointers to node tree for hit containers Allows configuration of hit node names for active and absorber volumes Maintains internal state including saved volume pointers track identifiers step status accumulated energy deposition and flags for activity absorber and black hole handling
