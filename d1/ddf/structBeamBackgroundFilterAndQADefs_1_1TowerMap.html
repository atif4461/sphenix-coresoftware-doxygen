Detailed Description

template<std::size_t H, std::size_t F>
struct BeamBackgroundFilterAndQADefs::TowerMap< H, F >

Helper type for building (eta, phi) maps of towers.

This is a lightweight struct to organize the above Tower structs into a handy (eta, phi) map. See the StreakSidebandFilter algorithm for an example usage.
