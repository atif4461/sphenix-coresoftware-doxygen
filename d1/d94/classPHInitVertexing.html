Detailed Description

Base subsystem for initializing vertex reconstruction in the Fun4All framework.

Base class for inital vertexing.

This class derives from SubsysReco and provides the common infrastructure required to prepare the tracking cluster container and the vertex map for subsequent vertexing algorithms. It implements the standard Fun4All callbacks InitRun and process_event, delegating detector‑specific actions to the virtual Setup and pure virtual Process methods which must be overridden by concrete tracker implementations. The protected members _cluster_map and _vertex_map are populated by CreateNodes/GetNodes and made available to derived classes.

The workflow is:

  • InitRun: create output nodes (CreateNodes) and retrieve input nodes (GetNodes).
  • Setup: optional additional initialization performed by derived classes.
  • process_event: invoke the detector‑specific Process implementation.

Derived classes should implement Process to perform per‑event vertexing logic using the provided cluster and vertex containers.
