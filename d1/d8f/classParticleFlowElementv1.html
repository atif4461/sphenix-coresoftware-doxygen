Detailed Description

Implementation of ParticleFlowElement for sPHENIX reconstruction.

Provides storage and accessors for particle flow information including type identifier momentum energy associated track and calorimeter clusters

Implements PHObject interface methods identify Reset isValid and the ParticleFlowElement abstract interface for type id momentum components energy track ecal clusters hcal derived kinematic quantities

Member data includes unique identifier type momentum array energy pointer to SvtxTrack vector of pointers to RawCluster for electromagnetic clusters and pointer to RawCluster for hadronic cluster

Class version is 1 for ROOT I/O
