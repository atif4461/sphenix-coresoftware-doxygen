Detailed Description

Class implements stepping action for MVTX detector in Geant4 simulation It records energy loss and creates hit objects in appropriate node containers The detector pointer provides geometry and material information Parameters container supplies user configurable flags such as support active and black hole UserSteppingAction processes each G4Step and fills hit containers SetInterfacePointers connects to the node tree and retrieves hit containers SetHitNodeName allows custom naming of hit and support nodes Private data members store pointers to detector hit containers current hit and shower objects and configuration flags
