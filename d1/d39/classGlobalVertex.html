GlobalVertex::ConstVertexIter GlobalVertex::begin_vertexes ( ) const
virtual

Retrieve a constant iterator to the first vertex in the GlobalVertex container.

This method provides read‑only access to the beginning of the internal vertex collection. The returned iterator can be used to traverse the vertices in forward order. If the container is empty, the iterator will be equal to the end iterator.

Returns
ConstVertexIter iterator pointing to the first vertex (or end if the container is empty).
