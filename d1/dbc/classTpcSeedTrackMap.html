Detailed Description

Interface for mapping TPC seed identifiers to associated track identifiers.

This abstract class derives from PHObject and defines a multimap based association between a TPC seed key and one or more track keys. It provides a Reset method that overrides the base class implementation and a set of pure‑virtual functions for managing the associations:

  • addAssoc: Insert a (seed,track) pair into the internal map.
  • getAssocTracks: Return an iterator range over all track keys associated with a given seed key.
  • getAll: Return an iterator range covering the entire seed‑to‑track map.
  • size: Report the total number of (seed,track) entries stored.

The class uses type aliases for the underlying map, constant iterator, and constant range to simplify client code. Concrete implementations must provide the storage and logic for these operations while preserving the interface contract.

The map key type is an unsigned int representing the TPC seed identifier, and the mapped value type is an unsigned int representing the track identifier. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
