Detailed Description

PHG4Particlev1 class provides concrete implementation of PHG4Particle interface stores particle name pid momentum components and barcode default constructor creates empty particle parameterized constructor initializes name pid and momentum components copy constructor creates particle from existing PHG4Particle instance destructor uses default behavior CloneMe method returns deep copy of particle identify method prints particle information to output stream get_pid returns particle identifier get_name returns particle name string get_px returns x component of momentum get_py returns y component of momentum get_pz returns z component of momentum get_barcode returns unique particle barcode set_name assigns particle name set_pid assigns particle identifier set_px assigns x component of momentum set_py assigns y component of momentum set_pz assigns z component of momentum set_barcode assigns unique particle barcode
