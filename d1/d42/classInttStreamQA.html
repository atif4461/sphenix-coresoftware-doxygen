Detailed Description

Definition of this analysis module class.

/*!

Analysis module for INTT raw data stream quality assurance.

Retrieves InttRawHitContainer objects from the Fun4All node tree, creates per FELIX link histograms of bunch crossing counters and hit distributions, fills them during event processing, and writes the histograms at the end of the job. The module is instantiated with a configurable name and uses internal helper functions to generate histogram prefixes and allocate ROOT objects.
