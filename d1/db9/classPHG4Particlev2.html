◆ identify()

void PHG4Particlev2::identify ( std::ostream & os = std::cout) const
overridevirtual

Print a textual representation of the particle to the provided output stream.

The method writes the particle name (or indicates missing), track identifier, vertex identifier, parent identifier, primary identifier, PDG code, momentum components (px, py, pz), derived azimuthal angle phi, pseudorapidity eta, and energy to the stream, followed by a newline.

Parameters
osOutput stream to which the particle information is written.
Returns
No return value.
