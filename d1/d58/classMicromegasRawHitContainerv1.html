Detailed Description

Container class for Micromegas raw hit objects version 1.

Implements the MicromegasRawHitContainer interface and provides event level management of MicromegasRawHit instances. The class owns a TClonesArray that stores the hit objects and offers methods to reset the container, identify its contents, check validity, add new hits, query the number of stored hits and retrieve a hit by index. All public functions override the virtual interface of the base container and are integrated with the PHObject framework used in the sPHENIX software stack.
