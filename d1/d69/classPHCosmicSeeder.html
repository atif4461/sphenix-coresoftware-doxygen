Detailed Description

PHCosmicSeeder is a SubsysReco module that creates cosmic track seeds from cluster positions It allows configuration of ADC cut XY tolerance and optional analysis output The module retrieves cluster containers builds seeds combines and chains them and stores results in a TrackSeedContainer Configuration methods enable setting of track map name tracker identifier and activation of analysis mode
