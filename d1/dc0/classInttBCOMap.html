Detailed Description

Determine whether a given BCO value is considered bad for a specific FELIX server and channel.

This function computes the difference between the provided BCO (bco) and the lower 7 bits of the full BCO (bco_full). It then checks the difference against the stored peak BCO value for the specified FELIX server and channel, as well as the immediate neighboring BCO values (peak ± 1, with wrap‑around at the 0–127 range). If the stored peak is uninitialized (‑1) or the difference matches the peak, the preceding, or the succeeding BCO, the BCO is accepted (function returns false). Otherwise, the BCO is flagged as bad (function returns true).

bco_fullFull 64‑bit BCO value from which the lower 7 bits are used for comparison.
bcoCurrent 7‑bit BCO value to be evaluated.
Returns
