Detailed Description

PHG4EtaPhiParameterization provides a parameterisation of cylindrical detector volumes in eta phi space The class maps copy numbers to eta and phi bins and computes dimensions and transformations for G4Tubs rings Constructor arguments define eta binning phi binning radial extents and longitudinal centre Z ComputeDimensions fills ring geometry based on precomputed bin edges ComputeTransformation positions physical volume according to bin indices GetIEta returns the eta index for a given copy number GetIPhi returns the phi index for a given copy number
