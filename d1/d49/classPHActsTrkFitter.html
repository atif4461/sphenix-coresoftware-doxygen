◆ InitRun()

int PHActsTrkFitter::InitRun ( PHCompositeNode * topNode)
overridevirtual

Get and create nodes.

Initialize the PHActsTrkFitter for a new run.

This method is invoked by Fun4All when a new run is encountered. It creates and retrieves required nodes in the node tree, configures alignment states, determines whether the magnetic field is constant, sets up the Kalman and directed Kalman fitting algorithms, selects material surfaces if needed, configures the outlier finder, optionally creates timing histograms, and initializes an optional ActsEvaluator. It also obtains the TPC geometry container from the DST node. Verbosity controls console output and logging level.

Parameters
topNodePointer to the top PHCompositeNode of the current job.
Returns
Fun4AllReturnCodes::EVENT_OK on successful initialization, Fun4AllReturnCodes::ABORTEVENT if node creation or retrieval fails, Fun4AllReturnCodes::ABORTRUN if the TPC geometry container cannot be found.
