Detailed Description

PHG4EnvelopeSubsystem provides the envelope detector subsystem integration within the Fun4All framework It creates the envelope detector geometry and associated stepping action registers them in the node tree and supplies accessors for the detector and stepping action objects The Init method constructs the detector and stepping action nodes The process_event method forwards event processing to the stepping action GetDetector returns the underlying PHG4EnvelopeDetector instance GetSteppingAction returns the associated PHG4EnvelopeSteppingAction instance This comment was generated by openai/gpt-oss-120b at temperature 0.1.
