Detailed Description

SingleMvtxPoolInput handles streaming input for MVTX raw hit data It inherits from SingleStreamingInput and implements pool management Functions include filling the pool based on minimum bunch crossing offset Cleanup of used packets pool depth checking and event clearing Provides configuration of BCO range negative BCO and raw event header name Supports reading strobe width from database and setting strobe width Can operate in MVTX standalone mode Manages internal maps of raw hits FEE clock and strobe information Creates DST node under top node for downstream processing This comment was generated by openai/gpt-oss-120b at temperature 0.1.
