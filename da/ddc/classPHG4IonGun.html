Detailed Description

PHG4IonGun class implements a generator for ion particles in the Geant4 simulation It derives from PHG4ParticleGeneratorBase and provides methods to set atomic mass number atomic number charge momentum excitation energy and to update the internal particle representation The process_event method creates the ion and inserts it into the top node The Print method outputs the current configuration This comment was generated by openai/gpt-oss-120b at temperature 0.1.
