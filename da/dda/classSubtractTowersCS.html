Detailed Description

creates new UE-subtracted towers

SubtractTowersCS implements a SubsysReco module that subtracts tower energies using a constant scaling factor and optional flow modulation The module is configured via SetFlowModulation SetAlpha and SetDeltaRmax InitRun creates the required output node in the PHCompositeNode tree process_event applies the subtraction to each event Internal flags control the use of flow modulation the alpha parameter and the maximum delta R This comment was generated by openai/gpt-oss-120b at temperature 0.1.

Using a previously determined background UE density, this module constructs a new set of towers by subtracting the background from existing raw towers. CS parameters are configurable

◆ process_event()

int SubtractTowersCS::process_event ( PHCompositeNode * topNode)
overridevirtual

Process a single event and apply constituent subtraction to EMCal, inner HCal and outer HCal tower collections.

The method retrieves calibrated tower containers and geometry objects from the Fun4All node tree, constructs FastJet pseudo‑jets for the raw tower energies, builds background pseudo‑jets using the TowerBackground service (optionally modulated by flow harmonics), and performs constituent subtraction with a configurable distance type, maximum distance, alpha exponent and ghost area. Subtracted tower energies are written into dedicated CS‑version tower containers (TOWER_CALIB_CEMC_RETOWER_SUB1CS, TOWER_CALIB_HCALIN_SUB1CS, TOWER_CALIB_HCALOUT_SUB1CS). Verbose output is produced according to the module verbosity level.
