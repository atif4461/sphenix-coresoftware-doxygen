Detailed Description

TpcSpaceChargeReconstructionHelper provides static utilities for building TPOT acceptance masks and performing z and phi extrapolations on three dimensional histograms used in space charge reconstruction The class defines enumeration for side selection and helper methods for guarding bins and splitting histograms Geometry ranges for phi and theta are stored as static members and can be configured through public setters This comment was generated by openai/gpt-oss-120b at temperature 0.1.

create TPOT acceptance mask, using provided binning

Fill a 3D mask histogram for TPC space charge reconstruction.

this histogram contains 1 in cells that match the TPOT acceptance only Micromegas in the central sector (TPC sectors 9 and 21) are considered

The method resets the supplied TH3 histogram and iterates over all bins The azimuthal coordinate phi radial coordinate r and longitudinal coordinate z are obtained for each bin The polar angle theta is computed as atan2 of z and r Bins whose phi lies inside the central phi range and whose theta lies inside the central theta range are set to 1 all other bins remain zero

void TpcSpaceChargeReconstructionHelper::extrapolate_z ( TH3 * source,
const TH3 * mask )
static

first z extrapolation

Extrapolate missing z bins of a 3D distortion histogram.

interpolate along z between the two micromegas modules fully equiped sector the mask is used to decide in which bins the extrapolation must be performed

The routine scans the source histogram bin by bin in phi (X) and radius (Y). For each z slice that is not marked as valid by the mask histogram, the nearest valid bins below (iz_min) and above (iz_max) are located. A linear interpolation between the distortion values of those two bins is performed and the result, together with the propagated error, is written back into the source histogram. Bins for which a valid interpolation range cannot be found are left unchanged.

Parameters
sourceModifiable TH3 histogram containing the quantity to be extrapolated. The bin contents and errors are updated in place.
maskConst TH3 histogram used as a mask; a bin is considered valid when its content is greater than zero.

std::tuple< TH3 *, TH3 * > TpcSpaceChargeReconstructionHelper::split ( const TH3 * source)
static

separate positive and negative z histograms

Split a 3‑dimensional histogram into two histograms separated at z = 0.

split histograms in two, the first with negative z values only, the second with positive z values this must be done before adding guarding bins around each axis, in order to prevent artifacts during calls to Interpolate at the central membrane (z = 0)

The input histogram is examined along its Z axis. All bins with a Z coordinate below zero are copied into a newly created histogram named “_negz”, while all bins with a Z coordinate equal to or above zero are copied into a histogram named “_posz”. Axis titles are preserved in both output histograms.

Parameters
sourcePointer to the source TH3 histogram to be split. May be nullptr.
Returns
A std::tuple containing two TH3 pointers. The first element points to the histogram with negative Z bins (or nullptr if source is nullptr); the second element points to the histogram with non‑negative Z bins (or nullptr if source is nullptr). The caller assumes ownership of the returned histograms and is responsible for deleting them. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
