Detailed Description

RawClusterLikelihoodProfile computes likelihood profiles for raw clusters using a CDF calculator It reads cluster and tower information from specified input nodes applies energy thresholds and stores the resulting profile in an output node Configuration parameters include profile file path node names minimum cluster energy tower threshold and profile dimensionality The module follows the SubsysReco lifecycle Init creates required nodes process_event performs per event calculations and End finalizes resources This comment was generated by openai/gpt-oss-120b at temperature 0.1.
