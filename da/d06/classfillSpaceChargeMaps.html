Detailed Description

Class fillSpaceChargeMaps implements a SubsysReco module that fills space charge maps for the TPC It reads configuration parameters such as frequency beam crossing and gain and produces histograms and trees The module processes each event to compute ion backflow contributions and stores results in an output file
