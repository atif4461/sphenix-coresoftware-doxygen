EMMaximization class implements expectation maximization optimization It is constructed with an EMRandom object and an integer generation method Default population size is 200 and default iteration count is 500 The objective function pointer is initialized to the default J function Best cost is initialized to -1 and internal parameters a and a2 are zero

brief Compute HS function for given momentum and angles

this function evaluates the product of the base momentum angle function mFunc and the projection of a unit vector defined by theta0 onto the spherical direction theta phi the result is multiplied by sin theta to account for the jacobian of the spherical coordinate system

param p momentum magnitude param theta polar angle of the particle param phi azimuthal angle of the particle param theta0 reference polar angle for the projection return value of the HS function
