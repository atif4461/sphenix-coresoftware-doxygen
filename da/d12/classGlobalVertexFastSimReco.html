Detailed Description

simple truth vertex smearing algorithm

Module reconstructs global vertex for fast simulation Initializes run level nodes and processes each event Allows configuration of smearing parameters for x y z and t coordinates Creates required nodes in the Fun4All node tree Uses GSL random generator for smearing This comment was generated by openai/gpt-oss-120b at temperature 0.1.
