◆ all_truth_hits()

std::set< PHG4Hit * > CaloRawClusterEval::all_truth_hits ( RawCluster * cluster)

what truth hits contributed energy to this tower?

Retrieve all truth G4 hits associated with a given raw cluster.

This method collects the set of PHG4Hit objects that contributed to the specified RawCluster. It first validates that the required node pointers are available and that the input cluster pointer is valid according to the current strictness setting. If caching is enabled, a previously computed result is returned from the internal cache. The function then iterates over the towers belonging to the cluster, either via the tower‑info map or directly through the tower container, and aggregates the truth hits reported by the associated RawTowerEval instance. Errors encountered during processing are counted in the _errors member. The final set of unique PHG4Hit pointers is optionally cached and returned.

Parameters
clusterPointer to the RawCluster for which truth hits are requested.
Returns
A std::set containing pointers to all PHG4Hit objects linked to the input cluster. An empty set is returned if node pointers are missing, the cluster pointer is null (when not in strict mode), or no associated hits are found. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
