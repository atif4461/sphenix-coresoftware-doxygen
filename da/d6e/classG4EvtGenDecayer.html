Detailed Description

Interface class that connects Geant4 external decay handling to the EvtGen generator.

Implements the G4VExtDecayer abstract class using the Pythia6 interface. According to TPythia6Decayer class in Root: http://root.cern.ch/ see http://root.cern.ch/root/License.html

Implements the G4VExtDecayer abstract interface to provide decay products generated by EvtGen for Geant4 tracks. The class allows configuration of the decay table file, optional XML usage, and the decay vertex via a G4LorentzVector pointer. A width threshold (in keV) can be set to define stable particles: particles with a mass width below the threshold are treated as stable and passed to Geant4, while those above are decayed by EvtGen at the same vertex as their parent. Verbosity of the decayer can be controlled with SetVerboseLevel. Internal members manage the EvtGen engine, random engine, external generator list, radiation correction engine, and any additional decay models.

Note
The copy constructor and assignment operator are declared private and not implemented to prevent copying of the decayer instance.
