◆ encode_emcal() [2/2]

unsigned int TowerInfoDefs::encode_emcal ( const unsigned int towerIndex)

Encode an EMCal tower index into a global eta‑phi key.

This routine initializes static mapping tables on first call, determines the super‑sector, packet, interface board and channel for the given linear tower index, and computes the corresponding global eta and phi bin numbers. The resulting key packs the global eta bin in the high 16 bits and the global phi bin in the low 16 bits.

Parameters
towerIndexLinear index of the EMCal tower within the detector geometry.
Returns
Unsigned integer key encoding the global eta and phi bin for the tower.
