Detailed Description

Version 1 implementation of the MvtxRawHitContainer interface.

Manages a collection of MvtxRawHit objects using a TClonesArray. Provides event level reset, identification output, validity check, and methods to add, retrieve and count raw hits.

The container owns the underlying TClonesArray and is responsible for its lifetime. All public methods override the abstract interface defined in MvtxRawHitContainer.

Note
The class is integrated into the Fun4All node tree and participates in the standard I/O mechanisms of the sPHENIX framework.
