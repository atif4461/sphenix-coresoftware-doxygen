Detailed Description

Container for MVTX event level metadata.

Stores generic integer floating point unsigned and string properties indexed by name Provides interface for setting and retrieving these values Implements PHObject virtual methods for cloning resetting identification and validity checking Includes placeholder methods for HB and strobe BCO handling to be overridden in derived classes
