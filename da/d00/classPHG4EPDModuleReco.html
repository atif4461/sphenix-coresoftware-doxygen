◆ InitRun()

int PHG4EPDModuleReco::InitRun ( PHCompositeNode * topNode)
overridevirtual

module initialization

Initialize run‑level resources for the EPD reconstruction module.

This method is invoked once per new run. It updates module parameters from the macro, reads timing and MPV settings, creates the required node hierarchy under the RUN node, and ensures that the EPD geometry node (TOWERGEOM_EPD) exists. If the geometry node is missing it is instantiated as an EpdGeomV2 object and attached to the detector‑specific node. The method then populates the geometry for all 744 EPD channels by encoding the channel key and setting the z, r, and phi (or phi0) coordinates using the corresponding helper functions.

Parameters
topNodePointer to the top PHCompositeNode of the current job.
Returns
Fun4AllReturnCodes::EVENT_OK on successful initialization, otherwise the job terminates via gSystem::Exit. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

This method stores the provided string as the prefix used when locating calibrated tower nodes for the Event Plane Detector (EPD) within the Fun4All node tree. The prefix is later combined with detector‑specific identifiers to form full node names for I/O operations.
