Detailed Description

Fun4AllDstOutputManager manages output of DST files within Fun4All framework It inherits from Fun4AllOutputManager and provides node selection stripping and compression configuration for writing PHCompositeNode hierarchies to ROOT The class owns a PHNodeIOManager for DST output and tracks user requested nodes and run nodes Copy construction and assignment are disabled Interface methods allow adding nodes stripping nodes controlling saving of DST and run nodes setting compression and initializing last event handling This comment was generated by openai/gpt-oss-120b at temperature 0.1.
