Detailed Description

Concrete implementation of the MbdGeom interface for version 1 geometry.

Provides storage and accessors for photomultiplier tube (PMT) positions in Cartesian (x, y, z), cylindrical (r, phi) coordinates. The class maintains fixed‑size arrays for up to 128 PMTs and implements the required virtual methods of the base class. Geometry can be updated via set_xyz; the Reset method is defined but performs no action in this version.

Note
The ClassDefOverride macro integrates the class with the ROOT object I/O system, assigning a version number of 1.

private: float pmt_x[128]{}; float pmt_y[128]{}; float pmt_z[128]{}; float pmt_r[128]{}; float pmt_phi[128]{};
