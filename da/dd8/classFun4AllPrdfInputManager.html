Detailed Description

Fun4AllPrdfInputManager handles PRDF input files within the Fun4All framework It derives from Fun4AllInputManager and provides methods to open close and iterate over events stored in PRDF format The manager maintains counters for total and per file events tracks the current segment and interfaces with the node tree via a top PHCompositeNode Event objects are accessed through an Eventiterator and synchronized using a SyncObject Public methods include fileopen fileclose run Print ResetEvent PushBackEvents GetSyncObject SyncIt HasSyncObject and GetString Internal state variables store segment index event counts node pointers and synchronization objects
