Detailed Description

PHG4EPDDetector implements the EP detector geometry and hit handling within the Fun4All framework The constructor registers the detector with the subsystem node and parameter set ConstructMe builds the logical volumes and places them into the world volume IsInDetector returns a non zero identifier when a physical volume belongs to the EP detector module_id_for static overload computes a unique module identifier from tile sector and arm indices module_id_for instance overload extracts the module identifier from a physical volume SuperDetector setter stores the name of the parent detector grouping SuperDetector getter returns the stored parent detector name GetDisplayAction provides access to the optional display action object Private helper construct_block creates an extruded solid for a given block index Member variables hold the display action pointer parameter pointer active and support flags sets of logical volumes maps of volumes and the superdetector name
