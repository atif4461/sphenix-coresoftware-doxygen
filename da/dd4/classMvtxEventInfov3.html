Detailed Description

MvtxEventInfov3 provides event level information for the MVTX detector It stores sets of strobe and L1 bunch crossing identifiers The class implements the PHObject interface allowing cloning resetting identification and validity checks Accessors return the number of stored strobes and L1s and the full sets Mutators insert new identifiers into the corresponding sets
