Detailed Description

brief Stepping action for cylinder detector in sPHENIX simulation this class implements the user stepping callback for the cylinder subsystem it records energy loss and hit information in the G4 hit container it provides options to save light yield and to store all hits interface pointers to the node tree are set via SetInterfacePointers the class holds pointers to the subsystem detector and parameters and manages temporary storage of hit and shower objects during stepping
