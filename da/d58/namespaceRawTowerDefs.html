Detailed Description

Namespace with functions to encode / decode CaloTowerID. The highest 8 bits of the tower ID encode a unique ID for the calorimeter the tower is in. The lower 24 bits uniquely identify the tower within a calorimeter.

Encode a raw tower identifier from a calorimeter ID and tower index.

This utility packs the calorimeter identifier and a tower index into a single RawTowerDefs::keytype value using the bit layout defined in RawTowerDefs. The tower index must fit within the 24‑bit field (maximum value 0xFFFFFF). If the index exceeds this limit an error message is printed and the program terminates.
