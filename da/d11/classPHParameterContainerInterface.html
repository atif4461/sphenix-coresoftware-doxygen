Detailed Description

Abstract interface for detector parameter containers.

Manages named parameters of type double, int and string, provides default values, and supports interaction with the Fun4All node hierarchy. The class owns a PHParametersContainer and a PHParameters object for defaults, and stores macro overrides in a map keyed by detector id. Copy construction and assignment are disabled to avoid unintended duplication of internal pointers.

Typical usage:

  • Derive from this class and implement SetDefaultParameters().
  • Call CreateInitialize(detid) to allocate containers.
  • Use set_*_param and get_*_param to modify parameters from macros.
  • SaveToNodeTree or PutOnParNode to place the container in the run or parameter node trees.
Note
The interface assumes ownership transfer to the Fun4All node system; objects are deleted by the framework when the node tree is cleared. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
