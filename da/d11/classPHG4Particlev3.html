Detailed Description

Class representing a Geant4 particle with ion specific properties Inherits from PHG4Particlev2 and adds atomic mass number and atomic number Provides methods to set and get atomic mass number atomic number ion charge and excitation energy Overrides identification cloning and ion classification methods
