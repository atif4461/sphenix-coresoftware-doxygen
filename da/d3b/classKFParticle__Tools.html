Detailed Description

Utility class for KFParticle based resonance reconstruction.

Provides methods to construct primary vertices, daughter particles and intermediate resonances from track and vertex maps. Implements track quality selection, impact parameter calculation, DIRA evaluation, flight distance chi‑square, and combination building for mother candidates. Supports PID handling via dE/dx fits and configurable selection criteria such as mass windows, decay length, decay time, transverse momentum and chi‑square cuts. Inherits protected KFParticle_MVA to access multivariate analysis tools. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
