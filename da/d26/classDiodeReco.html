void DiodeReco::PedestalCorrected ( int low_bin = -1,
int high_bin = 9999 )

Apply pedestal correction to the ADC waveform by subtracting the median pedestal value.

The method selects ADC samples within the range [low_bin, high_bin), computes the median of these samples, and subtracts this median (pedestal) from all entries of the ADC vector.

Parameters
low_binLower index of the pedestal region (inclusive). Values below zero are clamped to zero.
high_binUpper index of the pedestal region (exclusive). Values larger than the ADC size are clamped to the size.
Returns
None (the ADC vector is modified in place). This comment was generated by openai/gpt-oss-120b at temperature 0.1.
