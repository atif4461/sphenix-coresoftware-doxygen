Detailed Description

PHG4OuterHcalSubsystem implements the outer hcal subsystem for the g4 simulation It derives from PHG4DetectorSubsystem and provides detector stepping and display actions The class creates geometry stepping action and hit nodes during initialization It processes events by forwarding hit lists to the stepping action Light correction parameters can be set via SetLightCorrection This comment was generated by openai/gpt-oss-120b at temperature 0.1.
