Detailed Description

Manages calibration workflow within Fun4All framework Provides registration of histograms with associated CalReco objects Handles run and event timestamps and synchronizes calibration information with bookkeeping database Supports creation copying and overwriting of calibration tables as well as status queries and automatic run range updates Implements singleton access pattern via instance method Extends Fun4AllServer to integrate with event processing loop
