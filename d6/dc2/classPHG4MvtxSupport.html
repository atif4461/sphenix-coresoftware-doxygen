Detailed Description

Class responsible for constructing the mechanical support structures of the MVTX detector within the Geant4 simulation It creates logical volumes assemblies and cable bundles for end wheels cone layers CYSS and service barrel Materials are defined and optional overlap checking can be enabled The support geometry is built using the associated detector and display action objects * This comment was generated by openai/gpt-oss-120b at temperature 0.1.
