◆ process_event()

int PHG4MicromegasHitReco::process_event ( PHCompositeNode * topNode)
overridevirtual

event processing

Process a single event converting Geant4 Micromegas hits to reconstructed tracker hits.

This method retrieves the necessary input nodes (G4 hit container, geometry, tracker hit set container, and truth association), iterates over all Micromegas layers and their Geant4 hits, applies time window cuts, transforms hit positions from world to local coordinates, simulates primary electron drift, diffusion, and additional smearing, distributes charge among adjacent strips, applies gain, and fills the tracker hit set container. It also records the association between reconstructed hits and the originating Geant4 hit.

Parameters
topNodePointer to the top-level PHCompositeNode containing the full node tree for the current event.
Returns
Fun4AllReturnCodes::EVENT_OK on successful processing of the event.
