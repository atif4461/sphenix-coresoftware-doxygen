Detailed Description

Brief Subsystem implementation for the SPACAL detector in the Geant4 simulation

This subsystem creates the SPACAL detector geometry stepping action and associated hit nodes It integrates with the Fun4All framework by providing InitRunSubsystem process_event and accessor methods for the detector stepping action and display action The subsystem also supports optional cosmic ray setup via the CosmicSetup flag and allows printing of internal state

Note The detector object derives from PHG4Detector the stepping action derives from PHG4SteppingAction and the display action derives from PHG4DisplayAction
