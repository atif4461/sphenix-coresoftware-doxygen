Detailed Description

TowerInfoContainerSimv2 provides a container for simulated tower information It inherits from TowerInfoContainer and implements detector specific methods The class manages a TClonesArray of TowerInfoSimv2 objects It supports cloning identification reset and key encoding decoding Detector identifier is stored and accessible via get_detectorid
