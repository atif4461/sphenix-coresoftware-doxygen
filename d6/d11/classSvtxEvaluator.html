Detailed Description

Compares reconstructed tracks to truth particles.

Module evaluates svtx reconstruction performance within fun4all framework Generates ntuples containing information on tracks clusters hits vertices and seeds User can enable or disable evaluation of specific data types such as info vertex gpoint g4hit hit cluster g4cluster gtrack track gseed Strict mode forces additional consistency checks Options exist to use initial vertex or genfit vertex for dca calculations Light evaluation modes reduce output size Embedded and primary particle scanning can be activated The class follows the subsysreco lifecycle methods init initrun process event end This comment was generated by openai/gpt-oss-120b at temperature 0.1.

Plan: This module will trace the reconstructed tracks back to the greatest contributor Monte Carlo particle and then test one against the other.
