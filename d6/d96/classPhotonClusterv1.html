PhotonClusterv1 - extends RawClusterv1 with photon-ID helpers.

Photon cluster data container extending RawClusterv1

Provides photon specific properties such as shower shape parameters and implements photon selection cuts Inherits generic cluster interface and overrides reset identification and cloning methods Stores named shower shape values in an internal map This comment was generated by openai/gpt-oss-120b at temperature 0.1.
