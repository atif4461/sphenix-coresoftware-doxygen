void SingleTpcTimeFrameInput::CreateDSTNode ( PHCompositeNode * topNode)
overridevirtual

Create the DST node hierarchy for TPC raw hit data.

Parameters
topNodePointer to the top level PHCompositeNode of the current node tree.

The routine searches for a DST composite node under the supplied top node, creates it if missing, then ensures a TPC composite sub‑node exists within DST. Finally it obtains (or creates) a TpcRawHitContainer named by m_rawHitContainerName and attaches it to the TPC node as a PHIODataNode so that the container can be written to the DST.
