◆ process_event()

int WeightedFitter::process_event ( PHCompositeNode * top_node)
overridevirtual

Process a single event within the Fun4All framework.

Retrieves required nodes from the supplied top node, selects the appropriate TrackSeedContainer based on the configured track source, and iterates over each TrackSeed to extract cluster keys, build measurement points, perform a weighted fit, optionally refit using the primary vertex, and store the resulting track in the internal maps. The method updates the internal track identifier counter and returns a Fun4All status code.

Parameters
top_nodePointer to the top PHCompositeNode of the current event.
Returns
Fun4AllReturnCodes::EVENT_OK on successful processing, Fun4AllReturnCodes::ABORTEVENT if node retrieval fails, the TrackSeedContainer is missing, or any step in the track processing chain aborts for a given seed. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
