Detailed Description

===========================================================================

Base histogram filler for TrksInJetQA module

This class consolidates all common functionality for Histogram Fillers deployed in the TrksInJetQA module. The histogram fillers (as the name implies) define how to fill histograms for a specific population of objects:

  • Either those in-jets,
  • Or all of them ("inclusive").
