TrkrDefs::cluskey TrkrDefs::getClusKeyHi ( const TrkrDefs::TrkrId trkrId,
const uint8_t lyr )

Type alias representing a unique cluster identifier within the tracking system.

The cluskey encodes the combination of detector ID, layer, and cluster index into a single integral value used throughout the reconstruction framework.

Compute the highest possible cluster key for a given detector and layer.

This function generates the hit set key for the specified detector ID and the next layer (layer + 1), shifts it to the cluster ID field, and subtracts one to obtain the maximum cluster key value that can be assigned within the original layer.

Parameters
trkrIdIdentifier of the tracking detector subsystem.
lyrZero‑based layer index for which the upper cluster key bound is required.
Returns
The maximum cluster key (cluskey) that can be used for the specified detector and layer. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
