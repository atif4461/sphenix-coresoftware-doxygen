Detailed Description

FillClusMatchTree module fills a ROOT TTree with track cluster matching information The module interfaces with the Fun4All node tree to retrieve truth reconstructed tracks clusters and matching containers It creates output branches for event level counters track kinematics cluster properties and hit maps Optional flags control filling of cluster collections verbose hit information and unmatched Svtx tracks The class owns a TrkrClusterIsMatcher instance used to evaluate matching quality and a TrackClusEvaluator for derived quantities Lifecycle methods Init InitRun process_event End manage node creation and cleanup Helper methods clear clusvecs and print mvtx diagnostics provide internal diagnostics This comment was generated by openai/gpt-oss-120b at temperature 0.1.
