ClusKeyIter ClusKeyIter::end ( ) const

Return an iterator positioned at the end of the cluster key collection for the associated track.

The iterator is initialized based on the detector type of the track: if the track contains a TPC seed, the end iterator of the TPC cluster keys is used; otherwise, if the track is in silicon, the end iterator of the silicon cluster keys is used. If the iterator has no data, a default-constructed iterator is returned.

Advance the cluster key iterator to the next element.

If the iterator has no associated data the function returns immediately. After incrementing the internal iterator, when iterating over silicon clusters and the track also contains TPC clusters, reaching the end of the silicon range switches the iterator to the TPC seed cluster range.
