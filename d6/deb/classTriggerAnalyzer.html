Detailed Description

TriggerAnalyzer class provides methods to decode trigger information from the node tree decodeTriggers extracts trigger data from the supplied top node didTriggerFire returns true if the specified trigger fired using name or bit checkRawTrigger returns true if the raw trigger condition is satisfied using name or bit getTriggerPrescale returns the prescale value for the given trigger name or bit getTriggerName returns the trigger name associated with the given bit getTriggerRawScalers returns the raw scaler count for the specified trigger getTriggerLiveScalers returns the live scaler count for the specified trigger getTriggerScalers returns the combined scaler count for the specified trigger UseEmulator enables or disables emulator mode Print outputs the current trigger configuration This comment was generated by openai/gpt-oss-120b at temperature 0.1.
