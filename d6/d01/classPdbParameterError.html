Detailed Description

brief Represents a parameter with an associated error

Inherits from PdbParameter and adds storage for the error of the parameter value Provides accessor and mutator for the error and overrides the print method to include the error information The default constructor is protected to enforce creation with a name and nominal value The error is initialized to NaN
