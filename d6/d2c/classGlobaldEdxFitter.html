◆ get_minimum_new()

double GlobaldEdxFitter::get_minimum_new ( )

Find the optimal parameter A that minimizes the fit quality using a Minuit2 minimizer.

This method configures a ROOT Minuit2 minimizer, sets the fitting function to the member get_fitquality_functor, and performs a minimization starting from an initial guess of 20 with a step size of 0.01. The minimizer is limited to one million function calls, ten thousand iterations, and a tolerance of 0.1. After convergence the method returns the fitted value of the parameter A.

Returns
The value of parameter A that gives the minimum fit quality. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
