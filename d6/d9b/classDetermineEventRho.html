Detailed Description

DetermineEventRho analysis module for background rho estimation in Fun4All Provides configuration of jet algorithm eta ranges ghost area and input sources Supports multiple rho calculation methods and optional output node naming Implements standard Fun4All lifecycle methods InitRun and process event Creates required nodes in the PHCompositeNode tree and computes rho using fastjet selectors
