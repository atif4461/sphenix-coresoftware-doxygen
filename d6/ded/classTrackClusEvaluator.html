std::array< int, 3 > TrackClusEvaluator::find_matches ( )

Find matches between PHG4 and SVTX clusters using the configured matcher.

The method iterates over the sorted key maps of PHG4 (phg4_keys) and SVTX (svtx_keys), compares each pair of clusters that share the same hitset identifier, and records matches based on the TrkrClusterIsMatcher functor. It also optionally accumulates a match statistic if collect_match_statistic is enabled.

Returns
std::array<int,3> containing: [0] number of matched cluster pairs, [1] total number of PHG4 clusters examined, [2] total number of SVTX clusters examined. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

Evaluate cluster level correspondence between a Geant4 truth track and a reconstructed SVTX track.

std::vector< TrkrClusLoc > TrackClusEvaluator::svtx_clusloc_unmatched ( )

Retrieve the locations of SVTX clusters that have not been matched.

This routine iterates over the stored SVTX cluster keys and selects those entries for which the corresponding match flag is false. For each unmatched key the function constructs a TrkrClusLoc object using the g4evalfn::clusloc_SVTX helper and returns the collection of these objects.
