Detailed Description

Abstract base class for calibration channel objects in the PHENIX database.

This class derives from PHObject and defines a pure virtual print method that must be implemented by concrete calibration channel classes.

Note
The class uses the ROOT ClassDefOverride macro for I/O integration.
