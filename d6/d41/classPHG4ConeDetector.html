Detailed Description

PHG4ConeDetector implements a cone geometry detector in the Fun4All framework It inherits from PHG4Detector and provides methods to construct the detector geometry and to query the active cone volume The class holds a pointer to PHParameters the physical volume of the cone a display action object and identifiers for the detector layer and the super detector name Public interface includes a constructor a destructor the ConstructMe method the IsInConeActive query and accessor methods for the super detector name and the layer index This comment was generated by openai/gpt-oss-120b at temperature 0.1.
