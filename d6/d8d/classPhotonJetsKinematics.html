int PhotonJetsKinematics::Init ( PHCompositeNode * topNode)
overridevirtual

Initialize the PhotonJetsKinematics module.

Called during initialization. Typically this is where you can book histograms, and e.g. register them to Fun4AllServer (so they can be output to file using Fun4AllServer::dumpHistos() method).

Parameters
topNodePointer to the top PHCompositeNode (unused in this method).
Returns
Fun4All return code indicating successful initialization.

The method creates a TriggerAnalyzer instance, obtains the global histogram manager, converts the module name to lower case for histogram naming, builds a list of histogram identifiers, optionally appends a user tag, and allocates all required TH1D, TH2D and TH2F objects with appropriate axis titles and drawing options. Verbosity control prints an initialization message.

Enable trigger based event selection and specify trigger ID.
