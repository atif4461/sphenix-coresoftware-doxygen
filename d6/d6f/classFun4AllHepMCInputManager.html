Detailed Description

Fun4AllHepMCInputManager manages HepMC input within the Fun4All framework It inherits from Fun4AllInputManager and PHHepMCGenHelper Provides methods to open close files run events and reset event state Supports optional reading of Oscar format via ReadOscar flag Implements synchronization methods returning default codes Handles conversion from Oscar to HepMC GenEvent Maintains internal counters for total and per file events Stores temporary HepMC file name and top node reference This comment was generated by openai/gpt-oss-120b at temperature 0.1.
