Detailed Description

DumpGl1Packet is a concrete subclass of DumpObject that processes GL1 packet nodes in the Fun4All node tree The constructor takes the name of the node to be dumped The process_Node method is overridden to implement the specific handling of PHNode objects representing GL1 packets
