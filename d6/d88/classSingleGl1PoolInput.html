Detailed Description

Handles GL1 raw hit packet pooling for streaming input.

Provides methods to fill the pool clean up used packets verify pool depth clear event state request additional events print status and create DST node in the Fun4All node tree

Manages mapping between bunch crossing identifiers and packet identifiers and tracks used clock cycles

Member variables m_NumSpecialEvents count of special events processed m_BcoRange configurable range of bunch crossing identifiers m_packet_bco map from bunch crossing id to packet identifier m_Gl1RawHitMap map from clock cycle to vector of Gl1Packet objects m_FEEBclkMap set of clock cycles received from front end electronics m_BclkStack set of clock cycles currently in the processing stack This comment was generated by openai/gpt-oss-120b at temperature 0.1.
