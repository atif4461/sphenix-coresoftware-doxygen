Detailed Description

DumpTruthVertexMap extracts and prints truth vertex map from a specified node.

Inherits from DumpObject and overrides process_Node to access the PHNode containing the truth vertex map within the Fun4All node tree. The constructor receives the node name used to locate the map. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
