◆ LoadHitrates()

int InttCalib::LoadHitrates ( )

Load hitrate information from a ROOT file into the internal hitmap.

The function opens the ROOT file specified by the member variable m_hotmap_png_file, retrieves the tree named hitrate_tree, and reads the branches pid, fee, chp, chn, and hitrate. For each entry the corresponding raw data identifiers are constructed and the hitrate value is stored in the multi‑dimensional map m_hitmap. The event counter m_evts is set to 1.0 to indicate that a single calibration event has been loaded.

Returns
0 on successful loading of the hitrate tree, 1 if the file cannot be opened or the required tree is not found.
