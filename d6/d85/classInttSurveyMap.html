Detailed Description

InttSurveyMap provides a mapping from offline identifier to absolute geometry transforms The map stores Eigen affine transformations for strips sensors and ladders LoadFromFile reads a ROOT file LoadFromCDB reads from the conditions database GetStripTransform GetSensorTransform and GetLadderTransform retrieve the corresponding transform identify prints a short description size returns the number of entries GetAbsoluteTransform returns a pointer to the stored absolute transform Wildcard constant can be used to request any identifier
