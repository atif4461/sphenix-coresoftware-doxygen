int SpinDBOutput::GetDBContent ( SpinDBContent *& spin_cont,
int runnum,
int qa_level )

Retrieve spin database content for a given run and QA level.

This function first checks an internal cache for the requested run number. If the content is cached, it returns the cached object. Otherwise it connects to the database, queries the configured table for the specified run number and QA level, constructs a SpinDBContent object from the result set, stores it in the cache, and copies it to the output pointer.

Parameters
spin_contReference to a pointer that will receive the SpinDBContent object.
runnumInteger identifier of the run to retrieve.
qa_levelInteger quality‑assessment level associated with the run.
Returns
Integer status code: 1 on successful retrieval, 0 on failure, or a predefined error value on database error.
