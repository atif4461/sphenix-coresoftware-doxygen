◆ all_truth_particles()

std::set< PHG4Particle * > SvtxVertexEval::all_truth_particles ( const Vertex * vertex)

Retrieve the set of truth PHG4Particle objects associated with a given vertex.

Parameters
vertexPointer to the Vertex whose constituent truth particles are to be collected.
Returns
Set of pointers to PHG4Particle objects representing the most probable truth particle for each track attached to the vertex.

The method validates internal node pointers and optionally enforces strict null checks based on the _strict flag. If caching is enabled (_do_cache), previously computed results for the same vertex are returned from _cache_all_truth_particles. For each track linked to the vertex, the corresponding SvtxTrack is obtained from the track map. The track evaluator selects the truth particle with the highest number of associated clusters. All selected truth particles are inserted into the result set, which is cached if requested before being returned.
