Detailed Description

Class provides a GL1 pool input manager for streaming data It derives from SingleStreamingInputv2 and implements pool handling methods FillPool loads data for a given bunch crossing identifier CleanupUsedPackets removes packets older than a given bunch clock CheckPoolDepth verifies that sufficient data is available for a given bunch clock ClearCurrentEvent resets internal state for the next event GetSomeMoreEvents attempts to fetch additional events from the source Print outputs internal information optionally filtered by a keyword CreateDSTNode creates the required DST node in the top node hierarchy SetBcoRange configures the range of bunch crossing identifiers to consider SetNegativeWindow defines the backward window size in bunch crossings SetPositiveWindow defines the forward window size in bunch crossings SetTotalEvent limits the total number of events to process Internal containers map bunch crossing identifiers to packets and raw hit collections Additional maps store window limits and bunch numbers and sets track used clock values Flags indicate completion status and last event handling
