Detailed Description

Represents a version two track state in the svtx framework Provides getters and setters for path length position momentum and covariance matrix Implements the standard phobject interface methods identify reset isvalid and cloneme Supports error matrix element access and modification Allows association of a cluster key with the track state Enables naming of the state for identification purposes

SvtxTrackState_v2::SvtxTrackState_v2 ( float pathlength = 0.0)

Initialize a SvtxTrackState_v2 object with a given path length.

This constructor sets the internal path length, zeroes the position vector, initializes the momentum components to NaN, clears the symmetric 6x6 error matrix to zero, and labels the state as "UNKNOWN". The resulting track state can be stored in an SVTX track map for downstream analysis.

Parameters

◆ get_rphi_error()

float SvtxTrackState_v2::get_rphi_error ( ) const
overridevirtual

Compute the error on the r‑phi coordinate of the track state.

The method transforms the Cartesian covariance matrix elements (0,0), (1,1) and (0,1) into the polar r‑phi error using the track position angle. The angle phi is defined as the negative arctangent of y over x, and the error is calculated as the square root of sin²(phi)·σ_xx + cos²(phi)·σ_yy + 2·cos(phi)·sin(phi)·σ_xy.

Returns
The standard deviation of the r‑phi coordinate.
