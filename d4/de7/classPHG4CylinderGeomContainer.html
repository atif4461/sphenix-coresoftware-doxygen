Detailed Description

Container for PHG4CylinderGeom objects indexed by layer

Provides map based storage of cylindrical geometry definitions for detector layers Supports addition retrieval and iteration over stored geometries Stores magnetic field value associated with the geometry collection Implements PHObject interface for identification and ROOT I O
