Detailed Description

Class SingleInttEventInput provides streaming input handling for INTt detector events It derives from SingleStreamingInput and overrides methods for pool management event clearing printing and DST node creation The class maintains packet list beam clock to FEE mappings and raw hit collections Configurable parameters include BCO range and negative BCO offset Accessor methods expose the current beam clock stack and the beam clock to FEE map This comment was generated by openai/gpt-oss-120b at temperature 0.1.
