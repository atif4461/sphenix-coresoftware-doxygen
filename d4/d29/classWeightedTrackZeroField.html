void WeightedTrackZeroField::configure_minimizer ( ROOT::Math::Minimizer & minimizer)
overridevirtual

Modifies the minimizer so it is ready to begin the fit optimization, including setting initial parameters and ranges.

Initialize the ROOT minimizer with seed parameters for a zero field track fit.

The method verifies that the track contains at least two points. It computes an initial direction vector from the first two cluster positions, derives the polar angle theta and azimuthal angle phi, and determines the intercept of the line with the plane z = 0. These values are used to set the minimizer variables "x", "y", "theta", and "phi" with small step sizes. Angular limits are applied to theta (0 to pi) and phi (phi +/- 1.58 rad) to constrain the fit.

Parameters
minimizerReference to a ROOT::Math::Minimizer that will be configured.

◆ get_parameters()

void WeightedTrackZeroField::get_parameters ( double * params) const
overridevirtual

Mutates the members of the given array such that calling set_parameters would result in the same track parameterization (inverse of set_parameters)

Fill an array with the track parameters for a zero magnetic field.

The first two entries of the array are the x and y intercepts of the track. The third entry is the polar angle derived from the z component of the slope vector using the arccosine function. The fourth entry is the azimuthal angle computed from the x and y components of the slope vector using the two‑argument arctangent function.

Parameters
paramspointer to a double array of length at least four where the parameters will be stored. The array is indexed as follows: params[0] = x intercept, params[1] = y intercept, params[2] = polar angle (radians), params[3] = azimuthal angle (radians).
