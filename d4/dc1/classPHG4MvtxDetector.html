Detailed Description

PHG4MvtxDetector implements the MVTX detector geometry and stepping within the Fun4All framework It constructs the detector volumes and registers geometry nodes It manages active layers support and black hole configuration It provides accessors for layer and stave identification and sensor queries It handles optional misalignment application and global displacement parameters It integrates display actions for visualization of detector components
