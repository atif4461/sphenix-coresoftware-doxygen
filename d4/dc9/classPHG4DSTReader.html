void PHG4DSTReader::add_particle ( PHG4DSTReader::record & rec,
PHG4Particle * part )
protected

add a particle and associated vertex if _save_vertex

Copy particle data into a DST record and update bookkeeping.

Parameters
recReference to a record structure containing an array of particle objects and a fill counter.
partPointer to the source PHG4Particle whose fields are to be copied.

The function asserts that the source particle is valid, constructs a new particle object of the appropriate type in the record array at the current index, copies track, vertex, parent, primary identifiers, name, PDG code, momentum components and energy, inserts the vertex identifier into the internal vertex set, and increments the record counter.
