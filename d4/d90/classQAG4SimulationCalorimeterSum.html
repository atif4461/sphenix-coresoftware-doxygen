int QAG4SimulationCalorimeterSum::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run-specific resources and retrieve necessary node data.

This method is invoked at the beginning of each new run. It obtains the PHG4TruthInfoContainer from the node tree, configures calorimeter evaluation stacks for CEMC, HCALIN, and HCALOUT when cluster processing is enabled, and sets up the SvtxEvalStack for track projection processing when requested.

Parameters
topNodePointer to the top-level PHCompositeNode containing the Fun4All node hierarchy for the current job.
