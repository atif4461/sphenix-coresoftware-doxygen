int CentralityReco::FillCentralityInfo ( )

Fill centrality information for minimum bias events using the MBD total charge.

This method determines the centrality bin and centile by comparing the accumulated MBD charge to a predefined centrality map. The resulting values are stored in the CentralityInfo object.

Returns
Fun4All return code indicating successful event processing (EVENT_OK).

◆ FillVars()

int CentralityReco::FillVars ( )

Compute the total MBD charge for the event applying vertex and centrality scaling.

The function iterates over all 128 MBD channels, retrieves each PMT hit, applies charge and time quality cuts, and accumulates the scaled charge into the member variable m_mbd_total_charge. Verbosity control prints diagnostic information.

Returns
Fun4AllReturnCodes::EVENT_OK indicating successful processing of the event.
