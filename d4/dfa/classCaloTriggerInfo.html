Detailed Description

CaloTriggerInfo stores trigger level calorimeter information for multiple patch geometries It defines virtual setters and getters for EMCal 2x2 4x4 patches and FullCalo patches of sizes 0p2x0p2 0p4x0p4 0p6x0p6 0p8x0p8 1p0x1p0 The interface allows derived classes to implement storage of energy eta and phi for best and second best patches Inherits from PHObject and overrides identify and isValid methods for framework integration This comment was generated by openai/gpt-oss-120b at temperature 0.1.
