Detailed Description

HIJINGFlipAfterburner module flips the longitudinal direction of generated particles It inherits from SubsysReco and is called for each event in the Fun4All framework The process_event method retrieves the HepMC event from the node tree and applies a Z axis sign inversion when enabled The flipZDirection helper performs the sign change on all particle momenta The doFlip flag controls whether the transformation is applied
