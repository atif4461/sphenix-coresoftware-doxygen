Detailed Description

TpcSiliconQA analysis module for correlation of TPC and silicon seeds Inherits from SubsysReco and overrides InitRun process_event EndRun Creates histograms of crossing positions and differences in x y z phi eta between TPC and silicon seeds Provides user configurable cuts for x y eta phi via setXCut setYCut setEtaCut setPhiCut Histograms are booked in createHistos and stored as TH1 and TH2 pointers Event counter m_event tracks processed events This comment was generated by openai/gpt-oss-120b at temperature 0.1.
