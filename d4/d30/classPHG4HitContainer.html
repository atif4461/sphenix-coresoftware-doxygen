◆ genkey()

PHG4HitDefs::keytype PHG4HitContainer::genkey ( const unsigned int detid)

Generate a unique hit key for a given detector identifier.

This routine composes a 64‑bit key by combining the detector id with the next available hit id within that detector layer. It verifies that the detector id fits within the allocated bit field, obtains the current maximum hit id for the detector, increments it, and forms the final key. The function also checks that the generated key does not already exist in the internal hit map and aborts if a duplicate is found.

Parameters
detidUnsigned integer representing the detector identifier.
Returns
A PHG4HitDefs::keytype value encoding the detector id and the new hit id. The returned key is guaranteed to be unique within the container.
