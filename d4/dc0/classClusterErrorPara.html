ClusterErrorPara::error_t ClusterErrorPara::get_fix_tpc_cluster_error ( TrkrCluster * cluster,
TrkrDefs::cluskey key )

Compute fixed TPC cluster error for a given cluster.

This method extracts the detector layer from the cluster key, obtains the raw phi and z errors using the MVTX error models, applies layer‑specific fine‑pull scaling factors, enforces a minimum error value, and returns the squared errors as a pair.

Parameters
clusterPointer to the TrkrCluster object whose errors are to be evaluated.
keyCluster identifier (TrkrDefs::cluskey) used to determine the detector layer.
Returns
A pair (phiError², zError²) representing the squared uncertainties in the phi and z coordinates for the specified cluster.

double ClusterErrorPara::mvtx_phi_error ( TrkrCluster * cluster) const

Compute the azimuthal (phi) error for a MVTX cluster.

The error is derived from the intrinsic MVTX phi pitch error scaled by a size dependent factor. Specific scale factors are applied for clusters of size 1x1, 2x1, 1x2, 2x2, 2x3, 3x2 and 3x3. For all other size combinations the phi pitch error is multiplied by the phi size. The resulting value is further scaled by the global MVTX scale factor.

Parameters
clusterpointer to the TrkrCluster object whose phi error is to be calculated
Returns
phi error value after applying size dependent scaling and global scale
