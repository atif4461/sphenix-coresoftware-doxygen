Detailed Description

Reads charge maps from ROOT histograms or files and provides interpolation utilities.

The class maintains a fixed‑size 3‑D array of charge values and an optional charge‑density TH3 histogram. It can ingest source charge or ADC histograms, apply axis and content scaling, and generate internal representations suitable for fast lookup.

Construction

  • Default constructor creates an object with placeholder geometry.
  • Parameterized constructor defines radial (r), azimuthal (phi), and longitudinal (z) binning and bounds.

Input handling

  • ReadSourceCharge loads a charge histogram from a file or an existing TH3, applying optional axis and content scaling.
  • ReadSourceAdc loads ADC counts together with a gain histogram to convert to charge.

Geometry configuration

  • SetOutputParameters defines full binning and bounds in one call.
  • SetOutputBounds adjusts the physical limits without changing bin counts.
  • SetOutputBins changes the number of bins while preserving current bounds.

Accessors

  • GetChargeInBin returns the stored charge for integer bin indices.
  • GetChargeAtPosition interpolates the charge at a continuous (r,phi,z) location.
  • FillChargeHistogram writes the internal charge array into a user‑provided TH3.

Interpolation safety

  • CanInterpolateAt (static and member) checks whether a point lies inside the histogram bounds before performing trilinear interpolation.

Internal workflow

  • RegenerateCharge rebuilds the internal MultiArray when geometry changes.
  • RegenerateDensity recomputes the charge‑density histogram from the source data.
Note
The class does not take ownership of external ROOT histograms; they must remain valid for the lifetime of the ChargeMapReader instance. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
