Detailed Description

Tpc3DClusterizer performs three dimensional clustering of TPC laser hits within the Fun4All framework.

The module inherits from SubsysReco and implements the standard lifecycle callbacks InitRun process_event ResetEvent and End It accesses hit data from TrkrHitSetContainer and stores resulting clusters in LaserClusterContainer Geometry information is obtained from ActsGeometry and PHG4TpcGeomContainer Clustering uses boost geometry rtree structures and calculates cluster parameters based on ADC sums and size thresholds Configurable parameters include pedestal value minimum cluster size minimum ADC sum debug and output control Timing of major steps is measured with PHTimer objects This comment was generated by openai/gpt-oss-120b at temperature 0.1.
