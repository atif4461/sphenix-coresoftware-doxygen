◆ get_innermost_truth_hit()

PHG4Hit * SvtxTruthEval::get_innermost_truth_hit ( PHG4Particle * particle)

Retrieve the innermost truth hit associated with a given Geant4 particle.

The function scans all truth hits belonging to the supplied particle and selects the hit with the smallest radial distance (sqrt(x^2+y^2)) from the detector origin, using the entry point coordinates of each hit. It first verifies that the required node pointers are available; if not, an error counter is incremented and a null pointer is returned. In strict mode the input particle pointer must be non‑null (asserted), otherwise a null pointer also triggers an error count and a null return.

Parameters
particlePointer to the PHG4Particle for which the innermost truth hit is requested.
Returns
Pointer to the PHG4Hit with the smallest radius, or nullptr if no hit is found or if an error condition occurs.
