Detailed Description

SubsysReco module to produce flat TTrees and QA histograms for sEPD calibration.

This module extracts event-level info (vertex, centrality) and sEPD tower-level info (charge, phi, channel ID), applying basic event selections (Minimum Bias, Z-vertex) and tower-level cuts (charge threshold, zero-suppression).
