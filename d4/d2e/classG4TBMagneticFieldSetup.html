void G4TBMagneticFieldSetup::SetStepper ( )

Configure the integration stepper for the magnetic field propagation.

This method selects a G4 integration stepper based on the internal fStepperType identifier, instantiates the corresponding G4Stepper object with the magnetic field equation (fEquation) and a fixed number of variables (nvar = 8). Supported steppers include: G4ExplicitEuler, G4ImplicitEuler, G4SimpleRunge, G4SimpleHeum, G4ClassicalRK4 (default), and G4CashKarpRKF45. Stepper types that are not compatible with the magnetic field (e.g., G4RKG3_Stepper, G4HelixExplicitEuler, G4HelixImplicitEuler, G4HelixSimpleRunge) are reported and left unset.

The method prints a status message when the global verbosity level is greater than zero, showing the chosen stepper and the configured minimum step size (fMinStep). If no valid stepper is created, the program terminates with an error.
