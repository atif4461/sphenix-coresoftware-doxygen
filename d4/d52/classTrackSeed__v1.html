Detailed Description

TrackSeed v1 implementation of the abstract TrackSeed interface Provides storage for seed parameters and associated cluster keys Supports copy construction assignment cloning and reset operations Accessors return charge momentum components and geometric seed parameters Modifiers allow setting of seed parameters and management of cluster key set
