Detailed Description

Track seeding subsystem for reconstruction.

Base class for track seeding.

Provides interface for initializing node pointers processing events and finalizing. Configurable track map name hit association flag truth cluster usage and iteration count. Derived classes must implement Setup Process and End specific to tracker.
