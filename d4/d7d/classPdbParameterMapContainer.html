int PdbParameterMapContainer::WriteToFile ( const std::string & detector_name,
const std::string & extension,
const std::string & dir = "." )

write PdbParameterMapContainer to an external file with root or xml extension.

Write a copy of the parameter map container to a ROOT file with a standardized name.

The function constructs a file path using the provided detector name, file extension, and directory, appends a timestamp and bank identifier, and writes a deep copy of the current container to a newly created ROOT file. XML float formatting is temporarily set to extended precision to preserve parameter values. After writing, the function restores the original XML format, closes the file, and pauses briefly to avoid duplicate insertion times.

Parameters
detector_nameName of the detector used in the output file name.
extensionFile extension (e.g., "root") for the output file.
dirDirectory path where the file will be created.
