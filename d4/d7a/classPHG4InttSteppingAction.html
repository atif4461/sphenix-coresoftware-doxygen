Detailed Description

Implements the Geant4 stepping action for the INTT detector Creates PHG4Hit objects for active sensor layers and stores them in the hit container node Also creates absorber hits and stores them in a separate absorber hit container Uses parameter maps to translate geometry identifiers to tracker layers ladder types and strip positions Provides interface methods to set node pointers and hit node names during initialization

Destructor cleans up the last hit object.

Deletes the internal hit pointer. If the last hit was a zero energy deposit it is reset but memory remains allocated, requiring explicit deletion. If the hit was already saved the pointer is null and deletion is a no‑op.
