Detailed Description

Module that builds truth level clusters from Geant4 particles and hits Inherits from SubsysReco and integrates with the Fun4All node tree Implements InitRun to acquire geometry and truth containers, process_event to create clusters per event, and End for finalization Provides a flag to restrict clustering to primary track clusters only Private utilities fetch node pointers, collect all truth clusters and hits for a given particle, map hits to detector layers, compute cluster size and ADC values, and determine TPC sector Stores pointers to reconstructed cluster container, G4 truth info, hit containers for each subsystem, geometry containers and Acts geometry Contains layer specific error parameters for MVTX INTT TPC and MMS detectors This comment was generated by openai/gpt-oss-120b at temperature 0.1.
