std::set< PHG4Particle * > SvtxHitEval::all_truth_particles ( TrkrDefs::hitkey hit_key,
const TrkrDefs::TrkrId trkrid )

Retrieve all truth PHG4Particle objects associated with a given hit.

This method queries the truth evaluation subsystem to obtain the set of PHG4Particle pointers that contributed to the specified detector hit. It respects the module's strictness and caching settings, handling missing node pointers or invalid hit keys by incrementing the error counter and optionally printing diagnostic messages.

Parameters
hit_keyIdentifier of the hit for which truth particles are requested.
trkridTracker identifier indicating the subsystem (e.g., MVTX, INTT).
Returns
A set of pointers to PHG4Particle objects linked to the hit. Returns an empty set if node pointers are unavailable, the hit key is invalid, or no associated particles are found.

void SvtxHitEval::next_event ( PHCompositeNode * topNode)

Reset internal caches and prepare for a new event.

This method clears all cached containers that store truth hits, truth particles, hit‑to‑particle associations, and energy contributions. It then forwards the event initialization to the internal truth evaluation object and obtains the required node pointers from the supplied top node.

Parameters
topNodePointer to the top PHCompositeNode of the current event.
