void TpcSpaceChargeMatrixInversion::calculate_distortion_corrections ( const InversionMode inversionMode = InversionMode::FullInversion)

calculate distortions by inverting stored matrices, and save relevant histograms

Compute distortion correction histograms from space charge matrices.

Parameters
inversionModeselects the inversion strategy full inversion reduced inversion phi or reduced inversion z

The method checks that a matrix container is present extracts the grid dimensions creates three dimensional histograms for entry counts and distortion components phi z r iterates over all grid cells and for each cell with sufficient entries builds Eigen matrices from the container solves the linear system according to the selected inversion mode fills the histograms with the solution and its covariance finally splits the histograms at zero z adds guarding bins and stores them in a TpcDistortionCorrectionContainer This comment was generated by openai/gpt-oss-120b at temperature 0.1.
