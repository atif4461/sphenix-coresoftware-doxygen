Detailed Description

TowerInfoContainer provides an abstract interface for collections of TowerInfo objects It defines a map from unsigned integer keys to TowerInfo pointers and supplies methods for encoding and decoding tower indices for different detector subsystems The class enumerates supported detectors and offers virtual accessors for tower retrieval size query and geometric binning Subclasses implement detector specific encoding schemes and concrete tower storage This comment was generated by openai/gpt-oss-120b at temperature 0.1.
