Detailed Description

PHPy8ParticleTrigger provides a trigger based on particle selection and kinematic cuts Inherits from PHPy8GenTrigger Allows adding particles and parents by name or PDG id Supports high and low cuts on transverse momentum rapidity pseudorapidity absolute pseudorapidity momentum longitudinal momentum Option to restrict to stable particles only Configuration can be printed with PrintConfig

Set the upper momentum threshold for the particle trigger.

If a low‑momentum cut is currently active, the method switches to using both low‑ and high‑momentum cuts by enabling the combined cut flag and disabling the low‑momentum‑only flag.
