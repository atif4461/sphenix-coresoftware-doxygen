Detailed Description

TrackResiduals analysis module for residual calculation and tree output Inherits from SubsysReco and integrates with Fun4All node tree Provides configuration setters for output file alignment map names and selection flags Implements InitRun process_event and End callbacks to create ROOT trees with event track cluster hit and vertex information Utilizes Acts geometry TpcClusterMover and error parameterization to fill state vectors and perform line and circle fits This comment was generated by openai/gpt-oss-120b at temperature 0.1.
