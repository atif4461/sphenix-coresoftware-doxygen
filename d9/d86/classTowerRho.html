Detailed Description

Base class for tower rho calculations.

Provides an interface for setting and retrieving the background energy density rho, its uncertainty sigma, and the calculation method. The method enumeration includes NONE AREA and MULT options.

Derived classes should implement the virtual setters and getters to store actual values. Inherits from PHObject to integrate with the Fun4All node management system This comment was generated by openai/gpt-oss-120b at temperature 0.1.
