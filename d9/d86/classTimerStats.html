Detailed Description

TimerStats module records timing information for each event and writes results to a ROOT file It inherits from SubsysReco to integrate with Fun4All framework InitRun initializes the output tree and event counter process_event fills timing data for the current event End finalizes the tree and writes it to the specified output file OutFileName sets the name of the output ROOT file
