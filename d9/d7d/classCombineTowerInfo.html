Detailed Description

Brief CombineTowerInfo merges tower information from two input nodes into a single output node for a given detector The class inherits from SubsysReco and implements InitRun and process_event callbacks used by Fun4All InitRun creates or retrieves the required node hierarchy under the top node process_event accesses the input TowerInfoContainer objects and combines their contents and stores the result in the output container User can configure the names of the input nodes output node and detector via the provided setter methods Private helper CreateNodes builds the node structure if missing Member variables store the configured node names and pointers to the input and output TowerInfoContainer objects
