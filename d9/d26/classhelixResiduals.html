int helixResiduals::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run-specific resources for helix residual analysis.

This method is invoked at the start of each new run. It forwards the initialization call to the internal fitter, opens an output ROOT file, creates a TNtuple for storing seed residuals, and retrieves required nodes from the supplied top node.
