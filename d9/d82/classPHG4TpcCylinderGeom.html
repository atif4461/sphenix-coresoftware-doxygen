Detailed Description

PHG4TpcCylinderGeom provides geometry description for the TPC cylinder It inherits from PHG4CylinderGeom and implements layer radius thickness and binning accessors The class stores z and phi binning parameters and sector specific bias arrays Methods allow conversion between bin indices and physical coordinates and retrieval of bounds Static constant NSides defines number of readout sides The class supports streaming via operator<< and ROOT class definition macro
