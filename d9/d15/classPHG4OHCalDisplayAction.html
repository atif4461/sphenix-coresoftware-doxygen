Detailed Description

Class provides display actions for the outer hadron calorimeter geometry It inherits from the generic display action base class Constructor takes a name identifier Destructor cleans up allocated visualization attributes ApplyDisplayAction is called for each physical volume during geometry traversal SetMyTopVolume stores the top level physical volume of the detector AddScintiVolume registers a scintillator logical volume for custom visualization AddSteelVolume registers the steel logical volume AddChimSteelVolume registers the chimney steel logical volume AddSupportRingVolume registers support ring logical volumes Private data members hold pointers to the top volume steel and chimney steel volumes Vectors store visualization attribute objects Sets store collections of scintillator and support ring logical volumes This comment was generated by openai/gpt-oss-120b at temperature 0.1.
