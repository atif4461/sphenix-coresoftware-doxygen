Detailed Description

DumpParticleFlowElementContainer dumps the contents of a ParticleFlowElementContainer node.

This class derives from DumpObject and overrides the process_Node method to traverse a PHNode representing a ParticleFlowElementContainer. The constructor takes the name of the node to be processed. The destructor is defaulted.

Parameters
NodeNameName of the node in the Fun4All node tree that holds the ParticleFlowElementContainer.
Returns
The overridden process_Node method returns an integer status code indicating success or failure of the dump operation.
