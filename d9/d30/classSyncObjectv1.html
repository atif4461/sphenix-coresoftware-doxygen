Detailed Description

SyncObjectv1 provides concrete implementation of the SyncObject interface Default constructor creates empty object Explicit constructor copies data from another SyncObject CloneMe returns a new copy of this object Reset clears the event related fields identify writes a textual representation of the object to the provided stream isValid returns a non zero value when the object contains valid data EventCounter sets the internal event counter EventNumber sets the internal event number RunNumber sets the internal run number SegmentNumber sets the internal segment number EventNumber returns the stored event number EventCounter returns the stored event counter RunNumber returns the stored run number SegmentNumber returns the stored segment number
