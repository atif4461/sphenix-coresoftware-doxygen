Detailed Description

TrackFittingQA provides quality assurance histograms for track fitting It inherits from SubsysReco and implements the standard Fun4All callbacks The module retrieves a SvtxTrackMap from the node tree and fills histograms Configurable node name track map node name and histogram ranges and selection cuts Histograms include quality momentum pt pt error eta phi and detector state counts Cuts are applied on quality momentum pt absolute eta number of states and bunch crossing This comment was generated by openai/gpt-oss-120b at temperature 0.1.

int TrackFittingQA::Init ( PHCompositeNode * )
overridevirtual

Initialize the TrackFittingQA module and book QA histograms.

This method obtains the QAHistManager, creates a set of histograms for positively and negatively charged tracks (quality, momentum, transverse momentum, pt error, eta, phi‑eta, and detector state counts), registers them with the manager, and configures their visual attributes.
