Detailed Description

HFTrackEfficiency analysis module computes heavy flavor track reconstruction efficiency It accesses truth information decay finder input and output track maps and optional ntuple Configuration methods allow setting node names output file trigger on decay and matching percentage Internal counters track total and accepted decays and store kinematic variables for truth and reconstructed tracks The module creates a ROOT tree with branches for mother and daughter kinematics and vertex positions
