MbdRawContainerV2::MbdRawContainerV2 ( )

ctor

Construct a MbdRawContainerV2 and allocate a TClonesArray for MbdRawHitV2 objects.

The container holds a TClonesArray named MbdRawHits with a pre‑allocated size defined by MbdDefs::MBD_N_PMT. Each element of the array is an instance of MbdRawHitV2 representing a single hit with members for pmt, adc, ttdc, and qtdc. The constructor does not take any arguments and does not return a value.
