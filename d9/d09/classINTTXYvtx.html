std::pair< std::vector< TH2 * >, std::vector< TH1 * > > INTTXYvtx::GetHistFinal ( )

Retrieve the final set of analysis histograms.

This method returns a pair containing two vectors:

  • The first vector holds pointers to TH2 objects representing DCA distance and angular difference histograms for inner and outer phi peaks, as well as the XY vertex histograms and background histograms.
  • The second vector holds pointers to TH1 objects for the final background‑subtracted angular difference histogram.
Returns
A std::pair where the first element is a std::vector of TH2* histograms and the second element is a std::vector of TH1* histograms. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
