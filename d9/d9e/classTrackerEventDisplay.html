Detailed Description

TrackerEventDisplay module for Fun4All.

Implements SubsysReco interface to generate JSON output of tracker hits and clusters for each event Configurable output filename run number and date via constructor arguments Provides control over first event index and toggles for hit and cluster JSON generation Uses internal helper makeJsonFile to traverse the PHCompositeNode tree and write data to a ROOT compatible file Maintains output stream and event counters Designed for offline analysis and event display of cosmic ray data This comment was generated by openai/gpt-oss-120b at temperature 0.1.
