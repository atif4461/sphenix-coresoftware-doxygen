Detailed Description

JetMapv1 implements a concrete JetMap storing jets keyed by identifier It records the reconstruction algorithm and its parameter and tracks source identifiers Provides standard map operations insertion ownership semantics and iteration Supports cloning identification reset and validity interface required by PHObject
