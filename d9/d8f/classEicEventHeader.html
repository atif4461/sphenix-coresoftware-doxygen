Detailed Description

Event header object providing generic property storage for different event generators.

This class derives from PHObject and implements the standard identify copy and reset interfaces required by the Fun4All framework It defines a set of PROPERTY identifiers that can be used to store integer unsigned integer or floating point values The PROPERTY_TYPE enumeration describes the supported data types Helper functions are provided to query the existence of a property retrieve typed values and set values without performing range checks Specific generator related accessors for Milou weight true X and true Q2 and DEMP weight are declared as virtual stubs that can be overridden in derived classes The EvtGen enumeration lists the supported generators Static utility functions allow conversion between PROPERTY identifiers and their string representations as well as type checking

The internal map evInfo can be used to store additional freeform key value pairs This comment was generated by openai/gpt-oss-120b at temperature 0.1.
