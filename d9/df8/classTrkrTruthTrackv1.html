◆ identify()

void TrkrTruthTrackv1::identify ( std::ostream & os = std::cout) const
overridevirtual

Print a summary of the truth track to the given output stream.

The method writes the track identifier, the initial position (X0,Y0,Z0), the kinematic parameters (pseudorapidity, pt, phi) and the list of associated clusters. For each cluster the hit set key and the detector layer are printed, with up to eight entries per line.

Parameters
osOutput stream that receives the formatted track information.
Returns
