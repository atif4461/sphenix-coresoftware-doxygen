Detailed Description

Map Status of Calo Towers.

This Fun4all modules ingests calorimeter towers, and then plots the no. of towers per event and vs. eta/phi for a given status. CaloStatusMapper module maps calorimeter status information into histograms Configurable options include debug flag normalization histograms optional histograms trigger selection mode and trigger id Provides methods for initialization event processing and finalization within Fun4All framework Manages histogram manager trigger analyzer input nodes and event counters This comment was generated by openai/gpt-oss-120b at temperature 0.1.
