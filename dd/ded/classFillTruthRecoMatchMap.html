Detailed Description

FillTruthRecoMatchMap is an analysis module derived from SubsysReco It creates and fills matching containers linking reconstructed tracks to truth particles The module interacts with the node tree to retrieve SvtxPHG4ParticleMap and PHG4ParticleSvtxMap During InitRun it creates required nodes under the top node process_event fills the EmbRecoMatchContainer with match information for each event End writes the filled containers to the output tree This comment was generated by openai/gpt-oss-120b at temperature 0.1.
