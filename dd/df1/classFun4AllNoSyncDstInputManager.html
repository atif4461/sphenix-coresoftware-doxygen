Detailed Description

Fun4AllNoSyncDstInputManager provides a DstInputManager variant that disables synchronization checks and run tree reading to allow embedding Constructor initializes manager with optional name node and top node identifiers Destructor default SyncIt always returns SYNC OK indicating no synchronization required GetSyncObject returns SYNC NOOBJECT indicating absence of sync object NoSyncPushBackEvents forwards to base PushBackEvents setSyncBranches is a no op returning zero NoRunTTree disables runwise TTree reading for embedding SkipForThisManager forwards to PushBackEvents HasSyncObject reports false
