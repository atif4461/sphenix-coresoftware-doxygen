Detailed Description

TriggerPrimitive class provides a container for trigger sums keyed by TriggerDefs TriggerSumKey It inherits from PHObject and implements required virtual methods for reset identification and validity The class defines map types for storing vectors of unsigned int sums and provides accessors for retrieving and adding sums as well as iterating over the stored collections Size method returns number of stored keys
