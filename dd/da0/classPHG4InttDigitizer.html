Detailed Description

PHG4InttDigitizer digitizes simulated INTT energy deposits into ADC counts Implements SubsysReco interface for event processing within Fun4All Uses PHParameterInterface to expose configurable parameters such as noise mean sigma energy per pair adc scaling Handles loading of bad channel map and applies per ladder cell ADC scaling Provides methods for run initialization event processing and finalization Manages random number generator conforming to sPHENIX standards
