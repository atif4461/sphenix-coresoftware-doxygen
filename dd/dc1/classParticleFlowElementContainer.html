Detailed Description

Container for ParticleFlowElement objects indexed by integer identifier.

Manages a map from integer index to ParticleFlowElement pointer Provides methods to add a ParticleFlowElement, retrieve a ParticleFlowElement by index, obtain const and non‑const iterators over all stored elements, reset the container, check validity and identify the container contents Implements the PHObject interface required by the Fun4All framework Size returns the number of stored ParticleFlowElement objects
