Detailed Description

Class EpdGeom provides a generic interface for EP detector geometry It defines virtual setters for longitudinal (z) radial (r) and azimuthal (phi) coordinates and virtual getters that return NaN in the base implementation The class inherits from PHObject and uses the ROOT ClassDefOverride mechanism This comment was generated by openai/gpt-oss-120b at temperature 0.1.
