Generate a MVTX pixel key from detector coordinates.

This function composes a pixel identifier by first creating a hit set key from the specified layer, stave, and chip indices (with a fixed zero sub‑detector index) and then combining it with a hit key derived from the column and row indices.

Parameters
layerZero‑based index of the MVTX layer.
staveZero‑based index of the stave within the layer.
chipZero‑based index of the chip on the stave.
rowRow index of the pixel within the chip.
colColumn index of the pixel within the chip.
Returns
A pixelkey that uniquely identifies the specified pixel.
