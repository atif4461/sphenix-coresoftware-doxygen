◆ fillTruthBranch()

void KFParticle_truthAndDetTools::fillTruthBranch ( PHCompositeNode * topNode,
TTree * m_tree,
const KFParticle & daughter,
int daughter_id,
const KFParticle & kfvertex,
bool m_constrain_to_vertex_truthMatch )

Fill truth information for a daughter particle into the output branches.

This routine extracts the corresponding Monte‑Carlo truth track and vertex from the node tree, computes the true momentum components, total momentum, transverse momentum and the true production vertex. The results are stored in the member arrays indexed by daughter_id. If m_constrain_to_vertex_truthMatch is true, the true impact parameters (full 3‑D and transverse) with respect to the reconstructed vertex are calculated using KFParticle objects.

Parameters
topNodePointer to the top PHCompositeNode containing all event data.
m_tree(unused) Pointer to the output TTree; kept for interface compatibility.
daughterReference to the reconstructed daughter KFParticle.
daughter_idInteger index identifying the daughter in the output arrays.
kfvertexReference to the reconstructed vertex KFParticle associated with the decay.
m_constrain_to_vertex_truthMatchFlag indicating whether to compute true DCA with respect to the reconstructed vertex.
Returns
None (void). The function populates internal member vectors such as m_true_daughter_px, m_true_daughter_py, m_true_daughter_pz, m_true_daughter_p, m_true_daughter_pt, m_true_daughter_id, m_true_daughter_vertex_x/y/z, m_true_daughter_ip, m_true_daughter_ip_xy, and the true primary‑vertex coordinates. Create branches in a ROOT TTree for detector‑specific information of a daughter particle.

This routine adds residual and layer branches for the given daughter when detailed tracking information is requested (m_get_detailed_tracking). It then iterates over the configured subdetectors stored in the Use map and, for each enabled subdetector, invokes initializeSubDetectorBranches to add the appropriate branches.
