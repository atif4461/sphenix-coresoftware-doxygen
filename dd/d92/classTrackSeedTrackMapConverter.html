Detailed Description

TrackSeedTrackMapConverter converts track seeds from the tracking reconstruction into SvtxTrack objects stored in a SvtxTrackMap The module inherits from SubsysReco and is executed during InitRun process_event and End InitRun obtains geometry and field information and locates required nodes in the node tree process_event loops over track seeds in the configured seed containers creates or updates SvtxTrack objects and stores them in the track map End releases geometry resources The converter can be configured to treat cosmic ray tracks and to assume a constant magnetic field The field map name track map name track seed name and cluster map name are set via the corresponding setter methods Private helper functions retrieve node pointers addKeys copy identifier information between seeds and tracks getCosmicCharge estimate charge sign for cosmic tracks based on vertex radius and perform a simple line fit to initialize track parameters Member variables hold pointers to geometry SvtxTrackMap TrackSeedContainer instances for TPC silicon and cluster data the magnetic field strength and configuration flags and the names of the input nodes
