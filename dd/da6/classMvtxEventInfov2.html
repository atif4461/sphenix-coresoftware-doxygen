Detailed Description

MvtxEventInfov2 provides an implementation of the MvtxEventInfo interface for version 2 of the MVTX event information The class stores hardware bunch crossing HB number strobe and L1 bunch crossing identifiers and provides lookup utilities It supports cloning resetting identification and validity checking as required by the PHObject base class Strobe and L1 identifiers are kept in associative containers and can be queried individually or cross referenced This comment was generated by openai/gpt-oss-120b at temperature 0.1.
