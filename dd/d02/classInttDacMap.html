Detailed Description

InttDacMap provides mapping between online channel identifiers and DAC calibration values The class can load calibration data from the conditions database CDB or from a local file and can write the current mapping to a file Accessors return the DAC value for a given FELIX server channel chip channel and ADC range or using raw or offline data structures Verbosity control and default DAC settings are available Internally the mapping is stored in a four dimensional array indexed by FELIX server FELIX channel chip and DAC index
