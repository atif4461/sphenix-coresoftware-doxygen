Detailed Description

Class to calculate running average and RMS Class provides running statistics for a stream of values It maintains a fixed maximum number of recent entries Methods allow clearing the accumulator adding a new value retrieving size maximum count mean variance standard deviation and RMS Internal implementation uses a queue to store values and incremental sums of values and squares
