int HCalCosmics::Init ( PHCompositeNode * )
overridevirtual

Processing.

Initialize HCalCosmics module, create output file and histograms.

This method is invoked once when the module is registered with the Fun4All server. It prints configuration information, opens the ROOT output file, allocates per‑tower TH1F histograms for calibrated energy, raw ADC, ADC after energy cut, tower time and gain, and creates global TH2F and TH1F summary histograms. The event counter is reset.

Parameters
topNodePointer to the top PHCompositeNode (unused in this implementation).
Returns
Zero on successful initialization; non‑zero error codes are not used. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
