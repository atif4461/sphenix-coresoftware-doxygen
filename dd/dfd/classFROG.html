Detailed Description

FROG class provides methods to resolve logical file names to physical locations location method returns physical file name for a given logical name localSearch attempts to locate file on local filesystem dCacheSearch attempts to locate file on dCache storage XRootDSearch attempts to locate file via XRootD protocol LustreSearch attempts to locate file on Lustre filesystem MinIOSearch attempts to locate file on MinIO object storage RawDataSearch attempts to locate raw data files HpssRawDataSearch attempts to locate raw data on HPSS archive PGSearch attempts to locate files using PostgreSQL catalog Verbosity setter configures message verbosity level Verbosity getter returns current verbosity level AutoDisconnect setter enables or disables automatic disconnection of remote resources Disconnect private method handles resource cleanup This comment was generated by openai/gpt-oss-120b at temperature 0.1.
