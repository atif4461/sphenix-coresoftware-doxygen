Detailed Description

brief Reconstructs INTT hits from Geant4 simulation detail This module inherits from SubsysReco and PHParameterInterface to integrate with Fun4All framework detail It initializes run processes each event and provides configurable parameters such as detector name hot strip mask files time window crossing period and clustering thresholds detail It builds truth tracks and clusters using TrkrTruthTrackContainer TrkrClusterContainer and related containers detail Hot channel handling uses a set of raw data identifiers loaded from a file or CDB detail The class manages G4 hit stepping pixel threshold ratio and optional verbose output of cluster hits param name Module instance name default PHG4InttHitReco param d Detector identifier default INTT param m_hotStripFileName Name of hot strip mask file param m_localHotStripFileName Local hot strip mask file for testing param m_Tmin Minimum time for hit selection param m_Tmax Maximum time for hit selection param m_crossingPeriod Beam crossing period param m_pixel_thresholdrat Ratio for pixel threshold param max_g4hitstep Maximum allowed G4 hit step length param record_ClusHitsVerbose Flag to record verbose cluster hit information This comment was generated by openai/gpt-oss-120b at temperature 0.1.

PHG4InttHitReco::PHG4InttHitReco ( const std::string & name = "PHG4InttHitReco")

Constructor for the INTT hit reconstruction subsystem.

Parameters
nameIdentifier used by the Fun4All framework for this subsystem.

Initializes the SubsysReco and PHParameterInterface base classes with the provided name. Sets the detector label to INTT and initializes timing limits and crossing period to NaN. Allocates three three‑component GSL vectors for local output, path and segment calculations. Creates a new TrkrHitSetContainerv1 container for truth hits. Calls InitializeParameters to load configurable parameters. Builds node names for hit, cell and geometry containers based on the detector label.
