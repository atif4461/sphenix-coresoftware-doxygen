Detailed Description

Reconstructs Kshort candidates from Svtx tracks using pairwise DCA and invariant mass selection Configurable cuts include transverse momentum track quality pair DCA and vertex requirements Supports optional truth matching to PHG4 particles and optional saving of reconstructed tracks Operates within Fun4All framework initialization per event processing and finalization Utilizes Acts geometry for track extrapolation and PCA calculations This comment was generated by openai/gpt-oss-120b at temperature 0.1.
