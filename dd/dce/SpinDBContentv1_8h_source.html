2//
3// SpinDBContent class
4// Author : D. Loomis, D. Neff (from Y. Fukao PHENIX class)
5// Description : Content of spin database
6// Created : 2024-05-12
7//
8// runnum : Run number
9// qa_level : Level of QA for stored data.
10// fillnum : Fill number
11// badrun : Run QA by spin analysis, 1:bad 0:good
12// cross_shift : Crossing sfhit, corrected=(raw+cross_shift)%NCROSS
13// bpol : Blue beam polarization
14// bpolerr : Blue beam polarization error
15// bpolsys : Blue beam polarization systematic error
16// ypol : Yellow beam polarization
17// ypolerr : Yellow beam polarization error
18// ypolsys : Yellow beam polarization systematic error
19// bpat : Spin pattern at IP12. (NOT at sPHENIX)
20// ypat : Spin pattern at IP12. (NOT at sPHENIX)
21// scaler_mbd_vtxcut : scaler (MBD VTX)
22// scaler_mbd_nocut : scaler (MBD NS)
23// scaler_zdc_nocut : scaler (ZDC NS)
24// bad_bunch : Bad bunch QA, 0:good else:bad
25// cross_angle : Average relative crossing angle between blue and yellow beams in mrad for run. Sign is dictated by CAD convention, can be + or -
26// cross_angle_std : Standard deviation of relative crossing angle in mrad
27// cross_angle_min : Minimum value of relative crossing angle in mrad
28// cross_angle_max : Maximum value of relative crossing angle in mrad
29// asym_bf : very forward neutron TSSA magnitude in blue beam
30// asym_bb : very backward neutron TSSA magnitude in blue beam
31// asym_yf : very forward neutron TSSA magnitude in yellow beam
32// asym_yb : very backward neutron TSSA magnitude in yellow beam
33// asymerr_bf : very forward neutron TSSA magnitude uncertainty in blue beam
34// asymerr_bb : very backward neutron TSSA magnitude uncertainty in blue beam
35// asymerr_yf : very forward neutron TSSA magnitude uncertainty in yellow beam
36// asymerr_yb : very backward neutron TSSA magnitude uncertainty in yellow beam
37// phase_bf : very forward neutron TSSA phase offset in blue beam
38// phase_bb : very backward neutron TSSA phase offset in blue beam
39// phase_yf : very forward neutron TSSA phase offset in yellow beam
40// phase_yb : very backward neutron TSSA phase offset in yellow beam
41// phaseerr_bf : very forward neutron TSSA phase offset uncertainty in blue beam
42// phaseerr_bb : very backward neutron TSSA phase offset uncertainty in blue beam
43// phaseerr_yf : very forward neutron TSSA phase offset uncertainty in yellow beam
44// phaseerr_yb : very backward neutron TSSA phase offset uncertainty in yellow beam
