Detailed Description

Class provides interface to store and retrieve calibration constants in ROOT TTrees Supports per channel multiple and global single entries for float double int and uint64 types Allows setting values committing to the tree writing to file and loading existing calibrations Static method SetVerbosity controls diagnostic output Internal maps maintain pending entries locked flags prevent modification after commit Filename can be set for output tree names are Single and Multiple
