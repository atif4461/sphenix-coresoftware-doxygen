Detailed Description

JetEvalStack manages evaluation of reconstructed and truth jets within a Fun4All processing chain It holds a JetRecoEval instance and provides accessors for caching strict mode verbosity and sub evaluation stacks The constructor initializes the evaluation with the top node and jet collection names next_event updates internal state for a new event get_reco_eval returns a pointer to the JetRecoEval get_truth_eval returns the associated JetTruthEval Accessor methods expose SvtxEvalStack and CaloEvalStack objects for various detector subsystems
