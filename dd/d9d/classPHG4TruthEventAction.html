Detailed Description

Class PHG4TruthEventAction implements event action for truth information handling within Fun4All framework Provides hooks for begin and end of event resets event state and manages interface pointers to node tree Maintains a set of track identifiers to be written and maps detector hit containers for processing Handles search of relevant nodes pruning and processing of showers and storage of truth info container
