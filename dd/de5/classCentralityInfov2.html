Detailed Description

Concrete implementation of CentralityInfo version 2 Extends CentralityInfov1 to manage centrality bin mapping for each property identifier Uses an internal map from integer property id to integer centrality bin Provides methods to identify the object reset clone copy and query or set centrality bins
