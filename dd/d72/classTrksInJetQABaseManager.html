Detailed Description

===========================================================================

Base histogram manager for TrksInJetQA module

This class consolidates all common functionality for Histogram Managers deployed in the TrksInJetQA module. The histogram managers define what variables to histogram for a given object (e.g. tracks) and how to extract the relevant information from that object.
