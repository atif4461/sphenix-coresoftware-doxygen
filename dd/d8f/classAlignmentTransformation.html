void AlignmentTransformation::createAlignmentTransformContainer ( PHCompositeNode * topNode)

Create alignment transformation containers under the DST node.

Parameters
topNodePointer to the top PHCompositeNode of the current node tree.

The routine locates the DST composite node, then ensures that both a persistent alignmentTransformationContainer and a transient alignmentTransformationContainer are present. If either container is missing, it is instantiated, wrapped in a PHDataNode, and added to the DST node. A runtime_error is thrown if the DST node cannot be found.

void AlignmentTransformation::generateRandomPerturbations ( Eigen::Vector3d angleDev,
Eigen::Vector3d transformDev )

Generate random perturbations for alignment parameters.

This method creates random angular and translational perturbations drawn from normal distributions with zero mean and standard deviations specified by the input vectors. Each component is sampled independently only if the corresponding standard deviation is non‑zero.

Parameters
angleDevVector of standard deviations (rad) for the three rotation angles.
transformDevVector of standard deviations (cm) for the three translation components.

void AlignmentTransformation::setINTTParams ( double inttDevs[6])
inline

Set INTT alignment deviations from a six‑element array.

Parameters
inttDevsInput array where elements 0‑2 are angular standard deviations and elements 3‑5 are translational standard deviations.

The function copies the supplied deviations into the internal angle and translation deviation vectors, enables the INTT perturbation flag, and optionally prints the new settings when localVerbosity is true.
