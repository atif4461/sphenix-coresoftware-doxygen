Detailed Description

SpinDBInput provides an interface to the spin database for run QA management The class handles connection establishment run row verification creation deletion and initialization of database entries It offers overloaded methods to update scalar and array values of various numeric types and to set default QA parameters Internal helper functions construct SQL commands and execute them via an ODBC connection Constants define the database and table names and a sentinel error value
