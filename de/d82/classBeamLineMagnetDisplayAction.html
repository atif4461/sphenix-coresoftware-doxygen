void BeamLineMagnetDisplayAction::ApplyDisplayAction ( G4VPhysicalVolume * physvol)
overridevirtual

Apply visualization attributes to beam line magnet logical volumes.

Iterate over the internal map of logical volumes and their type identifiers. For each logical volume without existing visual attributes, create a new G4VisAttributes object, configure its visibility, solidity and colour based on the identifier (DIPOLE, QUADRUPOLE, SEXTUPOLE). Volumes identified as FIELDVOLUME or OFF are set invisible. Unknown identifiers trigger an error message and abort the program. The created visual attribute objects are stored for later deletion and attached to the corresponding logical volume.
