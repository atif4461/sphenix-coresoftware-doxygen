Detailed Description

Concrete implementation of the EP detector geometry version 2 Provides radial longitudinal and azimuthal tile parameters and allows modification of these parameters through the base class interface Stores tile radii z positions phi angles and reference phi0 values in fixed size arrays matching the EP detector segmentation

◆ get_phi()

float EpdGeomV2::get_phi ( unsigned int key) const
overridevirtual

Retrieve the azimuthal angle (phi) for a given EPD tower key.

The function determines the radial bin of the tower using TowerInfoDefs::get_epd_rbin. If the radial bin is zero, the phi value is taken from the tile_phi0 array; otherwise, it is taken from the tile_phi array. The phi value corresponds to the azimuthal position of the tile in the detector geometry.

Parameters
keyUnique identifier for the EPD tower (encoded as an unsigned int).
Returns
Azimuthal angle phi for the specified tower (in detector units, typically radians).

Set the azimuthal offset phi0 for a specific EPD tile.

This method updates the phi0 value stored in the tile_phi0 vector for the tile identified by the given detector key. The key is converted to the appropriate phi bin index using TowerInfoDefs::get_epd_phibin.

Parameters
keyDetector identifier used to locate the tile phi bin.
