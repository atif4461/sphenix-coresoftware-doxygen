Detailed Description

Kalman filter implementation for ALICE TPC tracks using cluster container and magnetic field map Provides transport rotation filtering and track seeding utilities Configurable gas mixture fractions and optional fixed cluster error model Supports constant magnetic field fallback and field map queries Verbosity control and covariance handling utilities included

double ALICEKF::getClusterError ( TrkrCluster * c,
TrkrDefs::cluskey key,
Acts::Vector3 global,
int i,

Retrieve the (i,j) element of the covariance matrix for a given tracker cluster.

Parameters
cPointer to the TrkrCluster object containing the raw cluster information.
keyUnique identifier (cluskey) of the cluster within the tracker geometry.
globalGlobal position vector of the cluster (x, y, z) in the detector coordinate system.
iRow index of the covariance matrix element to be returned (0‑based).
jColumn index of the covariance matrix element to be returned (0‑based).
Returns
The covariance matrix element err[i][j] in units of (position)^2. If fixed cluster error mode is enabled, the function returns the square of the fixed error for diagonal elements (i == j) and zero for off‑diagonal elements. Otherwise, the error is computed from a parameterized local error, rotated into the global frame using the cluster azimuthal angle.
