◆ get_tower_radial_position()

double PHG4CylinderGeom_Spacalv3::get_tower_radial_position ( const geom_tower & tower) const

get approximate radial position of tower

Compute radial position of a tower for SPACAL geometry.

Parameters
towerReference to geom_tower structure containing centralY coordinate.
Returns
Radial position of the tower in the same units as geometry.

The calculation depends on the current configuration. For full projective 2D taper configurations the centralY value is returned. For tilted configurations the radial position is adjusted using sidewall thickness, outer torus, assembly spacing, azimuthal tilt and number of azimuthal sectors. If an unsupported configuration is encountered the function prints an error, calls Print and exits.
