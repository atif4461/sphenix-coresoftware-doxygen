int TpcLoadDistortionCorrection::InitRun ( PHCompositeNode * topNode)
overridevirtual

global initialization

Initialize run‑time distortion correction containers.

This method is invoked at the beginning of each new run. It locates the RUN node in the Fun4All node tree, creates TpcDistortionCorrectionContainer objects for each enabled correction, loads the corresponding histograms from file, and configures their dimensionality, phi‑angle units, Z‑axis interpolation, and optional scaling factors.

Parameters
topNodePointer to the top PHCompositeNode of the current job.
Returns
Fun4All return code: EVENT_OK on success, ABORTRUN if the RUN node cannot be found.
