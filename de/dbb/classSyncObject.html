Detailed Description

SyncObject provides a base synchronization data container used within the Fun4All framework It inherits from PHObject and implements standard PHObject interface methods Reset clears internal state identify prints object identification isValid reports validity CloneMe creates a deep copy and operator= performs assignment Different compares this instance with another SyncObject and returns a difference metric EventCounter EventNumber SegmentNumber and RunNumber setters and getters are placeholders intended to be overridden by derived versions such as SyncObjectv1 The class is integrated with Fun4All input managers and dump utilities via friend declarations
