◆ InitRun()

int PHGenFitTrkFitter::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialization Run, called for initialization of a run.

Initialize the module for a new run.

This method creates the required nodes in the supplied top node, retrieves the TGeo geometry manager and the magnetic field map, and constructs a GenFit fitter instance using the configured fitting algorithm name and a RKTrackRep representation. The fitter verbosity is set to the module’s verbosity level. Diagnostic information about silicon micro‑meshes, Micromegas usage, and any disabled detector layers is printed to standard output. Finally, the method returns the standard Fun4All success code.

Parameters
topNodePointer to the top PHCompositeNode of the current job.
Returns
Fun4AllReturnCodes::EVENT_OK on successful initialization. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
