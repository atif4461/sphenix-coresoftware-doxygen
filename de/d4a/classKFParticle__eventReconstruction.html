Used to reconstruct simple decays with no intermediate states.

Build basic decay candidate chains from selected daughter particles.

This routine searches for two‑prong combinations of the provided daughter particles and then iteratively expands the search to N‑prong topologies up to the configured maximum number of tracks. The resulting track index groups are passed to getCandidateDecay which fills the output containers with mother candidates, associated secondary vertices and the corresponding daughter lists.

Parameters
selectedMotherBasicOutput vector that will contain the reconstructed mother KFParticle objects.
selectedVertexBasicOutput vector that will contain the secondary vertices associated with each mother candidate.
selectedDaughtersBasicOutput container of vectors, each inner vector holds the daughter KFParticle objects for a given candidate.
daughterParticlesBasicInput list of daughter KFParticle objects that passed the track selection.
goodTrackIndexBasicInput list of indices of tracks that satisfy the basic quality criteria.
primaryVerticesBasicInput list of primary vertex KFParticle objects used for decay vertex constraints.
topNodePointer to the top level Fun4All node tree, required for downstream node access.
Returns

Method to chose best candidate from a selection of common SV's.

Select the optimal candidate combination from a set of KFParticle objects.

This routine evaluates each candidate in possibleCandidates against the corresponding vertex in possibleVertex and chooses the index of the best combination. Selection criteria depend on the flags:

  • If PVconstraint is false, or isAnInterMother is true, or the internal flag m_select_by_mass_error is set, the candidate with the smallest mass error (GetErrMass) is chosen.
  • Otherwise the impact parameter chi‑square (IPχ²) to the vertex is used. When m_use_2D_matching_tools is true the XY deviation (GetDeviationFromVertexXY) is evaluated, otherwise the full 3‑D deviation (GetDeviationFromVertex) is used.
Parameters
PVconstraintFlag indicating whether a primary‑vertex constraint should be applied during selection.
isAnInterMotherFlag indicating that the candidate is an intermediate mother particle; forces mass‑error based selection.
possibleCandidatesVector of candidate KFParticle objects to be evaluated.
possibleVertexVector of vertex KFParticle objects corresponding to each candidate.
Returns
Index of the best candidate combination within the input vectors. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
