◆ WriteCDBHistos()

void CDBHistos::WriteCDBHistos ( )

Write all stored histograms to a ROOT file.

This method iterates over the internal histogram map (m_HistoMap) and writes each histogram object to a newly created ROOT file named by m_Filename. If the map is empty, a diagnostic message is printed and the function returns without creating a file. The original ROOT directory is saved before opening the file and restored after the file is closed to preserve the global directory state.
