◆ Reset()

void TrkrHitSetTpcv1::Reset ( )
overridevirtual

Reset the ADC time‑frame data for the TPC hit set.

This method zeroes the 2‑D ADC array used to store charge information while deliberately preserving the underlying memory allocation. The hit‑set properties (key, starting BCO, pad and time‑bin indices) are left unchanged because the same hit set may be reused in subsequent events, avoiding unnecessary reallocation overhead.
