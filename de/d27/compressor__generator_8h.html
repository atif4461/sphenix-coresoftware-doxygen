Insert a minimum value into a distance‑to‑set map.

The function searches the provided map for the given distance key. If the key does not exist, a new entry is created with a set containing the single element min. If the key already exists, min is added to the existing set associated with that distance.

Parameters
distanceThe distance value used as the map key.
minThe minimum value to be stored in the set for the given distance.
distance_min_set_mapPointer to a map that associates each distance with a set of minimum values. The map is modified in place.
