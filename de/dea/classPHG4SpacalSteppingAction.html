Detailed Description

Stepping action for the Spacal detector handling hit creation and geometry setup.

This class implements the user stepping action for the Spacal subsystem It records energy loss entry and exit points and manages hit containers for active and absorber volumes It also provides access to the light collection model and geometry nodes

Parameters
detectorpointer to the associated Spacal detector
parameterspointer to the run time parameter set
Returns
true if a step produced a hit otherwise false
