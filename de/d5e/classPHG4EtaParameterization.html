Detailed Description

Parameterisation of cylindrical detector rings in pseudorapidity.

Constructs a set of concentric cylindrical rings segmented in eta and phi The constructor receives the number of eta bins the eta range the phi start angle the phi step the inner and outer radii and the global Z centre of the rings Internally the class precomputes the Z position half thickness and phi edges for each bin and stores the mapping from copy number to eta and phi indices

The class implements the Geant4 PV parameterisation interface The ComputeDimensions method fills a G4Tubs solid with the appropriate dimensions for a given copy number ComputeTransformation positions the physical volume at the precomputed Z coordinate and rotates it to the centre of the phi bin

Configure the geometric dimensions of a G4Tubs ring segment.

This routine populates the provided G4Tubs object with the half‑length, inner and outer radii, and azimuthal angular limits that correspond to the detector element identified by the copy number. The half‑length is taken from the internal _zhalf vector indexed by copyNo, while the radii and angular parameters are common to all copies and stored in the member variables _radiusIn, _radiusOut, _startPhi, and _deltaPhi.

Parameters
ringReference to the G4Tubs object whose dimensions will be set.
copyNoInteger copy number identifying the specific detector element.

No return value.
