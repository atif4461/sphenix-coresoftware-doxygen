Detailed Description

Implements a beamline magnet detector within the Fun4All framework Derives from PHG4Detector and provides geometry construction and volume identification Holds a pointer to PHParameters a pointer to the magnet physical volume a pointer to the cylinder physical volume an integer layer index and a string superdetector name Public interface includes a constructor a destructor a ConstructMe method for geometry placement an IsInBeamlineMagnet method for volume test a SuperDetector setter and getter and a get_Layer accessor
