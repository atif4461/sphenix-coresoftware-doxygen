◆ bethe_bloch_vs_logp_wrapper_ZS()

Double_t bethe_bloch_vs_logp_wrapper_ZS ( const Double_t *const x,
const Double_t *const par )
inline

Compute a normalized Bethe‑Bloch energy loss and subtract a ZS offset.

This wrapper evaluates the Bethe‑Bloch formula for a particle with momentum given as log10(p) in the first element of x. The momentum is converted to linear scale, the absolute value is taken, and the result is divided by the particle mass m. The total Bethe‑Bloch value is multiplied by the normalization factor norm and finally the ZS loss offset ZS_loss is subtracted.

Parameters
xPointer to an array of size at least one containing log10(p) where p is the particle momentum in the same units used by the Bethe‑Bloch implementation.
parPointer to an array of size at least three containing:
  • par[0] : normalization factor (dimensionless)
  • par[1] : particle mass (same units as momentum)
  • par[2] : ZS loss offset (same units as the Bethe‑Bloch output)
Returns
Normalized Bethe‑Bloch energy loss minus the ZS loss offset.
