Detailed Description

Calorimeter vertex reconstruction module.

This SubsysReco derived class implements reconstruction of the primary vertex using calorimeter tower information. It creates the required nodes in the Fun4All node tree, provides utilities to compute tower eta, runs a tower‑based algorithm, and processes each event to fill a CaloVertexMap. Configurable parameters include the jet energy threshold, a global calibration factor, an energy cut for tower selection, and an optional z‑dependent energy scaling flag.

Parameters
nameOptional module instance name, default is CaloVtxReco.
jetnodenameName of the jet node to be used, default is zzjets06.
use_z_energy_depFlag to enable z‑dependent energy dependence, default is false.
Returns
createNodes creates CaloVertexMap node under the top node.
new_eta computes the pseudorapidity for a given tower channel.
InitRun performs run‑level initialization.
calo_tower_algorithm executes the tower‑based vertex algorithm.
process_event processes a single event and fills the vertex map.

◆ calo_tower_algorithm()

int CaloVtxReco::calo_tower_algorithm ( PHCompositeNode * topNode) const

Compute the calorimeter‑based vertex z position from calibrated tower information.

This method retrieves calibrated tower containers for the EMCal, inner HCal and outer HCal from the supplied top node, accesses the corresponding tower geometry containers, and calculates an energy‑weighted average of the tower z‑coordinates. Towers failing the quality flag or falling below the configured energy threshold are ignored. The resulting vertex z value is stored in a new CaloVertex object which is inserted into the internal CaloVertex map.
