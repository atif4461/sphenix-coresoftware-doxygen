Detailed Description

PHGeomIOTGeo store geometry information to DST files in the format of binary streamed TGeoVolume. It completely owns the geometry object For run-time use of TGeoManager, please use PHGeomTGeo For operation of this class with DST node, please use PHGeomUtility.

TGeoManager * PHGeomIOTGeo::ConstructTGeoManager ( )

Construct TGeoManager. The result TGeoManager is not yet closed and open for further editing.

Construct a ROOT TGeoManager from the PHGeomIOTGeo geometry.

This routine first validates the PHGeomIOTGeo object. If the object is not valid a null pointer is returned. The ROOT default length unit is forced to centimeters (the Fun4All internal unit) using the appropriate ROOT API for the detected ROOT version. A new TGeoManager named “PHGeometry” is created, the geometry volume obtained from GetGeometryCopy() is registered, and set as the top volume of the manager. The manager title is built from the volume name and title to indicate that it was created by PHGeomUtility::LoadFromIONode based on the RUN/GEOMETRY_IO node.

Returns
Pointer to the newly created TGeoManager, or nullptr if the geometry object is invalid. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
