Detailed Description

PHG4MvtxHitReco reconstructs MVTX hits from Geant4 simulation It inherits from SubsysReco to integrate with Fun4All event loop and from PHParameterInterface for configurable parameters The module initializes run level resources processes each event and builds truth clusters and tracks It provides methods to set detector name timing window and pixel threshold Internal utilities generate ALPIDE pulse shapes strobe timing and pixel masking Random number generation uses GSL RNG with custom deleter Truth information containers are managed for embedding and clustering

Deleter provides a custom deleter for GSL random number generator used by PHG4MvtxHitReco
