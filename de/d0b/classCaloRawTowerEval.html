◆ best_tower_from() [1/2]

RawTower * CaloRawTowerEval::best_tower_from ( PHG4Particle * primary)

which tower did the primary truth particle contribute the most energy to?

Retrieve the best calorimeter tower associated with a given primary PHG4Particle.

Parameters
primaryPointer to the PHG4Particle representing the primary particle of interest.
Returns
Pointer to the RawTower that best matches the primary particle, or nullptr if no suitable tower is found or an error occurs.

The function checks node pointer validity, enforces strictness constraints, validates that the input particle is a primary, resolves the true primary particle, and optionally uses a cache to avoid redundant lookups. If a primary shower is available, the best tower is obtained from that shower. Errors are counted in the internal _errors counter and result in a nullptr return.

◆ max_truth_primary_shower_by_energy() [1/2]

PHG4Shower * CaloRawTowerEval::max_truth_primary_shower_by_energy ( RawTower * tower)

which primary shower contributed the most energy to this tower?

Retrieve the primary truth shower that contributes the highest energy to a given tower.

This method searches all primary truth showers associated with the specified tower, evaluates each shower's energy contribution, and returns the shower with the maximum contribution. It respects the module's strict mode, optional caching, and error accounting. If node pointers are unavailable, the input tower is null (when not in strict mode), or no valid shower is found, the function returns a null pointer.

Parameters
towerPointer to the RawTower for which the dominant primary truth shower is sought.
Returns
Pointer to the PHG4Shower with the largest energy contribution to the tower, or nullptr if no such shower exists or an error condition is encountered.
