std::size_t SvtxAlignmentStateMap_v1::erase ( unsigned int track)
overridevirtual

Erase the alignment states associated with a given track identifier.

This method looks up the internal map for the specified track key. If found, it iterates over the stored state pointers, deletes each dynamically allocated state object, and then removes the entry from the map.

Parameters
trackUnique identifier of the track whose alignment states are to be removed.
Returns
Number of elements removed from the map (0 if the track was not present, 1 otherwise).
