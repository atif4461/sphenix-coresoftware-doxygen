Detailed Description

Utility class for ROOT directory manipulation in sPHENIX analysis workflow duplicateDir copies all objects from source directory to destination directory getHisto returns a pointer to a histogram with given name located in specified directory mkdir creates a new subdirectory hierarchy under a top directory according to a path optionally storing titles mkpath creates intermediate subdirectories along a path within a given directory copyToFile copies all objects from a source directory into a destination ROOT file pathIsInDir determines whether a path string is located inside a given directory splitPath splits a path string into its individual components stored in a vector
