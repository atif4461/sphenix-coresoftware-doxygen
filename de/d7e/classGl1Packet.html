Detailed Description

Gl1Packet class provides an interface for offline packet data handling It inherits from OfflinePacketv1 and implements versioned packet methods The class defines setters and getters for scaler values packet number bunch number trigger input live vector scaled vector GTMBusyVector GTMAllBusyVector It also provides overloads for iValue and lValue to access integer and long long data fields
