◆ FilterPhiPos()

double AnnularFieldSim::FilterPhiPos ( double phi)

Map an azimuthal angle to the positive range [0, 2π).

This routine takes an input angle phi that may lie in the interval [-π, 0] and shifts it into the interval [π, 2π] by adding 2π. If the resulting value exceeds 2π it is wrapped back into the valid range by subtracting 2π. A sanity check is performed and the program aborts if the final value is still outside the allowed interval.

Parameters
phiInput angle in radians, may be negative or greater than 2π.
Returns

void AnnularFieldSim::populate_full3d_lookup ( )

Populate the full 3‑D lookup table of unit electric field contributions.

This routine iterates over all measurement cells within the region of interest (ROI) defined by rmin_roi, rmax_roi, phimin_roi, phimax_roi, zmin_roi and zmax_roi and over all source charge cells defined by the grid dimensions nr, nphi and nz. For each pair of measurement cell (at) and source cell (from) the unit field vector is calculated with calc_unit_field and stored in the Epartial container. When the measurement and source cells coincide the field is set to the zero vector to avoid self‑interaction. Progress is reported to standard output in percent steps controlled by debug_npercent.

No input parameters are required. The function returns void.
