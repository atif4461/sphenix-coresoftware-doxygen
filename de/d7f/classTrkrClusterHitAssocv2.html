unsigned int hidx )

Add an association between a tracker cluster key and a hit index.

This method extracts the layer, sector, and side identifiers from the supplied cluster key and inserts the (cluster key, hit index) pair into the three‑dimensional association map if the identifiers are within the configured limits. Out‑of‑range identifiers generate a diagnostic message.

Parameters
ckeyTracker cluster key identifying the cluster.
hidxIndex of the hit to be associated with the cluster.
Returns
No return value.
