Detailed Description

PhotonClusterBuilder builds photon clusters from input calorimeter clusters It is a SubsysReco module that creates output photon objects applies ET threshold shower shape calculations and optional BDT based identification Configuration methods allow setting input and output node names ET cut minimum tower energy for shape BDT model file and feature list and flags for using BDT and subtracted isolation During InitRun the required nodes are created or retrieved process_event loops over raw clusters fills photon objects computes shower shapes evaluates BDT score if enabled and stores results in the output node Private helper functions handle node creation shower shape metrics BDT scoring tower geometry queries nearest HCAL tower search deltaR calculation and layer ET integration
