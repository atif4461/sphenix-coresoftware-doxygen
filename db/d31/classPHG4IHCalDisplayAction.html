void PHG4IHCalDisplayAction::ApplyDisplayAction ( G4VPhysicalVolume * physvol)
overridevirtual

Apply visualization attributes to inner HCal volumes.

This method iterates over all scintillator logical volumes stored in m_ScintiLogVolSet and assigns a green solid visible attribute if the volume does not already have a visual attribute. It also assigns a red solid visible attribute to the steel volume m_SteelVol when it has no existing visual attribute. Created G4VisAttributes objects are stored in m_VisAttVec for later cleanup.

Parameters
physvolPointer to the physical volume (unused in this action).
Returns
None.
