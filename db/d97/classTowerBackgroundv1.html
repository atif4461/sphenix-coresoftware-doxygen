Detailed Description

TowerBackgroundv1 implements the TowerBackground interface for storing underlying event background parameters Provides methods to set and retrieve UE vectors per layer flow coefficient v2 event plane angle Psi2 and counts of strips and towers used Includes a flag indicating flow calculation failure All data members are initialized to default values
