Detailed Description

PHG4IHCalDetector implements the inner hadronic calorimeter geometry for the sPHENIX simulation It constructs the detector volumes within the world logical volume and registers geometry nodes in the node tree Provides methods to query layer tower and sector identifiers from physical volumes Manages GDML configuration and raw tower geometry containers Supports super detector naming and assembly of scintillator tiles Tracks material volumes and active absorber flags for consistency checks This comment was generated by openai/gpt-oss-120b at temperature 0.1.
