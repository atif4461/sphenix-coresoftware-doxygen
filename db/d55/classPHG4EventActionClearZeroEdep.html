Detailed Description

Event action that removes hits with zero energy deposition from specified nodes.

The action iterates over the set of node names stored in nodename_set and clears entries with zero edep in the corresponding PHG4HitContainer objects located under the provided topNode. Nodes can be added dynamically via AddNode. This class is intended to be used in the Fun4All processing chain as a PHG4EventAction.
