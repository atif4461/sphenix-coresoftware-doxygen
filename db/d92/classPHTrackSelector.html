Detailed Description

PHTrackSelector is a SubsysReco module that filters tracks based on quality criteria It accesses the SvtxTrackMap from the node tree and applies configurable cuts on TPC cluster count MVTX hit count INTT hit count and chi2 per degree of freedom The module can be configured to operate on a specific iteration of the TrkrClusterIterationMap and to use a user supplied track map name The standard SubsysReco callbacks InitRun process_event and End are overridden to retrieve nodes apply the selection and clean up
