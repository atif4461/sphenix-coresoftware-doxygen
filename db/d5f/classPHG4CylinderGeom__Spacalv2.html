double PHG4CylinderGeom_Spacalv2::get_sec_azimuthal_width ( ) const

Compute azimuthal width of a sector after applying tilt corrections.

Compute the secondary depth of a detector sector The available depth is the total thickness minus the radial offset and assembly spacing If the available depth is smaller than the sector azimuthal width the function returns NaN Otherwise the depth is calculated from the Pythagorean relation between the available depth and the azimuthal width Returns the secondary depth in the same units as thickness or NaN
