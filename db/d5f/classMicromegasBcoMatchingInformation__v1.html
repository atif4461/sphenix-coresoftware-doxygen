void MicromegasBcoMatchingInformation_v1::cleanup ( uint64_t ref_bco)

cleanup

Remove outdated BCO entries and reset orphan tracking.

This method purges all entries in the GTM BCO list whose value is less than or equal to the supplied reference BCO. It also removes all matching pairs in the BCO matching list whose second element (the BCO) is less than or equal to the reference BCO. After the removals, any stored orphan information is cleared.
