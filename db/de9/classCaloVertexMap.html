Detailed Description

CaloVertexMap provides an abstract interface for mapping unsigned integer identifiers to CaloVertex objects within the sPHENIX analysis framework It inherits from PHObject to integrate with the Fun4All node tree and supports standard container operations such as iteration lookup insertion and removal The class defines iterator types const and mutable accessors and virtual methods that can be overridden by concrete implementations to manage the underlying storage
