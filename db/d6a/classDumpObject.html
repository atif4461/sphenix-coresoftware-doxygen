Detailed Description

DumpObject provides a generic interface for dumping node information within the Fun4All framework The class manages output file handling precision settings and optional suppression of output It can be attached to a PHNodeDump instance to traverse the node hierarchy Lifecycle methods correspond to the SubsysReco callbacks Init process_event and CloseOutputFile Configuration methods allow setting the output directory floating point precision and flags controlling run event labeling and event sequence printing Protected hooks enable derived classes to implement custom node processing and file opening logic This comment was generated by openai/gpt-oss-120b at temperature 0.1.
