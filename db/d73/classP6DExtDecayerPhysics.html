Attach a G4Pythia6 external decayer to all G4Decay processes of instantiated particles.

The method creates a single G4Pythia6Decayer instance, iterates over the global particle iterator, and for each decay process found replaces the internal decayer with the external one. If a forced decay type is configured it is applied to the external decayer. The external decayer is deleted only when it was not attached to any process.
