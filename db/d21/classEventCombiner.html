Detailed Description

Class provides functionality to combine multiple PRDF input nodes into a single output event It inherits from SubsysReco and implements the standard Fun4All callbacks Init creates or retrieves the output event object and registers the output node process_event merges data from all registered PRDF input nodes into the output event ResetEvent clears temporary buffers after each event AddPrdfInputNodeFromManager registers an input node obtained from the Fun4AllInputManager AddPrdfInputNodeName registers an input node by explicit name Member variables store the current event output array pointer output node name and the set of input node names
