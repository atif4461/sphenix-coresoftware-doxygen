Detailed Description

PHG4ZDCSteppingAction implements the stepping action for the ZDC detector in the sPHENIX simulation framework It records energy deposition in ZDC absorber and support volumes and creates hit objects in the node tree The class interfaces with the PHG4ZDCDetector and uses parameters from PHParameters It provides methods to locate detector indices compute response functions and manage hit containers Random number generation is performed with gslrng
