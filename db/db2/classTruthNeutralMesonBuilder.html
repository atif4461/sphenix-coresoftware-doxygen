Detailed Description

Builder module for neutral meson truth information within the Fun4All framework Inherits from SubsysReco and implements initialization end processing and event loop Provides methods to classify eta decay channels and to identify photon conversions Allows configuration of conversion radius and node name for output container Options to store eta and pi0 mesons can be enabled or disabled Internal data structures map Geant4 particles by barcode and id and track mother daughter relations Maintains sets of processed barcodes to avoid duplication This comment was generated by openai/gpt-oss-120b at temperature 0.1.
