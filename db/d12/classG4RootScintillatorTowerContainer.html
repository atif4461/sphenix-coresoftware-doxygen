Detailed Description

Container for scintillator towers produced by Geant4 simulation Manages a collection of G4RootScintillatorTower objects stored in a TClonesArray Provides methods to add a tower reset the container and identify its contents Stores detector identifier total energy ionization energy leakage fraction and event number All member variables are initialized to sentinel values

void G4RootScintillatorTowerContainer::Reset ( )

Reset the scintillator tower container to an undefined state.

Sets the total energy and leakage values to NaN, clears the event identifier, and empties the single‑tower collection. If the collection size exceeds the configured maximum (NMAX), the container is re‑sized to NMAX.
