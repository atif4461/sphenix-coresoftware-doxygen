Detailed Description

TpcSeedTrackMapv1 implements the TpcSeedTrackMap interface using a multimap to store associations between TPC seed identifiers and track identifiers Provides methods to reset the container add an association retrieve associated tracks for a given seed retrieve all associations and query the number of stored entries This comment was generated by openai/gpt-oss-120b at temperature 0.1.
