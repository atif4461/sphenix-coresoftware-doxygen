int HepMCCollisionVertex::InitRun ( PHCompositeNode * topNode)
overridevirtual

Initialize run-specific objects for HepMCCollisionVertex.

Called for first event when run number is known. Typically this is where you may want to fetch data from database, because you know the run number. A place to book histograms which have to know the run number.

This method is invoked at the beginning of each new run. It searches the node tree for the DST composite node and attempts to retrieve the PHHepMCGenEventMap object. If the map is not present, a new instance is created and attached to the DST node as a PHIODataNode. The function ensures that the event‑generator map is available for subsequent event processing.
