Detailed Description

CaloVertex provides a base implementation of the Vertex interface for calorimeter reconstruction The class inherits from Vertex and fulfills the PHObject virtual methods with minimal stubs All accessor methods return sentinel values indicating that derived classes must supply real data This class is not intended to be instantiated directly but to serve as a common base for specific calorimeter vertex types
