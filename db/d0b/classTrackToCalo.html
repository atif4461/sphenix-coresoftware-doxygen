Detailed Description

TrackToCalo module inherits from SubsysReco and processes each event to associate tracks with calorimeter clusters Init creates output ROOT file and tree and registers histograms process_event loops over tracks applies pt and tpc cluster cuts matches to EMCal clusters using delta phi and delta z and fills vectors End writes tree to file and closes output ResetTreeVectors clears internal vectors for next event PiRange returns phi difference wrapped to pi range EMcalRadiusUser enables or disables user defined EMCal radius cut setEMcalRadius sets user defined EMCal radius value setRawClusContEMName sets name of raw EMCal cluster container setTrackPtLowCut sets minimum track transverse momentum setEmcalELowCut sets minimum EMCal cluster energy setnTpcClusters sets minimum number of TPC clusters associated to track This comment was generated by openai/gpt-oss-120b at temperature 0.1.
