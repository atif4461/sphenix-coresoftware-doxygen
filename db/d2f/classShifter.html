Detailed Description

Class provides utilities to apply spatial shifts to detector positions using truth and correction histograms loaded from ROOT files Constructor loads optional truth and correction files Shift returns combined forward and backward shift ShiftForward applies only forward histogram ShiftBack applies only backward histogram Member variables store ROOT file pointers and histogram pointers for X Y Z R Phi components and their averages This comment was generated by openai/gpt-oss-120b at temperature 0.1.
