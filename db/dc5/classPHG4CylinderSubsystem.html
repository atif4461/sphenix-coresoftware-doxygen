Print info (from SubsysReco)

Print the cylinder subsystem parameters and optional stepping‑action information.

The method first prints the subsystem name. If BeginRun() has not been executed, it prints instructions on how to invoke BeginRun() and then returns. Otherwise, it prints the parameter set obtained from GetParams() and, if a stepping action is attached, forwards the print request to the stepping action using the supplied identifier.
