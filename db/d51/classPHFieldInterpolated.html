Detailed Description

Provides a best-fit cubic approximation of the Field In 3D cartesian coordinates Class PHFieldInterpolated provides interpolated magnetic field evaluation Inherits from PHField and implements thread safe caching of O3 Taylor coefficients The class loads field maps from ROOT files and offers methods to query field values Public methods include GetFieldValue for Geant4 interface getinterpolated returns interpolated field at a point loadfieldmap loads map from file with optional scaling factor print outputs diagnostic information printcoefficients outputs cached Taylor coefficients Internal cache structure stores per thread interpolation data Design matrix utilities and index conversion helpers support interpolation Thread safety ensured by mutex and cache pruning for up to MAXTHREADS threads
