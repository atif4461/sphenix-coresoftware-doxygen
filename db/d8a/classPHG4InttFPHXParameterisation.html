Detailed Description

PHG4InttFPHXParameterisation provides a parameterisation of the FP H X geometry for the INTT detector in the sPHENIX simulation The constructor takes offsetx offsety offsetz dz and ncopy as arguments ComputeTransformation overrides the base method to compute the placement of each copy using the internal arrays fXFPHX fYFPHX fZFPHX The arrays store up to twenty position values
