◆ ConvertFromOscar()

int Fun4AllOscarInputManager::ConvertFromOscar ( )

Convert the next Oscar event from the input file into a HepMC::GenEvent.

This method clears any previously stored event, checks for end‑of‑file, reads the Oscar formatted lines (handling optional compression), parses particle kinematics and vertex information, constructs HepMC::GenParticle objects, assembles them into HepMC::GenVertex objects, and finally inserts the completed HepMC::GenEvent into the global PHHepMCGenEventMap. Verbosity controls diagnostic output.

Returns
0 Conversion succeeded and the event was stored.
1 End‑of‑file encountered; no further events are available.

No input parameters are required; the method operates on the internal state of the Fun4AllOscarInputManager instance, including the input file stream, compression flag, and bookkeeping counters.
