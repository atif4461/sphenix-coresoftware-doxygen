◆ CopyFrom() [2/3]

void SvtxTrack_v1::CopyFrom ( const SvtxTrack & source)
overridevirtual

Copy the contents of a source SvtxTrack into this SvtxTrack_v1 instance.

Performs a deep copy of all track attributes, including identifier fields, charge sign, chi‑square, degrees of freedom, distance of closest approach parameters and their uncertainties. The base class SvtxTrack::CopyFrom is invoked to copy base‑class members. Existing state objects are cleared and cloned from the source. Cluster identifier and key sets are duplicated, and all calorimeter projection quantities (dphi, deta, 3×3 and 5×5 energies, cluster identifiers, keys and energies) are copied when valid.

Parameters
sourceReference to the SvtxTrack object to be copied.
