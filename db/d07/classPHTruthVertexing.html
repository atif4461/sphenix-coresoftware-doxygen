PHTruthVertexing::PHTruthVertexing ( const std::string & name = "PHTruthVertexing")

Construct a PHTruthVertexing instance with the given module name.

Initializes the base PHInitVertexing with the supplied name, allocates a GSL Mersenne‑Twister random number generator, and seeds it using the framework‑provided PHRandomSeed function. Member variables for the G4 truth container pointer, vertex error vector, and embed‑only flag are also initialized to their default values.
