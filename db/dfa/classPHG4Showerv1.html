Detailed Description

PHG4Showerv1 implements the PHG4Shower abstract base class for version 1 Provides getters and setters for shower unique identifier parent particle identifier parent shower identifier Stores mean position coordinates and covariance matrix of the shower hits Tracks number of hits energy deposition ionization energy light yield and electron hadron energy ratio per detector volume Manages collections of associated G4 particle identifiers G4 vertex identifiers and G4 hit identifiers Supports cloning resetting validity checking and identification output Container methods allow insertion removal iteration and size queries of the associated identifier sets This comment was generated by openai/gpt-oss-120b at temperature 0.1.
