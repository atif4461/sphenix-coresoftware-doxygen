Double_t MbdSig::MBDTDC ( const Int_t max_samp)

Retrieve the time value from a fixed sample index of the MBD pulse.

MBD method to get time, max_samp is the sample to use

This method extracts the amplitude at the specified sample index max_samp from the underlying sub‑pulse waveform and interprets it as a time measurement for MBD time channels. If the waveform pointer is null, a NaN is returned and an error message is printed. An additional sanity check forces the time to zero when the amplitude at sample index 2 exceeds 100.

◆ SetY()

void MbdSig::SetY ( const Float_t * y,
const int invert = 1 )

Load raw waveform samples, apply pedestal subtraction and optional pile‑up correction.

This method fills the internal raw pulse histogram and graph with the provided sample array y, then subtracts the configured pedestal (including optional pre‑sample or range‑based calculations). The resulting pedestal‑subtracted waveform is stored in the sub‑pulse histogram and graph, with bin errors set to the pedestal RMS. If a pile‑up condition is detected and a valid pile‑up pedestal offset is available, the pile‑up contribution is removed.

Parameters
yPointer to an array of Float_t containing the digitized samples.
invertSign factor applied to the pedestal‑subtracted samples (typically 1 or –1 to account for signal polarity).
