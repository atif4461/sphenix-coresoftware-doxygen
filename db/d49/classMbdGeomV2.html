Detailed Description

MbdGeomV2 implements the MbdGeom interface for version two geometry It stores the position of up to one hundred twenty eight photomultiplier tubes The arrays pmt_x pmt_y pmt_z pmt_r pmt_phi hold the Cartesian and polar coordinates get_x get_y get_z get_r get_phi return the corresponding coordinate for a given channel index set_xyz allows setting the three dimensional position of a channel download_hv initiates the download of high voltage settings from the database get_hvmap provides read access to the multimap that associates channel identifiers with high voltage values Reset is a placeholder that currently performs no operation This comment was generated by openai/gpt-oss-120b at temperature 0.1.

void MbdGeomV2::download_hv ( )
overridevirtual

Populate the HV channel map for the MbdGeomV2 detector.

This method inserts pairs of module identifiers and high voltage channel numbers into the internal pmt_hv map. The entries cover both the South and North sections of the detector and are later used to retrieve HV settings for each photomultiplier tube.
