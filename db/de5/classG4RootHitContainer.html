Detailed Description

Container for G4 hits with summary information

Stores individual PHG4Hit objects in a TClonesArray and provides total energy ionization energy leakage fraction and event identifier Implements PHObject interface for identification and reset

void G4RootHitContainer::Reset ( )
overridevirtual

Reset the hit container to an initial empty state.

This method clears the accumulated total energy and leakage values, resets the event identifier, and clears the collection of single hits. If the internal hit buffer exceeds the configured maximum size, it is re‑expanded to the predefined limit.
