Detailed Description

Class provides lightweight evaluation of calorimeter data within Fun4All framework It inherits from SubsysReco and implements required lifecycle methods InitRun process event End The module can be configured for different calorimeter types CEMC HCALIN HCALOUT via Calo enum Various setters allow control of fit range mode QA flags input tower node name and tower usage Getters expose fit limits and spectra bin width Additional utilities perform histogram retrieval fitting relative shifts QA plotting and chimney checks Internal data members store temporary ROOT files histograms for each calorimeter geometry and event counters This comment was generated by openai/gpt-oss-120b at temperature 0.1.
