void PHG4Subsystem::SetMotherSubsystem ( PHG4Subsystem * subsys)

Assign the specified subsystem as the mother subsystem if it is permitted.

This method checks whether the provided subsystem can serve as a mother subsystem via its CanBeMotherSubsystem() flag. If the check passes, the internal pointer m_MyMotherSubsystem is updated. Otherwise an error message is printed and the application terminates.
