Detailed Description

brief TpcClusterMover class provides functionality to transform TPC cluster positions the class stores start and projection coordinates and layer radius information it allows setting verbosity level and initializing geometry from PHG4TpcGeomContainer the processTrack method converts a list of global cluster keys and positions to local coordinates the initialize_geometry method updates internal geometry parameters based on cell geometry the get_circle_circle_intersection method computes intersection of two circles used in projection calculations

TpcClusterMover::TpcClusterMover ( )

constructor

Construct TpcClusterMover and compute radial spacing for inner, mid, and outer TPC layers.

Initializes the layer_radius array for all 48 layers using the predefined minimum and maximum radii for the inner, middle, and outer TPC regions. The spacing for each region is calculated from the corresponding radius range divided by 16 and the layer radii are set to the midpoint of each spacing interval.
