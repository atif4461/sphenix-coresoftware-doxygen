Detailed Description

Provides high level interface to calorimeter evaluation stack Manages raw cluster raw tower and truth evaluation objects Constructor requires top node pointer and calorimeter name Methods expose underlying evaluators and control caching strictness and verbosity get_caloid forwards request to truth evaluator next_event updates internal state for new event get_errors returns accumulated error count from cluster evaluator
