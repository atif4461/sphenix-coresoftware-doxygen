Detailed Description

/ Class representing generic level one trigger output object
/ Provides interface for trigger type identification and word access
/ Stores mapping from trigger identifiers to word vectors
/ Methods allow retrieval of trigger bits and evaluation of trigger conditions
/ Derived classes implement specific trigger subsystem behavior
/ Reset method clears event data
/ Identify method prints object information
/ Event and clock numbers can be set and queried
/ Trigger words can be accessed as const or mutable ranges* This comment was generated by openai/gpt-oss-120b at temperature 0.1.
