Detailed Description

CosmicTrackQA module implements SubsysReco interface for cosmic track quality assurance Initialization per run event processing and finalization are provided Configurable run range and histogram creation are handled internally Static utilities extract cluster keys from SvtxTrack and perform linear fits on cluster positions Histogram names are generated with a module specific prefix This comment was generated by openai/gpt-oss-120b at temperature 0.1.

int CosmicTrackQA::EndRun ( const int runnumber)
overridevirtual

End-of-run callback that records the average number of tracks per event for the given run.

This method retrieves the histogram manager, obtains the histogram named "<prefix>ntracksperrun", and fills it with the run number and the ratio of total tracks to total events processed in the run.
