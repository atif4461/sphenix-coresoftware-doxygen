Detailed Description

Container for simulated tower information associated with a specific detector.

Implements the TowerInfoContainer interface for simulation data. Manages a TClonesArray of TowerInfoSimv1 objects. Provides cloning, identification, reset, and tower access methods. Encodes and decodes tower keys for fast lookup. Stores the detector identifier and returns it via get_detectorid(). This comment was generated by openai/gpt-oss-120b at temperature 0.1.
