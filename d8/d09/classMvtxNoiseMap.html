Get the noise level for this pixels.

Retrieve the noise level for a specific pixel.

This method looks up the noise value for the pixel identified by the given chip, row, and column indices in the internal noisy‑pixel map. If the pixel is present in the map, its associated noise level is returned; otherwise the function returns zero.

Parameters
chipZero‑based index of the readout chip; must be less than the total number of chips stored in the noisy‑pixel container.
rowRow index of the pixel within the chip.
colColumn index of the pixel within the chip.
Returns
Noise level for the specified pixel, or zero if the pixel is not marked as noisy.
