Detailed Description

MomentumEvaluator evaluates true and reconstructed momentum information for each event It creates ntuples for true and reconstructed quantities and fills them during processing Configuration parameters control search scales layer requirements and z limits The class implements the SubsysReco interface with Init process_event and End methods This comment was generated by openai/gpt-oss-120b at temperature 0.1.
