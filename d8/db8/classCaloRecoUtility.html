void CaloRecoUtility::ShowerDepthCorrZVertex ( RawCluster * clus,
float vz )
static

corrects cluster Z (implicitly also eta) for updated z vertex

Apply a shower‑depth correction to the cluster z‑position using the event vertex.

This routine adjusts the longitudinal coordinate of a RawCluster to account for the displacement of the shower maximum relative to the true interaction vertex. The correction is derived from the cluster energy, its transverse radius, and the polar angles of the tower and the track‑like trajectory from the vertex.

Parameters
clusPointer to the RawCluster whose z coordinate will be modified.
vzReconstructed z‑vertex of the event (in cm).
