Detailed Description

base class for EventHeaders

Event level header storing run event and physics metadata Provides interface to reset identify and validate the object Stores integer and floating point values via generic name based setters Supports run number event sequence bunch crossing event type impact parameter event plane angle flow psi coefficients eccentricity number of binary collisions number of participants and timestamp

Returns
isValid returns non zero if data are valid
