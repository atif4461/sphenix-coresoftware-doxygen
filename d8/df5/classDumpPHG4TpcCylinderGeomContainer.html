Detailed Description

Dump PHG4 Tpc Cylinder Geometry Container.

Implements a DumpObject derived class that processes a PHNode containing a PHG4CylinderGeomContainer for the TPC subsystem. The constructor receives the name of the node to be inspected and the overridden process_Node method returns an integer status code.
