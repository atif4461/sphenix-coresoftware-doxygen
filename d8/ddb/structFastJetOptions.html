Detailed Description

FastJetOptions stores configuration parameters for FastJet based jet reconstruction It includes jet radius algorithm selection and various optional cuts and calculations Members control usage of eta and pt cuts constituent thresholds soft drop settings area calculation background density constituent subtraction and verbosity The struct provides methods to update options from a vector of FastJetOptItem print the current configuration and initialize internal defaults This comment was generated by openai/gpt-oss-120b at temperature 0.1.
