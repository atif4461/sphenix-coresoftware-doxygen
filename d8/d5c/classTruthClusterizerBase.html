Parameters

_verbosityInteger controlling the verbosity level of the module.

The routine stores the top node and verbosity, locates the DST node, ensures a TRKR subnode exists, and creates or retrieves the TrkrTruthTrackContainer, TrkrClusterContainer and PHG4TruthInfoContainer needed for truth based clustering. Missing containers are instantiated and attached to the node tree. An assertion aborts if the truth info container cannot be found. ◆ print_clusters()

void TruthClusterizerBase::print_clusters ( int nclusprint = 20)

Print detailed information about truth clusters associated with each track.

This method iterates over all truth tracks stored in the internal track map, outputs basic track parameters (track id, phi, eta, pt) and the total number of associated clusters. Depending on the verbosity level, it either prints a newline after the track summary or lists the individual cluster identifiers. For each cluster the hit set key and cluster index are displayed. The optional argument nclusprint limits the number of clusters printed per track; if the limit is reached the output is truncated with an ellipsis.

Parameters
nclusprintMaximum number of clusters to print for each track. A non‑positive value disables the limit and prints all clusters. This comment was generated by openai/gpt-oss-120b at temperature 0.1.

void TruthClusterizerBase::update_track ( )

Update the internal pointer to the current truth track.

This method selects the appropriate truth track from the truth track container based on the embedding flag. If the event is embedded (m_is_emb is true) the truth track identified by m_trkid is retrieved using m_truthtracks and stored in m_current_track; otherwise m_current_track is set to nullptr.
