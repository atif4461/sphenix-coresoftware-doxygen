Detailed Description

ParticleFlowJetInput provides jet input from particle flow objects Implements JetInput interface to retrieve jets constructed from particle flow candidates stored in the Fun4All node tree get_input extracts particle flow collections from the top PHCompositeNode and returns a vector of Jet pointers identify prints a short description to the supplied output stream Verbosity controls diagnostic output
