Detailed Description

KFParticle MVA helper class Provides methods to initialise a TMVA reader and evaluate a multivariate score for a KFParticle and its decay vertex initMVA creates a TMVA Reader configured with the variables listed in m_mva_variable_list and returns the reader together with a vector of Float_t placeholders evaluateMVA fills the placeholder vector with the kinematic and topological quantities of the supplied particle and vertex and returns the TMVA response Protected data members store the number of input parameters the MVA expects the list of variable names the type of MVA method and the path to the weight file Private data members cache the number of parameters derived from the variable list and the full method identifier string
