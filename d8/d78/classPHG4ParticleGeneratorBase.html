Detailed Description

Base class for particle generators in the sPHENIX simulation framework Provides configuration of particle type momentum vertex and time Manages a list of PHG4Particle objects and random number generation Implements SubsysReco interface methods InitRun process event and Print Supports embedding flag reuse existing vertex and seed control Utility functions convert between PDG names codes and masses This comment was generated by openai/gpt-oss-120b at temperature 0.1.
