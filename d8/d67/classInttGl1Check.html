Detailed Description

Analysis module for checking INTTRAWHIT data in the INTTRAW subsystem.

Inherits from SubsysReco and implements the standard Fun4All callbacks:

  • Init: locate and prepare the INTTRAWHIT node.
  • process_event: execute per‑event checks on beam clock values and hit differences.
  • End: finalize the analysis and report summary statistics.

The module allows the user to override the default event node name via MyEvtNode.

Internal bookkeeping:

  • m_EvtNodeName stores the name of the node containing INTTRAWHIT data (default “INTTRAWHIT”).
  • bclk_seen records all unique beam‑clock identifiers encountered.
  • diffcnt maps each beam‑clock identifier to the number of observed differences.
