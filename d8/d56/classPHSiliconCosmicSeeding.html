Detailed Description

Silicon cosmic track seeding module for sPHENIX Inherits from SubsysReco and implements the standard Fun4All callbacks Retrieves TrkrClusterContainer builds doublet seeds from cluster positions Extends seeds with additional clusters combines and prunes them Stores results in a TrackSeedContainer named SiliconTrackSeedContainer Configurable parameter maximum doublet distance in centimeters
