Detailed Description

brief Base class representing a truth level vertex in the sPHENIX reconstruction framework

Inherits from Vertex and provides default implementations that return invalid or NaN values This class is intended to be subclassed for concrete truth vertex representations It implements the PHObject interface with placeholder methods

The vertex parameters id position time and their uncertainties are overridden to return sentinel values indicating the absence of real data The protected constructor enforces creation through derived classes

ClassDefOverride macro integrates the class with ROOT IO This comment was generated by openai/gpt-oss-120b at temperature 0.1.
