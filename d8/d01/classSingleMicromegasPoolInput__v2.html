◆ GetSomeMoreEvents()

bool SingleMicromegasPoolInput_v2::GetSomeMoreEvents ( )

Evaluate the Micromegas event pool and decide if additional events should be retrieved.

The function checks for completion, verifies that the raw hit map meets the minimum pool size, and ensures that each front‑end electronics (FEE) has advanced beyond the current bunch crossing offset. If a FEE is detected with a stalled clock, it is removed from the tracking map.

Returns
true if more events are required or pool conditions are not yet satisfied, false if processing can continue without fetching additional events. This comment was generated by openai/gpt-oss-120b at temperature 0.1.
