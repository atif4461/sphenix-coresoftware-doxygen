Detailed Description

RawTowerv1 but allow flexible tags.

Raw tower version two provides extended properties for scintillation and cerenkov photon counts This class inherits from RawTowerv1 and adds a generic property map for additional tower attributes Constructors allow creation from existing tower objects identifier keys or calorimeter indices Reset clears the property map and restores base class state isValid reports validity based on base class criteria identify prints a textual description of the tower Property accessors get and set scintillation and cerenkov gamma values using the generic property mechanism The generic property interface enables checking existence retrieving and assigning arbitrary numeric properties
